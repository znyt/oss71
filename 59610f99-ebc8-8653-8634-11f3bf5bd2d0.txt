　　这个中医院叫“万草堂”，是司徒家的重要产业之一。
<br/>　　万草堂是司徒家的私人产业，可它却是得到国家中医协会认证，得到政府支持的，而且万草堂一些被传承了数百年的诊治方法和药方甚至获得非物质文化遗产的保护，这就使得万草堂的地位立刻跟其他那些挂着羊头卖狗肉的中医院有了天差地别的区别。
<br/>　　现在万草堂出诊的是司徒空的几个堂兄弟，大掌柜也就是司徒空的大伯司徒青扬已经很少再过问万草堂的事情，更不会为人出诊，这次完全是因为司徒空的关系才会接待慕容飞和佟千雪。
<br/>　　“你出去吧，让他们自己进来。”
<br/>　　屋内传来一道略显厚重的声音，声音中带着不容置疑的命令之意。
<br/>　　年轻女子对着慕容飞和佟千雪做了一个“请”的动作，说道：“慕容先生，佟小姐，大掌柜就在里面，你们自己进去就好。没有大掌柜的允许，谁都不准进入到这间屋子的。”说完，就转身离开，沿着刚才来的那条路原路返回。
<br/>　　慕容飞又打量了一遍自己眼前这间古色古香的厢房，这里跟唐易的茶舍在建筑风格上有些想象，只是唐易的茶舍里弥漫着的是茶香，而这里漂浮在空气中的是药香。
<br/>　　跟平时闻到的煎中药味道，这里的药香闻着让人觉得很舒服，完全没有那种闻到气味就联想到中药苦涩的感觉。
<br/>　　慕容飞推门走进屋子，在门打开的刹那，那股药香味变得更加浓郁了。
<br/>　　“你们来了，随便坐！”司徒青扬放下手中的线装书，对慕容飞和司徒空招呼道：“既然是小空的兄弟就别拘束。”
<br/>　　听刚才司徒青扬的声音，慕容飞还以为他会是个很严肃的人，可是看到本人，慕容飞才发现司徒青扬跟自己想象中的样子有很大的出入，是个很慈善的老者。
<br/>　　司徒青扬穿着一身布长袍，留着胡子，就跟影视剧里看到的老中医一个样。
<br/>　　招呼慕容飞和佟千雪坐下后，司徒青扬坐到旁边的座椅上，仔细打量着佟千雪，观察着她的气色；然后细听她的声息；紧接着是切脉。
<br/>　　望闻问切，司徒青扬没有“问”就对佟千雪的病情有了了解，说出的结果跟在医院诊治后的结果基本一致，这让慕容飞察觉到了司徒青扬医术之高明。
<br/>　　在医院各种检测做了一个又一个，又是各种专家凑在一起会诊了半天，才得出跟司徒青扬相差不多的诊治结果，孰高孰低，立见分晓。
<br/>　　至于司徒青扬为什么省去了最后的“问”，是因为他已经从司徒空那边对佟千雪的情况有了一个了解，所以没必要再向佟千雪再问一遍。
<br/>　　“司徒先生，我妻子到底能不能治好？”
<br/>　　慕容飞马上就要开始执行“追魂”行动，所以他现在急于知道佟千雪的确切情况。
<br/>　　“不难不难。”司徒青扬摆了摆手，说道：“在西医看来，这是个难题但是在我看来这并不是什么大难题。司徒家的老祖宗曾是两朝皇帝的御医，古时候三宫六院嫔妃勾心斗角是常有的事情，司徒家的老祖宗曾留下过方子，就是应对佟小姐这种受到外力影响而流产的情况，我们也曾有过治愈的先例，所以佟小姐的病症并不是一个难题。”
<br/>　　司徒青扬提起桌子砚台上的小楷笔，在纸上写了一个药方，吹干墨迹，对慕容飞说道：“拿着这个药方去前厅拿药，有些治疗细节我要跟佟小姐叮嘱。”
<br/>　　慕容飞点了点头，接过药方离开了屋子去拿药。
<br/>　　“司徒先生刚才是有意隐瞒了我的实际情况吧？”佟千雪直接开口对司徒青扬问道。
<br/>　　司徒青扬轻笑着摇了摇头，说道：“还是没有能佟小姐，果然要骗一个医生还是很难的。”
<br/>　　“司徒先生有话就请直说。”佟千雪深吸一口气，使自己尽可能保持平静的对司徒青扬说道：“我有最坏打算的准备。”
<br/>　　司徒青扬摸着自己的胡子，轻轻点了点头，说道：“佟小姐能这样想最好。”
<br/>　　司徒青扬也不再对佟千雪有所隐瞒，直言道：“如果佟小姐只是受到外力的影响导致流产，这并不难治，只要稍加的调养就能治愈；关键是你在流产后又进行长途的移动，并且自身又处在一个恶劣的环境中，这就使得问题变得有些棘手起来，而你的心情起伏波动也是对你的情况产生了影响。”
<br/>　　“如果只是我之前说的那种情况，只需要服用几贴药，就能治愈，但你的这个情况不是服药就能解决的，需要配合针灸，内服外敷。我让你先生去拿的药你先拿回去喝一个疗程，并且配合我方子上的敷贴进行调养治疗。服完这剂药，你再来这里找我，我配合针灸进行第二步的治疗。你自己也是学医的，应该明白中医跟西医最大的区别在于疗程的长短，西医可能只需要几粒药片解决的事情，中医的疗程就要比它长很多，这也使得现代快节奏的生活跟西医更契合。”
<br/>　　佟千雪点头，表示认可。
<br/>　　现代都市快节奏的生活，让绝大多数人无法接受中医冗长的治疗方案，而更倾向于西医“简单暴力”的治疗方式。没有什么事情是一瓶点滴解决不了的，如果有，那就两瓶！
<br/>　　“还有一点，我需要提醒你。”司徒青扬对佟千雪说道：“在治疗期间，要杜绝房事。在你身体调理过来之前，贸然进行房事会严重影响到治疗效果，我相信你应该能理解。”
<br/>　　“司徒先生，这个治疗需要多长时间？”佟千雪对司徒青扬问道。
<br/>　　“短则半年，长则一年甚至两年也有可能，这要看你的情况而定。”司徒青扬回答道。
<br/>　　“可是我今年已经33岁了，如果需要两年甚至更长时间来治疗，我怕……”佟千雪言语间有些为难。
<br/>　　“你是怕生育困难？”司徒青扬立刻就看出了佟千雪的顾忌，笑道：“这个你不需要担心，只要调养好身体，一切都不是问题。”
