　　“混沌神殿是干什么的？”
<br/>　　“收天骄，登天路，踏歌行，培养人成圣，成帝，成仙。”青铜古棺内，那虚无缥缈的声音再次响起。
<br/>　　哗！
<br/>　　听到此话，众人心中难掩的火热，成圣、成帝、成仙，那不就意味着我们可以成仙了？一时之间，众人激动不已，竟然忘却了刚才的惊慌。
<br/>　　“为什么会选中我们？”有人问出这样一个问题。
<br/>　　“此乃混沌神殿第一次运行，随机抽取，算你们运气好，能够被抽中。”
<br/>　　“啊，我们可不可以回去啊？”
<br/>　　“不行，生或者死，二选一。”
<br/>　　“啊？完了，完了，我还没有娶媳妇呢，我还不想死？”叶梵好友庞博大声叫道。
<br/>　　“啊？该怎么办？我想回家！”有人哭喊道。
<br/>　　“我要成仙！”也有人这样喊道。
<br/>　　“天下没有免费的午餐，需要我们付出什么？”林佳比较细心，问出这样一个问题。
<br/>　　“生命、灵魂、资源、道经，什么都可以。”
<br/>　　“我们现在要去哪里？仙界吗？”有人这样问道。
<br/>　　“第九个问题，正在星空之中，将前往荧惑星。”
<br/>　　“荧惑星？荧惑是什么地方啊？”有人问道。
<br/>　　“荧惑星是火星。”
<br/>　　“好了，问题回答完毕，现在开始宣布规则。”那神秘的声音接着说道。
<br/>　　“啊？这就完了？”有人狠狠的瞪了那人一眼，竟然问出如此愚蠢的问题。
<br/>　　“最后一个问题不算，我们可不可以重新问？”有人对着铜棺喊道。
<br/>　　“不行，吾现在将宣布主神规则。混沌神殿是为了培养人成帝，成仙的地方，这里不需要废物，不需要傻子。规则如下：”
<br/>　　“第一，不得有意或者无意向他人泄露神殿存在，违者，抹杀！”
<br/>　　“第二，神殿向弟子们颁布任务，通过完成任务获得价值点，也可以用任务中获得的宝物兑换价值点，之后来神殿空间兑换宝物，价值点低于一千者，抹杀！”
<br/>　　“第三，任务失败者，抹杀！”
<br/>　　“好了，暂时只有这三条，现在发布第一次任务：探险任务。”
<br/>　　“探险任务：探索荧惑古星上的大雷音寺遗迹。”
<br/>　　“任务要求：踏入荧惑古星，待足一个小时，活着回到混沌神殿。”
<br/>　　“任务奖励：成功，一千价值点，失败，死亡。”
<br/>　　“任务提醒：还有五个小时，任务开始。”
<br/>　　轰！
<br/>　　三个大大的抹杀，顿时吓住了所有人，抹杀，那就意味着死亡，那可是真正意义上的死亡，没人能够面对死亡还能冷静的。
<br/>　　“我要回家。呜呜——”有人哭喊道。
<br/>　　“我不想死！”有的心志弱小的人大叫。
<br/>　　“好了，大家安静，现在哭也没用。”叶梵大声叫道，安慰众人。“现在要做的是好好想想怎么完成任务？”
<br/>　　“叶梵说得对，我们不能乱了阵脚。”他的好友庞博大声叫道。
<br/>　　“那现在，我们该怎么办？”尽管有人恐慌不已，但还是有人镇定，小声问道。
<br/>　　“我们现在想太多也没用，现在要想的是任务。任务只有一个要求，那便是活着。显然，这个任务具有很大的危险性，但想必也是有办法解决的，要不然，这就是叫人去死，也不会有这个任务了。”叶梵分析道。
<br/>　　“不错，叶梵说得不错，我们现在要做的是完成这个任务，而不是在这里白白浪费力气。”周易站出来大声说道。
<br/>　　叶梵点了点头，向着夜邪旁边坐下，该说的他已经说了，听不听在他人，只要自己小心一些就是了。
<br/>　　“叶子，你们走的好快，我赶都赶不上。”这时，叶梵好友庞博走到他身边，对他说道。
<br/>　　“庞博，你不是不来吗？”叶梵问道。
<br/>　　“最初，我家中有事，我未能来得及参加宴会，后来听说大家要来泰山，我处理完家中的事，自然赶来了。”庞博解释道。
<br/>　　夜邪看着他满脸苍白，气血败亏，明显是受过伤。他若有所思，看来是庞博身上的大妖血脉已经开始觉醒了。
<br/>　　“朋友，你叫什么名字？”庞博对着夜邪问道。
<br/>　　“夜邪。”夜邪冷声道。
<br/>　　“夜邪，呃？爷爷——咳咳咳！”庞博先是不以为意，反应过来后便是大声惊叫，满脸尴尬。
<br/>　　“嗯。”夜邪轻声应答道。
<br/>　　“咳咳！容我先缓缓。”庞博咳嗽一声。
<br/>　　旁边叶梵强忍着脸上的笑意，不动声色。
<br/>　　良久，直到众人都喊累了，才安静下来，铜棺之中顿时陷入无声的气氛中。
<br/>　　五个时辰后。
<br/>　　“砰！”
<br/>　　一声巨震，像是飞机在高空穿越寒冷的云层，结了一层厚厚的冰甲一般，不断地摇颤着。
<br/>　　“轰！”
<br/>　　又是一声巨震，简直像惊雷一般，似欲震碎所有东西，明显可以感受出来，青铜巨棺发生了大碰撞。
<br/>　　不过，也正是在此时，青铜巨棺刻图发出点点光芒，棺盖打开，光芒射了进来。
<br/>　　“怎么回事，到火星了吗？”有人惊疑不定道。
<br/>　　“光！外面透进来的光芒。”李小曼惊呼。
<br/>　　所有人齐转头，向李小曼所看的方向看去，前方果然出现点点暗淡的光芒。
<br/>　　正在此时，那神秘的声音再次在众人脑海之中响起：
<br/>　　“任务开始，探索古遗迹，为期一小时。成功，一千价值点，失败，死亡。”
<br/>　　“咻！”
<br/>　　一阵光芒闪烁，所有人都发现，在各自的左手手腕处，有了一道淡淡的青色印记，那印记正是青铜巨棺的模样，只不过是被缩小了不知多少倍。
<br/>　　同时，那棺盖也彻底被打开了，无限光芒照射了进来，照亮了神殿。
<br/>　　众人发出一声惊呼，争抢着向前冲去，走出青铜棺。
<br/>　　不过，当人们走出青铜棺后，却全部如泥塑的雕像一般呆住了。
<br/>　　眼前的大地像是被血水浸染过一般，呈红褐色，冷硬而枯寂，入眼所见，一片荒凉与空旷，地面上灵性矗立着一些巨大的石碑，像是一座座墓碑。
<br/>　　天地昏暗，一片昏沉。众人呆若木鸡，这里绝不是泰山。
<br/>　　同时，也有人爬上石头，看到了钟鼎文，认出那是“荧惑”两字，此时，人们内心更加绝望了。
<br/>　　也有人仰天，看见了两轮秀小的月亮挂在天边，他们顿时色变。
<br/>　　“这……这是……两颗月亮？”
<br/>　　他们知道，此处绝对不是地球，于是，他们彻底绝望了。
<br/>　　最后，他们不得不听从任务，按其所做。因为，他们发现，那个神殿说得，都是真的，这里真的不是地球，而是火星。
<br/>　　一阵绝望之后，众人不得不重拾心情，开始按照任务要求而来，去探索大雷音寺遗迹，说不得还有转机，毕竟那里可是神话传说中的地方。如若不然，他们就只有等死了。
<br/>　　沿着发现的残瓦碎片走出数百米后，距离光源越来越近。而就在此时，有人惊呼，发现了一片坍塌的建筑物，似乎是一座古老的亭台，在岁月的侵蚀下倒塌了。
<br/>　　“那就是大雷音寺遗迹吗？”有人惊呼，纷纷向那里跑去。
<br/>　　……
