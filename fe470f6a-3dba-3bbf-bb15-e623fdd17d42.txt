
    第二天照常启程，这一次队伍里则是充斥了诡异的气氛，至少今天的王惊蛰就没去故意舔三井美奈了，陈三岁则是眯着眼睛瞅着前面的人，压低声音跟孔良提醒了一句。<br/>　　<br/>　　    “老孔啊，我猜他们今天可能是要杀人灭口了，你得小心点了”<br/>　　<br/>　　    孔良慌张的“啊”了一声，呼吸明显急促的问道：“你怎么知道，这是如何说的？”<br/>　　<br/>　　    “血气上涌，杀意外漏，他们自然不可能是自己人来自相残杀吧，那目标就明显是我们咯？我估计啊，人家是到了想找的地头上了，咱们这么碍眼他们肯定不愿意继续跟着啊，劝又没办法劝，那就只好在这荒郊野外杀人灭口了”陈三岁眼睛有点放光的舔着嘴唇，说道：“一会他们要是动手了，老孔你啥都不用管，自己顾着自己，其他的我和王惊蛰来处理，唉，你说咋样？”<br/>　　<br/>　　    “什么咋样？”王惊蛰回过神来了，他刚才忽然间发觉这片山脉的走势似乎有点问题。<br/>　　<br/>　　    “杀人者恒杀之……”陈三岁手起刀落。<br/>　　<br/>　　    王惊蛰皱眉说道：“不要搞事情，我没空搭理他们，这片山脉有点古怪，很不合常理，有点意思，让我有些想不太清楚”<br/>　　<br/>　　    陈三岁顿时诧异的说道：“不是，你的血性呢，你的霸气呢，说好的敌不犯我我不犯人呢，他们都要灭口了，你不得一巴掌扇回去么？”<br/>　　<br/>　　    王惊蛰根本都没搭理他，朝着孔良示意道：“你之前曾经说过，这头獒王产生了灵性，有可能是跟吃了什么天才地宝有关，如果龙骨草真的在这片雪域高原上的话，一般人找不到，那獒王是完全有可能找得到的，就是因为獒王的足迹也许踏遍了雪域上的每一个角落，更有可能是它出没的地方，生长着外面没有的草药，是这个意思吧？”<br/>　　<br/>　　    孔良点头道：“这个可能性非常大，有灵性的动物你都不能用常理来判断，我就见到过雪山峭壁上生长的雪莲，旁边就会有金雕搭的窝，它就一直在守着雪莲成熟，然后再来采摘”<br/>　　<br/>　　    王惊蛰搓了搓手，掏出烟来点上咬着烟嘴，轻声说道：“这里的海拔有六千多米了，孔哥你说如果在这片地方，有一个类似于沙漠上绿洲的存在，那头獒王又生活在这里，那你说这里生长着龙骨草的话，是不是就更有这个可能性了？”<br/>　　<br/>　　    孔良顿时迷茫了，完全听不懂的摇头道：“你在说什么，我没听明白”<br/>　　<br/>　　    陈三岁惊愕的说道：“他，他是说在这片山脉里，很，很有可能有一处春暖花开的地方”<br/>　　<br/>　　    孔亮当即摆手，笃定的说道：“这不可能，这里海拔超过六千米了，植被最多只能生长出一些苔藓，松木类或者灌木这种植物，如此低的温度还有大雪封山，哪里来的大片绿洲，这是违反尝试的，绝对不可能”<br/>　　<br/>　　    这时候，前面三井家族的人，已经走到了一处山头上，从他们这个方向向下眺望，在山脉底部隐约有个村庄的轮廓，三井小叶看了眼导航仪上的坐标，扭头朝着小林昙一指了指山脚下的村子。<br/>　　<br/>　　    “没错，就是这里，这些没开化也从来没有和外界接触过的藏民描述过，几十年代马丁鲍曼确实来过这片山脉……”<br/>　　<br/>　　    藏区，占据了全国至少六分之一的国土面积，这是什么概念呢，笼统的说就是什么东三省，华南，华北这些省份全都摞一块了，也就勉强能赶得上藏区的面积，而在整个藏区中至少有半分之八十以上的土地是没有被开发甚至人烟罕至的，但这些地方却也仍旧生活着一些人，就是那种很古老的藏民，比如传说中的珞巴族人。<br/>　　<br/>　　    这些几乎等同于未开化，始终保持着原始生活常态的人过着与世隔绝的生活，鲜少被人打扰，而藏区哪怕就是有人知道这些人在哪，也从来没有刻意的接近他们，让这片星空下一直保留着那一块净土。<br/>　　<br/>　　    这处山脉的下方就是如此，一个延续了千年的古老村落。<br/>　　<br/>　　    三井美奈瞥了眼离他们大概十米左右距离的王惊蛰三人，朝着三井良河打了个手势，示意他们这个时候可以灭口了。<br/>　　<br/>　　    但是，就在这时，山谷中忽然传出一声沉闷的尖啸。<br/>　　<br/>　　    “嗷呜……”<br/>　　<br/>　　    这沉闷的动静从响起之后，直到消逝，声音回荡了许久，始终荡漾在层层叠叠的山谷中。<br/>　　<br/>　　    “唰”王惊蛰，陈三岁还有孔良顿时抬头四处寻觅，这毫无疑问是那头他们追踪了几天的獒王传出来的。<br/>　　<br/>　　    于此同时，没来得及动手的三井美奈几人也诧异的东张西望着，这突如起来的吼声让他们吓了一跳，如此连绵不绝和沉闷的巨大动静，明显是个庞然大物吼出来的。<br/>　　<br/>　　    十几双眼睛来回眺望了片刻，小林昙一忽然指着他们对面的一座山头，急促的说道：“看，快看那里”<br/>　　<br/>　　    那座山头上，站着道雄姿壮硕的身影，虽然离得挺远但是它迈着前蹄，昂首的样子仍旧十分壮观，你甚至都能感觉到它来的雄浑气息，十分的摄人。<br/>　　<br/>　　    “吼！”山头上的身影又再次仰天长啸了一声，然后身影晃动了几下后就调转了身子，随即一头马熊爬上了山头，一熊一狗相隔不远的对峙着。<br/>　　<br/>　　    山头的那边顿时上演了一场令人惊异的战斗，獒王稍稍往后退了几步半弓着腰，蓄势待发，只见它速度极快的径直冲向了那头直立起来比它至少要高出大半个身子的马熊。<br/>　　<br/>　　    马熊似乎不甘示弱，直立身躯伸出两只熊掌挥舞了两下后，也同样怒吼了一声，就前倾着身子朝着獒王挥起手掌拍了过去，但让人绝对无法想象的是，半空中前扑的獒王，直接张开大嘴不闪不避的朝着马熊咬了过去。<br/>　　<br/>　　    虽然离得挺远，但观望的人都在獒王和马熊相接的瞬间，在心里响起了“嘭”的一声，因为这两头牲口的体型实在是太大了。<br/>　　<br/>　　    獒王撞向马熊，张开大嘴狠狠的就咬在了它的脖子上，同时熊掌也一巴掌拍在了獒王的后背以下，两头牲口纠缠着就从山上往下滚了过去。<br/>　　<br/>　　    小林昙一顿时指着那边山头说道：“我知道了，是那头藏獒啃食了牦牛……”<br/>　　<br/>　　    想和更多志同道合的人一起聊《天命赊刀人》，微信关注“热度网文或者rdww444”与更多书友一起聊喜欢的书
