　　然而，
<br/>　　就在叶尘准备将灭世魔刀封印在丹田之中的时候，
<br/>　　轰隆隆！
<br/>　　上方忽然传来一道惊天巨响！
<br/>　　此时，
<br/>　　岩浆深渊的上方，除了冥沧荨和其他三大家族的人之外，不知道什么时候，又来了一队人马，大约三十来人。
<br/>　　这些人，个个一身白袍，气息浑厚无比，竟然有将近一半，都在化神境之上！
<br/>　　足足十多位化神强者！
<br/>　　而站在众人最前方的，则是一名身穿紫袍的白眉老者，气息更是强大的可怕，甚至还远远在之前，擎苍宗的那两名强者之上！
<br/>　　而且，这些人的衣袍之上，全都绣着一个漩涡状的图案！
<br/>　　如果叶尘在这里的话，一定可以一眼认出，这些人身上所穿的服饰，都是来自于那个门派，无极宗！
<br/>　　原来，这两千年来，
<br/>　　云苍生和整个无极宗，无时无刻都在调查灭世魔刀的下落！
<br/>　　为此，无极宗几乎在每个星域都安插了自己的探子，一旦得到关于灭世魔刀和灭世仙君的消息，就会有特使前来核实。
<br/>　　而眼前的这个紫袍老者，便是无极宗的特使！
<br/>　　“你们的意思是说，一个只有区区元婴境的小子，不但赶跑了擎苍宗的韩月公子，而且进入这深渊下的岩浆之中，已经整整一天时间，都还没有出来？”
<br/>　　紫袍老者瞥了一眼三大家族的那几位化神强者，苍老的脸庞之上，露出一抹质疑之色。

<br/>　　“回禀特使大人！此事千真万确，这是我们所有人亲眼所见，绝对不敢欺瞒特使大人您呐！”
<br/>　　“哼！谅你们也没有这个胆子！！”
<br/>　　紫袍老者冷哼一声，眉头顿时皱的更紧了。
<br/>　　以他半步返虚境的修为，可以清晰的感受到，这深渊下方的岩浆，显然极为恐怖，甚至可以说深不可测！
<br/>　　倘若真如这些人所言，之前进入岩浆中的那个神秘年轻人，具备斩杀化神境中期的实力，就算实力不如他，应该也差不了太多了。
<br/>　　“特使大人，我们要不要也下去看看？”
<br/>　　无极宗的一名普通弟子，忽然开口询问道。
<br/>　　紫袍老者摆了摆手，
<br/>　　“不急！再等等！”
<br/>　　在他看来，既然三大家族的人，把那个神秘年轻人说的神乎其神，应该不会这么容易死掉，倒不如守株待兔，倘若真让那小子得到了那件宝贝，他直接出手抢过来，也就是了！
<br/>　　紫袍老者才刚刚打定主意，
<br/>　　轰隆隆！
<br/>　　下方岩浆，突然一阵晃动！
<br/>　　“嗯？”
<br/>　　紫袍老者眉头微微一挑，瞬间感应到下方岩浆之中，突然出现了一道气息，朝上方飞速而来。
<br/>　　“那小子出来了！大家列阵准备！”
<br/>　　随着紫袍老者一声低喝，
<br/>　　唰唰唰！
<br/>　　十多名化神强者，当即在深渊口的周围，分布站定，形成合围之势。
<br/>　　片刻之后，
<br/>　　彭！
<br/>　　随着一道巨响，深渊下方的岩浆，顿时一下子爆炸开来！
<br/>　　继而，
<br/>　　嗖！
<br/>　　一道瘦削的人影，自下方岩浆中爆射而出，冲天而起。
<br/>　　此人自然正是刚刚炼化了灭世魔刀叶尘！
<br/>　　“好强的毁灭气息！好小子！看来灭世魔刀，果然已经被你拿到了！既然如此，那就交出来吧！”

<br/>　　更何况，他们这边，可是足足十几位化神强者！
<br/>　　再加上他这个半步返虚境，即便是真正的返虚真君在此，面对如此强大的阵容，恐怕也得认怂！
<br/>　　叶尘悬空而立，在周围众人身上扫了一圈，不由得嘴角微微一扬，
<br/>　　“无极宗吗？你们来的倒是挺快啊！”
<br/>　　如果说，叶尘对灭世仙君之前的话，还心有存疑的话，现在他才算是完全彻底的信了。
<br/>　　南斗星域离无极宗相隔数百个星域，他们居然能够在这么短的时间之内，带着这么一批强者来到此地，而且点名要自己手里的灭世魔刀，显然对此事极为重视！
<br/>　　紫袍老者傲然一笑，
<br/>　　“小子，既然知道我等的来历，那你就应该清楚，灭世魔刀乃是天下邪物，不是你一个区区元婴境的小子可以驾驭的，只要你主动交出来，本特使可以保你成为我无极宗的核心弟子，甚至你之前和擎苍宗的恩怨，本特使也可以帮你解决，如何？”
<br/>　　哗！
<br/>　　紫袍老者此话一处，南斗星域三大家族的众人，顿时纷纷露出羡慕、嫉妒的神情。

<br/>　　那可是无数修真者，梦寐以求的东西啊！

<br/>　　他面对的是叶尘。

<br/>　　叶尘双手负在身后，脸上挂着一抹淡淡的嘲讽笑容，慢悠悠的开口道。
