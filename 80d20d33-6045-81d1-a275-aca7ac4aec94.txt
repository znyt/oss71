　　楚风眠打定了主意，今日可以不杀这碧螺老祖，但是这建木树枝楚风眠一定要得到。
<br/>　　现在碧螺老祖只是炼化了建木树枝很小的一部分，楚风眠还可以利用手段将这建木树枝从碧螺老祖的身躯之中给剥夺出来。
<br/>　　这建木树枝，对楚风眠的神树有着极大的好处。
<br/>　　楚风眠下定决心，无论如何这建木树枝他今日也要得到。
<br/>　　只见楚风眠的身形闪烁，刹那之间冲到了碧螺老祖的面前。
<br/>　　祖龙至尊剑，巨阙神剑，双剑齐出，耀眼的剑光之中，蕴含极致剑意，向着碧螺老祖斩杀而去。
<br/>　　“没用的，就凭你也想要破开我的朽木真身？”
<br/>　　看到楚风眠斩杀而来的剑光，碧螺老祖的嘴角露出了几分不屑笑容。
<br/>　　任由这剑光，斩在了他的身躯之上。
<br/>　　嘶拉！
<br/>　　剑光斩下，这碧螺老祖化为的一尊朽木，被当即斩的四分五裂。
<br/>　　但是接下来，跟刚刚的景象一模一样，随着一股生命力的凝聚，这一尊朽木之上的裂痕，纷纷恢复过来。
<br/>　　这碧螺老祖化身的朽木，仿佛如同不死之身一般，承受着在强大的攻击，都足以化解恢复。
<br/>　　“这朽木，绝对是不亚于建木树枝，甚至是还要超越这建木树枝的至宝！”
<br/>　　楚风眠盯着那朽木，眼神闪烁。
<br/>　　这种力量，绝不是属于那碧螺老祖的力量，而是属于那一段朽木的。
<br/>　　现在这碧螺老祖跟这一段朽木，人木合一。
<br/>　　楚风眠剑锋斩杀上去的力量，都打入到了这朽木之中，被这一段朽木所化解了。
<br/>　　“小子，不用白费力气了，再不逃一会你就别想逃了！”
<br/>　　碧螺老祖看着楚风眠，故意出言讥讽道。
<br/>　　“一个乌龟壳罢了！我倒是要看看你可以抗到什么时候！”
<br/>　　楚风眠眼神之中露出了几分愤怒，好像是真的被碧螺老祖的话激怒了一般，凝聚剑光，剑锋闪烁，又是数道剑光轰击在了这一尊朽木之上。
<br/>　　一道一道的剑光斩在这朽木之上，这朽木一次次的破碎，千疮百孔，破碎不堪。
<br/>　　可是纵然是被斩的千疮百孔，这朽木之上依然是在散发着惊人的生命力。
<br/>　　在剑锋的间隔之中，这朽木便是足以恢复过来。
<br/>　　甚至是以楚风眠纯阳天火尝试，灼烧着朽木，居然都无法燃烧这朽木。
<br/>　　火克木，本是天经地义的事情，可是纯阳天火，居然都拿这一段朽木无可奈何。
<br/>　　“怎么回事？碧螺老祖，你被谁逼到了这番地步？”
<br/>　　轰隆！
<br/>　　突然之间，光幕闪烁，一道身影，突然踏入到了这光幕之中，穿越光幕，来到了密室之中。
<br/>　　这是一尊九劫古帝，看身上的力量，应该是一位沧海都的九劫古帝。
<br/>　　这一尊九劫古帝的到来，一眼就看见了碧螺老祖，看着碧螺老祖化身朽木，吃了一惊。
<br/>　　他跟碧螺老祖也是老相识了，自然懂得碧螺老祖这一招是他的保命绝技，不碰到生死危机的时候根本不会动用。
<br/>　　现在这碧螺老祖居然是被人逼迫到了底牌尽出的地步。
<br/>　　碧螺老祖看到来人，脸色大喜，大声的喊道。
<br/>　　“葵海王，你来的正好，这小子就是楚风眠，你来跟我一起联手，杀了此子！”
<br/>　　“楚风眠？”
<br/>　　那被称为葵海王的九劫古帝，盯着楚风眠，眼神一寒。
<br/>　　楚风眠之前以一敌二，从碧螺老祖，五行刀祖二人手中逃出生天的消息，已经是传递了出去。
<br/>　　楚风眠实力的成长，实在是太恐怖了，以至于现在三大势力之中的九劫古帝，已经是对楚风眠下了必杀令。
<br/>　　要在楚风眠成长起来之前，一定是要杀了楚风眠。
<br/>　　在听到碧螺老祖的话后，这葵海王便是毫不犹豫，突然之间出手了，在他的手中突然一把三叉戟飞了出来。
<br/>　　三叉戟，相传是海神的兵器。
<br/>　　这沧海都的古帝，许多都是以三叉戟为兵器，这三叉戟一出上面凝聚着一股灵力，凝聚在锋芒之上，一戟对着楚风眠轰击而来。
<br/>　　“剑破！”
<br/>　　楚风眠瞥了一眼那三叉戟，手心剑锋一闪，一道剑光相对而去。
<br/>　　剑戟相撞，葵海王顿时感觉到了一股巨力的轰击而来，差一点都拿不住手中的三叉戟，连连退后，眼神中露出不可置信的光芒。
<br/>　　“怎么会有如此强大的力量，此人不是才是八劫古帝境界吗？”
<br/>　　这葵海王都有些没有想明白楚风眠怎么会有这样的力量，楚风眠虽然只是八劫古帝的境界，可是凭借力量，比起许多的九劫古帝还要强横。
<br/>　　寻常八劫古帝的力量，甚至不足九劫古帝的十分之一，在葵海王看来，纵然是楚风眠得到了几分奇遇，最多也就是跟他抗衡。

<br/>　　再不用全力，别说是杀了楚风眠，可能都要败在楚风眠的手中。

<br/>　　葵海王长啸一声，施展出了他最为强横的手段，一尊海神虚影，凝聚在他的背后。

<br/>　　那一根根翠竹，还有三叉戟的轰击，都落入到了这一道星河之中，被其中的力量无穷无尽的冲刷着，旋即全部化为了碎片。
<br/>　　其中的力量，都化为了星河的一部分。

<br/>　　于此同时，楚风眠突然出手了，一枚吞天种子，不知道什么时候打入到了碧螺老祖的身边。
<br/>　　随着楚风眠的催动，这一枚吞天种子，当即爆发了出来，化为了一道吞天秘术。
