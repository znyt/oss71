　　　　天上掉下一只百鸟之王，涂小安还能跟它客气吗。
<br/>　　至于这孔雀为什么会落下来，涂小安大致明白了，从这只孔雀吃蛇胆的行为，肯定是现场几百条的毒蛇将它给吸引了过来，准备饱餐一顿。
<br/>　　孔雀喜欢吃蛇胆，这也算是千古奇闻，都说蛇胆明目，莫非这只孔雀想让自己的眼睛变的更加的美丽？
<br/>　　在看现场，这只拖着长长美丽尾巴的孔雀已经在蛇群堆中大吃四方了，尖尖的嘴一啄，便是一颗鲜活跳动的蛇胆，见血封喉一般，没有一条毒蛇是它的一招之敌。
<br/>　　这些让人恐惧的毒蛇，在这只孔雀面前宛如待宰的羔羊，只有等死，纵然是蛇口咬在孔雀的身上，那也仅仅是咬到它丰厚的羽毛上，根本不顶用。
<br/>　　不过区区半响，已经有不下三十条毒蛇被啄走蛇胆而亡。
<br/>　　毒蛇被逼急了，四面八方几十条的同时进攻，几百条毒蛇，还能被一只鸟给团灭了？
<br/>　　可就在这时...
<br/>　　嘤嘤嘤...
<br/>　　孔雀亮晶晶的眼眸发光发亮，发出了一道嘹亮的尖叫声，紧接着，它长长的尾巴抖得哗哗响，漂亮的尾巴就像仙女手中的彩扇，慢慢的展开，展开的彩屏像一把巨大的羽毛扇，羽毛上闪烁着无数只眼睛，每一只眼睛闪闪发光，像是有无数颗星星，亮瞎人的眼。
<br/>　　“孔雀开屏？”
<br/>　　涂小安看的兴奋，好一副美轮美奂的画面，没有真正见过孔雀开屏，你是无法想象它的美。
<br/>　　况且这还是一只血统无比高贵的孔雀，不是动物园中那些供人欣赏的蓝孔雀可比。
<br/>　　但美归美，漂亮归漂亮，却是带刺的玫瑰，会伤人。
<br/>　　只见孔雀一开屏，羽毛上闪闪发光，空气中，有一股无形的力量在激荡，看着看着，你既然发现自己不由得被催眠似的，晕晕沉沉，深陷其中无法自拔。
<br/>　　涂小安顿时心中一惊，眼睛浑然一眯，蛇眸立刻清明起来。
<br/>　　旋即，他一个放眼望去，只见一条条冒着凶凶光的毒蛇软绵绵的趴在地上一动不动，仿佛陷入了一场美丽的梦境中，不可自拔。
<br/>　　嘤嘤嘤...
<br/>　　孔雀青枭开心的叫唤了几声，显得非常的得意，它先用自己尖尖的嘴梳理了一下自己美丽的羽毛，然后优雅的迈着自己的长腿准备收割。
<br/>　　尖尖的嘴冲着一条沉迷的毒蛇一啄，便是一颗鲜活的蛇胆。
<br/>　　就这样，孔雀轻松的一颗一颗蛇胆吞着乐此不疲，这些毒蛇死了都不知道自己是怎么死的。
<br/>　　忽然，有一条毒蛇吸引了孔雀的注意力，整个现场的毒蛇都趴在地面上，集体沉迷，唯有这条蛇还昂首而立，非常的突兀。
<br/>　　孔雀傲娇的看了这条毒蛇一眼，还别说，这条毒蛇长的挺与众不同，不像别的毒蛇那么丑陋，但旋即，它极度自恋的露出了鄙夷。
<br/>　　优雅的迈着长腿，朝着这条毒蛇走去，走到跟前，亮晶晶的眼眸流露出了轻蔑。
<br/>　　这顶多是一条长像不丑的毒蛇，还是一样要被自己吃蛇胆。
<br/>　　长长锐利的嘴骤然的朝着这条毒蛇的蛇胆位置啄去，非常的风轻云淡。
<br/>　　“铿锵...”
<br/>　　一道金属碰撞的声音迸发，孔雀的尖嘴好像啄到了钢铁上，被反弹了回来。
<br/>　　霎时间，孔雀眼中人性化的流露出了诧异，轻蔑的眼神收了回来。
<br/>　　这还是它第一次失手啊。
<br/>　　在看这条长的不丑的毒蛇，它的蛇眸之中居然也流露出了一道很熟悉的眼神，同样是轻蔑鄙夷。
<br/>　　“吃了那么多蛇胆，你还没吃够吗”
<br/>　　这条蛇说话了，显得非常的诡谲，吓得一阵山风都赶紧的呼啸而走。
<br/>　　“嘤嘤嘤...”
<br/>　　孔雀着实被吓了一跳，顿时警惕了起来，它明白了，这不是一般的毒蛇。
<br/>　　遇到强敌了，一股危机感油然而生。
<br/>　　顿时间，它长长拖地的尾巴再次展开了，孔雀开屏，羽毛间无数颗星星闪烁，空间微微的波动涟漪起来。
<br/>　　美轮美奂，让人不禁的陷进这种美丽之中，无法自拔。
<br/>　　不管是生物，还是人类，没有非常强大的意识力，难以保持本心，都会被孔雀开屏俘虏。
<br/>　　“你这招，对我没用的，收起来吧”
<br/>　　涂小安的蛇眸浑然一眯，无形中一股强大的意识力笼罩在孔雀的身上，硬生生将它展开的羽毛给合并了起来。
<br/>　　“轮到我了，意识锁定，给我收”
<br/>　　他现在等级十七级，这只孔雀虽然血统高贵不凡，但也仅仅是十五级。
<br/>　　孔雀感觉浑身笼罩这一股巨大的压力，吓的连忙挣扎起来。
<br/>　　“哼...乖乖的当我的宠物吧”
<br/>　　涂小安只有一只笨狗小八，现在弄只孔雀当宠物，提升一下档次。
<br/>　　毕竟他现在可是有身份的蛇，蛇王是也，这孔雀用来当坐骑也不错啊。
<br/>　　如此一想，有点让人欢喜雀跃起来，可旋即...
<br/>　　“叮..强制性收服高等生物孔雀失败”
<br/>　　嗯？涂小安傻眼了。
<br/>　　失败，怎么会失败。
<br/>　　说好的只要生物等级没自己高，他都能强制性收服？
<br/>　　敢情是假的？
<br/>　　这无法强制收服，就有点麻烦了，莫非要先将这只傲娇而美丽的孔雀打个半死，在收服吗。
<br/>　　“丑八怪，我不吃你的蛇胆了”
<br/>　　孔雀有点害怕了，发出了一阵清脆悦耳的声音，好似一个刚学会说话的小女孩。
<br/>　　咦，这孔雀居然能做到口吐人语了，涂小安闻言，顿时大吃一惊。
<br/>　　就算这只孔雀血统不凡，高等生物，可它还是一阶啊，按照涂小安知道的，生物修炼到二阶，才能具备口吐人言的能力。
<br/>　　看来高等生物就是不一样，含着金钥匙出生，起步就是高，好比那些天才，一岁会说话，两岁会认字，三岁读诗文。
<br/>　　但越是如此，涂小安越不能放过这只孔雀，收服了当宠物，潜力无限。
<br/>　　说不定以后利用系统的能力，能将孔雀变凤凰也不一定。
<br/>　　这是一个潜力股，有着无限的可能。
<br/>　　“跟我走，你想吃多少蛇胆，我就给你多少蛇胆”
<br/>　　涂小安准备忽悠这只孔雀，按照刚才听到的声音，这只孔雀显然年龄不大，是个小女孩，而且看这只孔雀的体型，也不大，还在发育期。
<br/>　　心智应该不会太成熟。
<br/>　　孔雀闻言，傲娇的看着眼前这条金银色的蛇，满是轻蔑之色，一副你配不上我的样子，道：“丑八怪，泥奏凯”
<br/>　　涂小安一头黑线，既然叫他丑八怪，他可是蛇类界的超级大帅哥呀。
<br/>　　“臣服我，跟我走，不然我吃了你”
<br/>　　只见金银蛇血口一张，模样凶煞狰狞，软的不行，只能用硬的。
