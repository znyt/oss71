<br/>				陈默缓缓站起身，将悬浮在自己面前的乾坤珠收了回来，然后自言自语的说道：“看来这个特管局我可能要加入了啊！”<br/>　　<br/>　　没有办法，无论是药材的收集，还是这种异种能量的研究，都需要强大的力量来支持。那么特管局的力量就可以达到，所以陈默才想着要加入。<br/>　　<br/>　　但是这个加入，是有条件的加入。在他的想法中，首先不能自己开口，需要特管局的邀请。这样一来，他才能提出一些要求，并且能够按照自己的想法来。<br/>　　<br/>　　另外就是，加入特管局最好是凭借自己的医术，而不是其他。要是将自己定身为一线的作战人员，到时候死都不知道是怎么死的。<br/>　　<br/>　　“轰！”的一声，陈默朝空气中挥了一拳，发泄~了一下自己心中的兴奋，练气四层可期啊!本来依靠自己修炼，外加乾坤珠内的灵气，怎么也要等个一年多吧！没有想到的是，异种能量带来升级的希望，陈默都想化身成任我行，逮着一个就吸一个！<br/>　　<br/>　　看时间也已经是晚上了，看来剩下的几个病人要放到明天了。<br/>　　<br/>　　转身走出练功房，然后就和李普河、常文斌说了一声，让他们先照顾好病人，等明天早上再治疗。<br/>　　<br/>　　借口也很好说，当然也不算借口，就是下午给韦俊才治病的时候，自己用内力将异种能量导出了一部分，造成自己的内力后继无力，所以要等到明白早上了。<br/>　　<br/>　　其实在陈默治疗韦俊才、和那个昏迷的男人，有了初步的成果后，就回去休息。李普河和常文斌就给这两个人做了一个初步的检查。<br/>　　<br/>　　检查结果发现，两个人身体内的异种能量已经减少很多！也就是说，这两个人的伤势真的是减轻了。按照目前检查的看来，这两个拖了这么长时间的伤势，能有希望治疗好了，这个真的是幸运啊。<br/>　　<br/>　　其实特管局内，很多被异种能量侵入后的伤者，要么就慢慢拖着，然后等这种能量消散，不过时间上就有的等了。<br/>　　<br/>　　要不就是先天高手出手，利用先天之力将异种能量去除。因为将自身能量滞留到敌人体内，基本都是达到圣骑士级别的高手。<br/>　　<br/>　　因为特管局也不是将那些后天三层一下的人员，派出去执行不可能的任务。那不是完成任务，那是炮灰去送死。<br/>　　<br/>　　实际上，一般特管局的任务都是分等级的，根据相关调查和分析，然后派出相应级别的武者。<br/>　　<br/>　　一般武者等级低，都是作为后勤和技术支援来利用的。就像是袁若珊，虽然她在特管局的级别高，但是在任务期间，她还是技术支援和统管全局，一线是不容许去的。<br/>　　<br/>　　与西方势力交战了上千年，武者和骑士之间的信息，基本上都是很明面的东西了。所以这种被异种能量侵入，基本上就是特列。有的时候就是倒霉催的，在交战过程中碰到高手被反制。<br/>　　<br/>　　有的时候就是信息调查不明，然后碰到圣骑士后，被打受伤。<br/>　　<br/>　　这些人，在特管局中的伤者里面，也算是少数。因为基本上圣骑士出手，更多的是威慑和挑衅。因为圣骑士和华国的先天高手一样，都是重要的保留力量，轻易间不出动的。<br/>　　<br/>　　而且圣骑士出手以后，容易引发华国这边派出先天高手，造成西方世界的骑士力量损失。<br/>　　<br/>　　所以，韦俊才等人的受伤，算是特例。所以这种伤者，在特管局也比较少，这次一股脑的都送到陈默这里，也是经过调查分析后，陈默的针灸能力非常不错，可以将血栓都能通过针灸疏通了，那么这些异种能量是否也能疏通掉呢？<br/>　　<br/>　　因为要是靠先天高手出力将异种能量化解，那么先天高手至少有一段时间不能出力，只能闭关恢复，这是特管局所不能接受的。<br/>　　<br/>　　本来先天高手就少，一但这样做的后果，就会引来西方修炼者的进攻，那么这个错误谁也担负不起。所以伤者只能拖着，也算是无能为力。<br/>　　<br/>　　特管局对于这些伤者，虽然有些无力，但是对于伤者的照顾，还是尽心尽力的，无论是花费大价钱买药，还是邀请各个医术高手治疗，都会尽心尽力的去做。这次送到陈默这里，也是特管局领导商量后的结果。<br/>　　<br/>　　伤者移动倒是没有什么，而且异种能量也比较稳定，不会爆发出来。所以给伤者挪个地方倒是没有什么问题，只要有治疗的希望，他们还是会努力去做的。<br/>　　<br/>　　李普河和常文斌，两个人都是特管局培养的医师。虽然他们的武者等级低，但是医术还是不错的。暂时照顾这些病人，没有任何的问题。<br/>　　<br/>　　通过检查后，李普河就给特管局的宁永志打了电话，并汇报了这里的基本情况，包括陈默在治疗时候所发生的事情，全部都反应给了他。<br/>　　<br/>　　宁永志接到电话后，顿时也就放下了一部分心。因为这次将伤者送去陈默哪里，他也是担惊受怕的，很多时候，作为领导肩负着重大的责任，也是很考验心脏的承受能力。<br/>　　<br/>　　尤其是听到陈默能治疗异种能量，顿时心情就是忽高忽低的，犹如过山车一般。<br/>　　<br/>　　“对于陈默的要求，你和常文斌一定要全力配合，做好后勤工作。另外，对于世兴文，先将他送回来吧！我明天安排人手，将他接回。并再送过去一个被异种能量所伤的人员，你们注意接收，安排好一切。”<br/>　　<br/>　　宁永志说道。对于世兴文这种不配合治疗的伤者，但是有时特管局的武者，只能先接回去，然后安排其他治疗，虽然要等，但这是世兴文他自己所考虑的，所以也就只能这样做。<br/>　　<br/>　　因为陈默已经再三确认了，那么世兴文不配合，宁永志也是没有什么好办法的，只能软处理了。<br/>　　<br/>　　对于陈默的医术，经过韦俊才和另外一个伤者的治疗后，他已经有了很大的信心。<br/>　　<br/>　　“小袁，你现在方便么？”宁永志直接拨通了袁若珊的电话。<br/>　　<br/>　　“您说！我现在没有问题。”<br/>　　<br/>　　“好！我今天接到李普河的汇报，你推荐的陈默非常不错，治疗了两位异种能量侵入的伤者，取得不错的疗效。李普河已经做了检查，证明了治疗效果。”<br/>　　<br/>　　“那我就放心了，要不然还真的是有些担心！做任务的时候都在想这件事情呢！”<br/>　　<br/>　　“你啊！是这样，我给你电话呢，是想着你完成这次任务后，再去一趟陈默哪里。看看治疗效果，如果可以，你必须要将陈默加入咱们上市的特管局！”<br/>　　<br/>　　“领导，这个有些困难啊！我已经和陈默说了两次了，他都没有答应！”<br/>　　<br/>　　“不管再困难，你也要努力完成。要知道现在特管局里，武者虽然缺，但是特种人才更加的缺，尤其要是陈默的医术得到证明的情况下，西市的李济深可是会近水楼台先得月的！这个可不能便宜了那个家伙。”<br/>　　<br/>　　“好！我记住了！”<br/>　　<br/>　　袁若珊听到如此说，当然也是会为自己的组织争取利益的。如果陈默的医术真的不错，那么一定要将他加入上市特管局。<br/>　　<br/>　　“一定要完成任务！”<br/>　　<br/>　　“是！保证完成任务！”<br/>　　<br/>　　宁永志这才放心，安排明天的伤者送去并将世兴文接回来。<br/>　　<br/>　　另外由于不放心，又给李普河打了个电话，交代了一下关于治疗保密的事情。因为要是被西市的李济深听到后，绝对会和自己抢人的。<br/>　　<br/>　　：。：<br/><br/><br/>Ps:书友们，我是小学嗣业，推荐一款免费小说App，支持小说下载、听书、零广告、多种阅读模式。书友们快关注起来吧！<br/><br/><br/>
