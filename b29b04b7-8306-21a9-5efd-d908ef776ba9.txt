　　木璃前脚刚走，苏莫正准备返回阁楼中，却又有两人来到了住处。
<br/>　　这两人，正是聂乾和韩盈。
<br/>　　“苏兄！”聂乾和韩盈来到苏莫身前，向他微微抱拳。
<br/>　　“你们怎么来了？”苏莫笑着问道，这两人还不错，他已经将两人看做朋友了。
<br/>　　“苏兄，我们都听说了你要和冷江约战的事情！”聂乾脸色凝重了下来，面带担忧的说道。
<br/>　　“进来说吧！”苏莫点了点头，招呼两人进了阁楼之中。
<br/>　　阁楼内，有小型的客厅，三人分宾主落坐。
<br/>　　“苏兄，一个月之后，你万万不能去应战！”刚一坐下，聂乾顿时急声说道。
<br/>　　“为何？”苏莫不解的问道。
<br/>　　“苏兄，诚然你实力很强，但根本不可能是冷江的对手！”聂乾告诫道，他可是了解冷江的实力。
<br/>　　“这你无需担心，是不是对手，要打过了才知道！”苏莫不在意的摇了摇头，他知道，估计所有神武学府的弟子，都不会认为他能抗衡冷江。
<br/>　　“苏兄你有所不知！”
<br/>　　韩盈开口了，俏脸凝重的说道：“冷江此人，虽然是武帝境三重的修为，但有接近武帝境四重的实力，非常的强大！”
<br/>　　“是吗？”苏莫闻言，也不禁心中微沉，那冷江有接近武帝境四重的实力？
<br/>　　“这还不是最主要的，最主要的是，这冷江心狠手辣，曾经两次在战界与人约战，最后都将对手彻底斩杀！”
<br/>　　韩盈叹了口气，继续道：“所以，你若是与他一战，只要战败，必然会陨落于战界！”
<br/>　　她和聂乾来此，就是要劝阻苏莫，放弃与冷江一战，免得白白丢了性命。
<br/>　　她们已经知道了事情的始末，冷江之所以要对付苏莫，就是上次魔云海敲诈之事。
<br/>　　所以，他们不能眼睁睁的看着苏莫死在冷江的手中。
<br/>　　“杀人的话，学府不会管？”苏莫问道。
<br/>　　“自愿在战界决斗，学府一般不会管！”韩盈道。
<br/>　　“原来如此！”苏莫点了点头，战界是神武学府弟子比斗的地方，看来也是一处充满杀机之处。
<br/>　　沉吟了片刻，苏莫目光凝视两人，沉声道：“二位放心，我心中自有打算，若是没有把握，到时候我不会去战界，你们不必担忧！”
<br/>　　聂乾和韩盈闻言，顿时松了口气。
<br/>　　他们都知道苏莫的武魂天赋，若是死了真是太可惜了！
<br/>　　沉吟一番，聂乾再次开口道：“苏兄，听说你刚来学府不久，还有一事，要提醒你一番！”
<br/>　　“何事？”
<br/>　　“神武学府内势力非常的复杂，各个派系之间明争暗斗，你最好不要卷入其中！“聂乾道。
<br/>　　“哦？都有什么派系？”苏莫好奇的问道。
<br/>　　“神武学府乃是由十三个超级大势力组建而成，其内自然也有十三个派系，目前学府十八位圣子之中，就有十三位圣子，各自分属一个派系，而冷江就是赤霞洞天派系的人！”聂乾徐徐说道。
<br/>　　“赤霞洞天！”苏莫闻言眉头一皱，这神武学府内的水，还不浅啊！
<br/>　　“加入派系的弟子，是不是将来就要加入背后的势力？”苏莫问道。
<br/>　　“不错，这十三个超级势力，每一代都会派不少超级天才进入学府，然后成为圣子，拉拢人心，那些投靠某位圣子的弟子，将来就会加入某个势力！”聂乾点头说道。
<br/>　　苏莫心中恍然，偌大的神武学府，培养的天才不计其数，十三个大势力每一个都想收拢更多的人才。
<br/>　　而这些人才在神武学府学成之后，都是自愿选择其中一个势力加入，如此的话，这十三个大势力自然要提前拉拢人心。
<br/>　　而作为那些导师的人物，自然不好明面上拉笼，不然学府就会乱了套。
<br/>　　所以，各大势力便让各自的超级天才，成为圣子，然后收拢人才。
<br/>　　“那还有五位圣子呢？”苏莫又问道。
<br/>　　“剩下的五位圣子，明面上不属于任何一个派系，所以将来会加入哪个势力，也说不一定！”韩盈说道。
<br/>　　“原来如此！”苏莫点了点头，这五位圣子天赋如此之强，不加入任何派系，估计各方派系肯定会争相拉笼吧！
<br/>　　当然，也有可能都暗中加入了某个派系，只不过明面上没有公开而已。
<br/>　　苏莫现在才发现，这神武学府之内，也是风起云涌啊！
<br/>　　若是他表现出了足够的天赋，估计也有人会来拉拢他。
<br/>　　“对了，那位至高无上的神子，是哪个派系的？”苏莫突然问道，神子可是要比圣子还要强大，还要崇高的存在。
<br/>　　“这一代的神子，不属于任何派系！”韩盈道。
<br/>　　苏莫闻言讶然，神子不属于任何派系，那十三个派系就热闹了！
<br/>　　神子，可是将来有希望达到虚神境的无敌天才，对于这种人物，十三个派系必定会疯狂的拉拢吧！
<br/>　　果然，只听韩盈唏嘘的说道：“因为神子没有加入任何派系，而其在学府已经待了二十五年，只有五年的学习时限了，这两年各大派系都急坏了！”
<br/>　　“去年，十三个大势力中，都有长老人物亲自来到学府，面见神子，发出招揽之意，但是神子并未明确表态！”
<br/>　　“若是再过两年，神子还是不表态，估计十几个大势力，就要开始下血本了！”
<br/>　　韩盈的面上露出羡慕之色，神子，因为天赋和实力，是整个学府最耀眼的存在。
<br/>　　每一代的神子，只要不是来自那十三个大势力，几乎都被十三个大势力捧在手心里，满足一切需求。
<br/>　　数百年前，曾经发生过为了争夺一位神子，十几个大势力不惜大打出手的局面。
<br/>　　苏莫闻言也是有些感慨，这才是无敌天才的待遇啊！
<br/>　　少倾，他摇了摇头，不再多想此事。
<br/>　　此事与他无关，他要考虑的是如何在一个月后，战胜冷江。
<br/>　　这冷江居然还是赤霞洞天派系的人，这可是有些棘手了。
<br/>　　随即，苏莫又向两人询问了一番赤霞洞天派系的情况，详细的了解了一番。
<br/>　　直到一个时辰之后，聂乾和韩盈才告辞离去。
<br/>　　随后，苏莫便也离开了阁楼，向南苑的交易场所走去。
<br/>　　他准备去看看，有没有兽魂等魂类宝物售卖，好提升武魂的等级。
<br/>　　与冷江之战，还有一个月的时间，他只能尽力去突破武帝之境了，不然的话绝对是败多胜少。
<br/>(本章完)
