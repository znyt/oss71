        “星儿，谁给你打的电话啊？你脸色这么难看？”顾千凝见连星儿走到一旁看着手机发呆，便关问道。
<br/>
<br/>        连星儿回过神来，微微耸了耸肩，没好气地回答：“一个发酒疯的人。”
<br/>
<br/>        还有，她什么时候花心了？
<br/>
<br/>        明司寒那家伙，真是的……
<br/>
<br/>        连星儿刚准备收起手机，手机铃声又响了起来。
<br/>
<br/>        是一组陌生的来电号码。
<br/>
<br/>        连星儿犹豫了一下，才接听了电话。
<br/>
<br/>        “你现在来临海市生态公园，郎安受伤了，我会发定位给你。”
<br/>
<br/>        “你是谁？”
<br/>
<br/>        “裴萱妍。”
<br/>
<br/>        裴萱妍说完后便把电话给挂断了。
<br/>
<br/>        连星儿随后收到了裴萱妍发过来的定位信息。
<br/>
<br/>        然后，她和室友们撒了个谎，说自己回祖爷爷家，便离开了寝室。
<br/>
<br/>        顾千凝她们对她突然离开寝室，也未怀疑，三个女生依旧凑在一台笔记本电脑前继续追剧。
<br/>
<br/>        连星儿搭了一辆出租车，按着手机上的路线提示，赶到了生态公园的后门。
<br/>
<br/>        她下车后，出租车扬尘而去，而立在她面前的后门是铁栅栏门，还上了锁。
<br/>
<br/>        连星儿下意识地环顾四周，见此刻无人，于是攀上铁栅栏门上的铁杆，翻了过去。
<br/>
<br/>        进了生态公园里后，她再次掏出手机，一边看路线提示，一边往一片高高的灌木丛里走去。
<br/>
<br/>        她拨开一层又一层的条状叶子，光顾着看前方去了，而忘记了脚下。
<br/>
<br/>        连星儿打了个趔趄，差点被什么东西绊倒在地。
<br/>
<br/>        当她打开手机灯，照过去时，发现是一只被开膛破肚了的小羊羔，心里“咯噌”一声，浑身起了鸡皮疙瘩。
<br/>
<br/>        就在这个时候，身后传来兽物的奔跑声。
<br/>
<br/>        连星儿冷不丁地打了个寒颤，怔愣地转过身去后，看到了一双金色的兽眸。
<br/>
<br/>        她瞬间瞪大了双眼，眼睁睁地看着一双硕大的兽爪向自己扑了过来。
<br/>
<br/>        连星儿被这双兽爪按倒在地，吓得闭上了眼睛，脑海里瞬间回忆起了在外公的实验室门外的那次，本能地唤了声：“郎安。”
<br/>
<br/>        而她的这一声呼唤，恍若令周遭都安静了一般，连风吹草动的声音都消失了。
<br/>
<br/>        连星儿怯懦地睁开双眼，月光的照耀下，她见到了郎安那张英俊又邪肆的脸孔。
<br/>
<br/>        不知不觉中，她忽然感到肩头和胸前一片凉意。
<br/>
<br/>        郎安的那双眼睛依旧发着金色的微光，不经意间地埋下头来。
<br/>
<br/>        此刻，连星儿只觉两道刺痛感从肩窝处传来，郎安的脑袋紧紧地贴在了她的脸颊上。
<br/>
<br/>        她双目呆滞地凝望着夜空，恍若看到了在云层中漫游的满月。
<br/>
<br/>        不知过了多久，连星儿感到有些头晕眼花，慢慢地失去了意识。
<br/>
<br/>        裴萱妍将明司寒从KTV包厢里带出来后，送到了他在临海城里所买的单身公寓里。
<br/>
<br/>        她还给他喂了特效醒酒药，不过一分钟，他便跑到厕所里吐了个干净。
<br/>
<br/>        明司寒吐完后，洗了把脸，又漱了个口，顿时感觉整个人要清醒了很多。
<br/>
<br/>        “那个郎安，压根就不是连星儿的贴身保镖。”裴萱妍倚着洗漱间门框，双手抱臂，若有所思地说道。
<br/>
<br/>        明司寒回头看了裴萱妍一眼，反问：“不是又能怎么样？”
<br/>
<br/>        “不是的话，就糟糕了啊！”裴萱妍啧了啧嘴。
<br/>
<br/>        明司寒顿时一脸不悦地皱起了眉头。
<br/>
<br/>        裴萱妍接着说道：“你最好劝连星儿离他远点，不然，免得到时候，连星儿为了他，跟你反目成仇噢！”
<br/>
<br/>        “你待在我身边也有这么久了，除了让我答应你的上司加入你们，到底还有什么目的？为什么，我感觉，你很忌惮郎安？”明司寒有些咄咄逼人。
<br/>
<br/>        裴萱妍很不喜欢他这副样子，于是故意跟他撒起娇来：“我待在你身边的原因，你不是早就知道了吗？再说了，若不是上司的命令，我也不想成天这么盯着你。不仅要保护你，还得被你使唤。切，你以为我愿意啊？”
<br/>
<br/>        “既然，你说郎安不是星儿的贴身保镖，那么他又是谁？为什么会跟在星儿身边？星儿又为什么会护着他？”明司寒质问道。
<br/>
<br/>        裴萱妍嘴角微扬：“只要你答应进我们的组织，我就告诉你郎安和连星儿之间的秘密。”
<br/>
<br/>        “算了，我直接去问星儿。”明司寒淡然道。
<br/>
<br/>        裴萱妍却笑了笑：“你问她，她是不会告诉你的。在她眼里，郎安是她自己人，你明司寒只不过是个外人。”
<br/>
<br/>        “你今晚很奇怪，特意跑来找我，就是为了说这些废话？”明司寒嗤之以鼻。
<br/>
<br/>        裴萱妍故作淡定地摸了摸肚子，只身往那边开放式厨房里走去：“我只是肚子饿了，想来这儿自己动手做点儿吃的。你想吃什么？”
