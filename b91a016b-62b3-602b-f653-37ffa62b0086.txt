这份营养餐是吴妈做的，现在厉嫣然说的好像是她亲自下厨了一样，在陆寒霆面前邀功。

<br/>        吴妈并没有说些什么，女孩子的小心思她看得出来，而且厉嫣然已经蛮有心的了，当然，她还是没有办法跟少奶奶相比。

<br/>        吴妈又想起了夏夕绾，如果少奶奶还在就好了…

<br/>        “那我们一起上楼去看奶奶吧。”
<br/>
<br/>        “好啊。”厉嫣然心花怒放。

<br/>
<br/>        “奶奶，你醒了？”陆寒霆迅速将老夫人扶了起来，在她后面垫了一个软垫，“嫣然过来看你了。”

<br/>        “陆奶奶，”厉嫣然乖巧的上前，“我早就想过来看你了，好久不见，我都想你了，听说你胃口不好，我给你量身定制了营养餐，你多少要吃一点哦。”

<br/>        陆老夫人看着厉嫣然，“原来是嫣然丫头啊，你费心了。”

<br/>        “陆奶奶，你不要跟我客气，来，我喂你喝点汤。”厉嫣然坐在床边，拿着汤匙舀了一小口的汤吹了吹，然后送到陆老夫人的嘴边。
<br/>
<br/>        陆老夫人张嘴，配合的将汤喝了下去。
<br/>
<br/>        厉嫣然刚开心，正准备喂第二口的时候，老夫人一弯腰，直接将那口汤吐了出来。
<br/>
<br/>        “老夫人！”吴妈迅速上前给老夫人拍背。
<br/>
<br/>        厉嫣然没想到老夫人现在对食物的反应已经这么大了，本来她还踌躇满志的用自己的营养餐来治好老夫人，在陆寒霆面前好好露面的。
<br/>
<br/>        陆寒霆用温热的毛巾帮老夫人擦嘴，老夫人止住了呕吐和咳嗽，整个人异常疲倦，她挥了挥手，“阿霆，你送嫣然丫头离开吧，我什么都不想吃，想休息了。”
<br/>
<br/>        陆寒霆不敢违逆，他轻声道，“奶奶，那我先送嫣然离开。”
<br/>
<br/>        两个人离开了房间，厉嫣然失望委屈的看着陆寒霆，“寒霆哥哥，对不起。”
<br/>
<br/>        陆寒霆抿了一下薄唇，他神色很淡的开腔道，“不是你的问题，这些日子来的厨娘做的营养餐，奶奶都是吃了一口然后吐了，我先送你回家吧。”
<br/>
<br/>        ……
<br/>
<br/>        房间里，吴妈给老夫人擦手，“老夫人，你一定要振作起来，你看少爷每日陪侍床前，无比记挂担忧你的身体，不停想办法想要让你进食，你怎么舍得丢下少爷？”
<br/>
<br/>        老夫人叹息了一声，“吴妈，这些天的晚上我总是做梦，我梦到…绾绾了…”

<br/>

<br/>        吴妈就知道老夫人这个是心病，吴妈的眼眶也变红了，她只能握紧了老夫人的手，“老夫人，这世上没有如果啊，如果少奶奶还活着，她看到你这样，会第一个心疼的…少奶奶是一个好孩子，她真的很好很好…”
