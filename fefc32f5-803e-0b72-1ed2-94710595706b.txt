　　楚风眠又斩杀三人，这三人身上的精血，如今全都被楚风眠直接吞噬，楚风眠的气势，可以说是越战越勇。
<br/>　　三尊魔尊，都被楚风眠一剑之下所斩杀。
<br/>　　“此子的实力，怎么可能恐怖到这一步，他还仅仅只是生死境四重的境界？”
<br/>　　小魔主看着楚风眠的剑光，心中都是忌惮无比。
<br/>　　楚风眠的境界，仅仅只有生死境四重的境界，便是有着如此可怕的实力，这也就意味着，楚风眠的实力还没有远远达到极限。
<br/>　　若是这样，让楚风眠将境界提升到生死境巅峰的境界，只怕是超越生死的人物，都无法击败楚风眠了。
<br/>　　“天魔祭炼！”
<br/>　　小魔主眼神一冷，突然冰冷的怒喝一声。
<br/>　　只见他坐下的天魔宝车，其中的八位拉车天魔，突然在他的一声怒喝之下，纷纷燃烧起来。
<br/>　　这八位天魔，被小魔主突然祭炼。
<br/>　　这每一尊天魔的实力，都不亚于一尊生死境巅峰的武者，如今八尊天魔全部祭炼，也就相当于八位生死境巅峰的武者，一同祭炼。
<br/>　　每一位生死境巅峰的武者，所蕴含的力量都是无比巨大的，更何况这样一下子祭炼八位。
<br/>　　天魔宝车上的灵力，顿时变得澎湃起来，天魔宝车，以他现在的实力，还远远无法动用全力，所以如今他便是祭炼天魔，来爆发出更强的力量。
<br/>　　“楚风眠，死吧！今日就算是手段通天，也难逃一死！”
<br/>　　小魔主大声咆哮一声，他坐下的天魔宝车，便是化为一道急速的遁光，直接冲着楚风眠撞了过来，目标直击楚风眠的胸口。
<br/>　　“群魔乱舞！”
<br/>　　其他的魔尊，也是随着小魔主一起，同时出手，无数武技，四面八方之间便是向着楚风眠轰击过来。
<br/>　　楚风眠眼神看向这天魔宝车冲击过来，脸色也是严肃无比，眼神认真。
<br/>　　他就算是可以轻视小魔主，也不敢轻视这天魔宝车，曾经在围攻剑道之主的时候，当时的魔主便是利用这天魔宝车，无数次的将剑道之主逼入绝境。
<br/>　　天魔宝车，绝不亚于天下间的任何一件天级灵器，就算是楚风眠的祖龙至尊剑，也只能是跟他平起平坐，而无法超越。
<br/>　　眼下小魔主祭炼天魔，简直便是将天魔宝车的所有力量，在瞬间爆发了出来。
<br/>　　这一击，是绝对的绝杀。
<br/>　　“造化！”
<br/>　　楚风眠轻喝一声，在他身躯之中，那本命剑元的力量，都被直接催动起来，一股股造化之力，围绕在楚风眠的剑锋之上。
<br/>　　楚风眠眼神中凶光一闪，他的剑锋，直接冲着天魔宝车，斩了过去。
<br/>　　小魔主看到楚风眠的举动，眼神中都是露出了几分冷笑。
<br/>　　敢跟天魔宝车硬抗，楚风眠，必死。
<br/>　　八尊天魔献祭的力量，足以是让天魔宝车，瞬间爆发出全力，任何强者，都不想要抵挡得住。
<br/>　　“轰！！！”
<br/>　　这一道剑锋，跟天魔宝车的光芒，猛然碰撞的一刻，便是响起惊天雷鸣声。
<br/>　　这一股灵力冲击，猛然炸裂开来，令周围的无数武者，都是纷纷退后。
<br/>　　同时漫天武技，也是在这瞬间，全部落下，令这场面更是变的混乱了许多。
<br/>　　“轰隆！”
<br/>　　每一道武技轰击落下，都是掀起一阵轰鸣声，无数魔尊的联手之下，所有武技，都已经是向着楚风眠轰击过来。
<br/>　　只见这整个宫殿，如今在这巨大无比的爆炸之下，都开始晃动起来，这曾经云羽天宫建造的宫殿，都有些承受不足如此攻击。
<br/>　　宫殿都是如此，更何况是在这无数力量中心的楚风眠。
<br/>　　这所有的力量，都是轰击在了楚风眠的身上，楚风眠所承受的力量，比起这余波还要恐怖十倍，百倍。
<br/>　　天魔宝车，献祭天魔，无数魔尊联手之下，为的就是要轰杀楚风眠，所展现出来的力量，只能够是以恐怖来形容。
<br/>　　“这样的攻击之下，此子就算是隐藏了再多的实力，都要必死无疑了吧。”
<br/>　　一尊魔尊，看着眼前他们轰击的地方，如今已经是一团烟雾，看不清其中。
<br/>　　只是在这样的攻击之下，他们也不相信，楚风眠居然真的可能会还能活下来。
<br/>　　在这样的攻击下，别说是尸体，只怕是任何一丝一毫东西，都不可能留得下来，只有可能会被直接全部泯灭。
<br/>　　小魔主看着烟雾，许久之后，脸色的表情才是轻松一些。
<br/>　　这样的攻击下，换做他，就算是有着天魔宝车的他，受到如此轰击，都会当场陨落，身形俱灭，不会留下任何一点东西。
<br/>　　楚风眠就算是在妖孽，在可怕，天资在高，也终究是一位天才，而并非是强者。
<br/>　　在如今他们联手，小魔主布置了如此之多的算计之后，楚风眠终究还是要陨落了。
<br/>　　楚风眠的死，令小魔主都有些庆幸，他的资质，在九魔城中，甚至是在整个九域，都足以称得上是天下无双。
<br/>　　不然的话，九魔城也不可能将他捧上小魔主的位置上，可如今他看到楚风眠，甚至都感觉有些恐惧。
<br/>　　楚风眠的资质，的确是太过妖孽。
<br/>　　这样的妖孽，不死，将来必成大祸。
<br/>　　“你们去看看，这小子就算是陨落，他手中的祖龙至尊剑也不会泯灭。”
<br/>　　小魔主看着身边的一位魔尊，开口道。
<br/>　　祖龙至尊剑，天地之中能够毁灭这种灵器的存在，还从来没有。
<br/>　　“是。”
<br/>　　旁边那位魔尊，听到小魔主的话，脸上露出几分欣喜，直接向着烟雾走了过去。
<br/>　　楚风眠就算是陨落，他身上的奇遇也会有着不少留下来的。
<br/>　　楚风眠以生死境四重的境界，就足以跟他们这么多人联手之下抗衡这么久，整个九域都找不出这样一个人物。
<br/>　　谁可以得到楚风眠身上的机遇，也许便是可能成长成为下一个楚风眠。
<br/>　　“嘶！”
<br/>　　就在这魔尊，靠近烟雾的一刻。
<br/>　　一道剑光，突然从烟雾中闪耀出来，在这一抹剑光之上，眼前这魔尊，顿时身躯都被直接搅碎。
<br/>　　同时一道身影，如同一把利剑一般，从这烟雾中，突然冲了出来。
