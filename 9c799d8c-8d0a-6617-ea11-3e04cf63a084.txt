　　狼牙基地在遭受巨大毁坏后，很快安罗城便出动了一批队伍前来补救，当看到满目苍夷的战后场景时，众人在很多天后还心有余悸。
<br/>　　基地的重新补救只是一部分，更多的是诸多情报和狼牙防御能力的崩盘，现在所有人都明白在神秘家族的面前，狼牙所有的防御系统都毫无用处。
<br/>　　狼牙墓地完好无损，这里埋葬的尸骨还在安睡，陆玟丞将一束花放在了一座墓碑前，望着眼前这块被记满了文字的石碑，良久都说不出话。
<br/>　　石碑的上方中间刻着墓碑主人的名字——
<br/>　　顾凉晨。
<br/>　　名字下方排满的小字，比旁边的人都要多，似乎都快要挤不下，连刻墓碑的老匠都不断感叹，说这位名不见传的少将竟然有如此传奇的一生，也可惜的英年早逝。
<br/>　　“中将。”巨大墓碑处的一人朝陆玟丞开口：“少将一排的名字第一格已经有人了。”
<br/>　　陆玟丞头也不抬：“那就刻在中将一列里，他在长官那里早就是中将了。”
<br/>　　“陆中将。”此时有一人前来，汇报道：“准备好了。”
<br/>　　陆玟丞点点头转身走向大门处，那里已经聚集了大批的群众及狼牙防守队成员，这些都是狼牙地界现存不多的兵力。
<br/>　　而这些人聚集在这里的原因，很大一部分原因就是顾凉晨的事迹终于被曝光。
<br/>　　一个一直默默隐藏在暗处，为狼牙做出了无数贡献的人，却只在死后被世人知晓，将一生的荣耀带进了棺材。
<br/>　　所以很多人聚集在此，自发的送顾凉晨一程。
<br/>　　墓园开放，一批又一批的人排着队进入，这是狼牙极少数的一次对外开放墓园来让民众悼念狼牙英魂，场面很壮观却安静。
<br/>　　人们排着队来到顾凉晨的墓碑前，将一支支鲜花放下，细细记住顾凉晨的一生，将这位之前并没有听说过的少将事迹熟知。
<br/>　　而后再从旁边的第一座墓碑开始，依次延着小道走过，每路过一座墓碑，他们都会深深鞠躬，仔细读着其上的文字，记住这些为了狼牙而死去的英雄们。
<br/>　　每一次的狼牙墓园开放，场面都是如此，甚至还有很多人在走出墓园后痛哭很久，这座墓园里或许还埋葬着他们的亲人。
<br/>　　看着这样肃穆的场景，陆玟丞忍不住扭过头不忍再看，只是在他刻意远离的时候，却看到对面一个情报部成员小跑着而来。
<br/>　　陆玟丞快步走上前：“怎么了？”
<br/>　　那名情报部成员气喘吁吁，却双眼发亮的开口：“暗部传来消息，曹氏研究院可以突破了！”
<br/>　　陆玟丞瞳孔剧烈收缩，立即往回走，同时快速下令：“召集一百人手立即出发，协助暗部攻破曹氏研究院！”
<br/>　　“是！”
<br/>　　很快，狼牙所剩不多的直升机全部起飞，陆玟丞亲自带队，将一百名防守队的精英带着，火速向着曹氏研究院而去。
<br/>　　“最后一次确定。”即将到达时陆玟丞在对讲机内严肃开口：“位置？”
<br/>　　“十二点钟方向一千米，一切准备就绪。”
<br/>　　陆玟丞心中紧张，道：“暗部那里？”
<br/>　　“正在尝试交接。”对讲机内传来一名情报部成员的声音，很快‘滴滴’两声后，声音再次响起：“交接完毕，一分钟后曹氏研究院大门开放，里面没有敌人。”
<br/>　　陆玟丞很激动，双眼放光：“很好，这一次一定要将所有人都救出来！”
<br/>　　一分钟后，呼啦啦！
<br/>　　直升机全部降落，曹氏研究院那个从地下而来的升降机也准时的来到地面，陆玟丞紧紧握着佩刀，身后的一百名防守队精英也全部武装精良，大步走向了升降机。
<br/>　　虽然内部的暗部成员传来没有敌人的消息，但他们还是全副武装，将这次任务看的尤其重要。
<br/>　　升降机一层层下潜，足以见得这个研究院建立在多深的地下，很快陆玟丞等人就来到了最底层，旁边已经有两名暗部成员在等待。
<br/>　　“末世里的天空为什么带着红？”
<br/>　　“番茄你个西红柿。”
<br/>　　快速的暗语对接过后，两名暗部成员都很激动，用不是很标准的军礼朝着陆玟丞敬礼。
<br/>　　陆玟丞回礼，心中焦急：“有话回去再说，先带把人都救出去。”
<br/>　　一名暗部成员快步走在前面带路：“中将跟我来，我们事先筛选过一批人让他们准备好，另外的人还不知道情况，到时候强行带走。”
<br/>　　“好，你们分成两队把人带出去。”陆玟丞快速安排着，而后面带急色的四处看：“老楚先生在哪里？”
<br/>　　此话一出，前方的暗部成员顿时面色一变，道：“很抱歉，目标人物不在这里，他被罗杰带走了！”
<br/>　　陆玟丞大惊，猛然察觉到有什么事不对劲，同时一股挫败和无奈感涌上心头，一下子好似泄·了气的皮球，声音也忽然低沉了下来。
<br/>　　“其他人情况，胡梦浩怎么样了？”他再次问道。
<br/>　　“胡梦浩是我们的保护对象，一切安好，我马上带您过去见他。”暗部成员加快脚步，带着陆玟丞等人来到了研究院深处。
<br/>　　很快前方的门被打开，一个坐在轮椅上的年轻人呈现在陆玟丞的面前，正是胡梦浩。
<br/>　　陆玟丞刚想说什么却忽然愣住，看着结巴了一下：“您，您这是……”
<br/>　　这是他第一次见到胡梦浩，知道对方双·腿残疾，但却不知道对方这么古怪。
<br/>　　眼前是一张长桌，其上摆着各种研究器皿，此时全部被胡梦浩用一种绿色的液体浸满，旁边还有一个大包裹，显然是胡梦浩打包要带走的东西，而整个研究室此时都空空荡荡，被他洗劫一空。
<br/>　　“这些都带走，来帮我提着。”胡梦浩一点不客气，头也不抬的朝陆玟丞使唤：“桌上这些带不走，我还差一点就洒完了，半小时后这里就会被毁。”
<br/>　　陆玟丞吓了一跳，心想这研究人员就是诡异，连忙上前将那一大包裹背起，而后站在门外等着。
<br/>　　很快胡梦浩就坐着轮椅出来，小心的将门关上后擦了擦汗，这时他才抬眼看向陆玟丞：“好了，我们快走。”
<br/>　　陆玟丞不敢多呆，带来的防守队精英都已经将其他人带上了地面，眼下只剩下他、胡梦浩和两名暗部成员。
<br/>　　四个人快速走上升降机，期间胡梦浩不断催促着让大家快些，一开始陆玟丞还不是很懂为什么，可当他们刚刚来到地面时——
<br/>　　轰！
<br/>　　一声闷响，就这么在地下爆开。
<br/>　　紧接着‘轰轰轰’连续好几声，一连串的闷响不断爆炸！
