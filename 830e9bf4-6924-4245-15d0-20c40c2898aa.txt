        “我的私事，我自有分寸。至于苏漫雪，你找个机会接近她，给她提个醒。让她好自为之。”傅夜沉叮嘱道。
<br/>
<br/>        夏玄炘点了点头，随即从沙发上站起来，从手提包里掏出笔和纸来，留下了自己的银行账号，放在了茶几上：“封口费麻烦夜沉哥打入这个账号里，聊到这里，我也该走了。祝你好运。”
<br/>
<br/>        “谢谢。”傅夜沉礼貌性地迎了迎手，“慢走，不送。”
<br/>
<br/>        夏玄炘这次过来，无非是想拿筱颖的事情，从他这里捞一笔巨额零花钱罢了。
<br/>
<br/>        至于筱颖的真实身份，夏玄炘的揣测是否是真的，他傅夜沉真的从未去核实过。
<br/>
<br/>        因为，不管结果如何，都不影响他和筱颖在一起。
<br/>
<br/>        唯独受影响的人，只怕是他的弟弟傅昼景了。
<br/>
<br/>        如果，真如夏玄炘所推测的那样，筱颖是夏海东和龙沉香的女儿，那么筱颖岂不是阿景的小姨，就算筱颖不是夏海东和龙沉香的女儿，而是他傅夜沉的干爹的女儿的话，筱颖就成了阿景的表妹。
<br/>
<br/>        所以，真正接受不了这层关系的人，应该是傅昼景才对。
<br/>
<br/>        傅夜沉心里刚想到傅昼景，落以柔送走了夏玄炘又来通报说：“傅总，您弟弟在楼下，说有事找您。”
<br/>
<br/>        “让他上来。”傅夜沉还真没料到，这是“想曹操，曹操就到”。
<br/>
<br/>        只是傅昼景这次主动找过来，不知道是为了什么事情？会不会也是筱颖的事？
<br/>
<br/>        傅夜沉特意坐在了待客区的沙发上，让落以柔泡好了茶，等着傅昼景进门。
<br/>
<br/>        傅昼景进来后，看到哥哥傅夜沉从容淡定的模样，心里顿时也没了底，走过去后，强颜欢笑地将手里的请柬递给了傅夜沉。
<br/>
<br/>        傅夜沉坐在单人沙发里，敞着双腿，颀长冷峻的身躯优雅地靠着沙发背。他一只修长洁净的手抵在下颚边，另一只手则悠悠然地接过傅昼景手中的请柬。
<br/>
<br/>        傅昼景随之在一旁的沙发上坐下，开门见山地说道：“大哥，这是我和思妍的婚礼请柬。”
<br/>
<br/>        “恭喜。”傅夜沉淡漠道。
<br/>
<br/>        “筱颖是不是又回到大哥你的身边去呢？”傅昼景按捺不住地问。
<br/>
<br/>        昨天，他开车路过市中心，看到童筱颖打扮精致地上了一辆满载紫玫瑰的敞篷跑车。
<br/>
<br/>        而开跑车的男人虽然戴着墨镜，穿着休闲装，但是他一眼就认出来，那个男人就是他的大哥傅夜沉。
<br/>
<br/>        是不是所有女人都如此爱慕虚荣？
<br/>
<br/>        就连她童筱颖也一样。
<br/>
<br/>        童筱颖可以一次又一次地拒绝他傅昼景的求复合，可为什么却选择一次又一次地回到他的大哥身边去？
<br/>
<br/>        大哥压根就不缺女人，哪怕大哥嫌弃她童筱颖，最后还抛弃了她童筱颖，可童筱颖那蠢女人竟然又只身前往大哥的身边？
<br/>
<br/>        傅昼景想不明白，他的大哥傅夜沉，除了有钱，好像也没有别的值得一个女人去反复留恋的了！
<br/>
<br/>        归根究底，童筱颖也不过是个世俗的女人，喜欢钱，想当豪门富太。
<br/>
<br/>        傅昼景抿了抿唇，眼神里透着一丝狭隘的目光，接着说道：“大哥，你可能不知道吧！筱颖的那层膜是补的！大哥，你真的一点都不介意吗？我们俩是亲兄弟啊！居然共用了一个女人！”
<br/>
<br/>        “啪”地一声，一个迅雷不及掩耳之势的巴掌，掴在了傅昼景的脸颊上。
<br/>
<br/>        傅夜沉不知何时已经从单人沙发上站了起来，一把揪住了傅昼景的衣领，将他提到了自己的眼皮子底下，疾言厉色道：“阿景，是我将筱颖强行留在自己身边的！三年前，如果不是因为我让着你，筱颖早就是我的女人了！”
<br/>
<br/>        “大哥你这话是什么意思？”傅昼景惊愕道。
<br/>
<br/>        傅夜沉黑眸里瞬间布满了血丝，眉宇间透着一丝犀利与锋冷：“三年前，救我的那个女人不是苏漫雪而是筱颖，我对筱颖一见钟情，我想娶她为妻，为了娶她，我答应爷爷退役。但是，当我发现刘叔接错人时，筱颖已经是你傅昼景的妻子了。你知道，那个时候，我的心有多痛吗？”
<br/>
<br/>        “我不信！筱颖从未跟我提及过，是她救了你！”傅昼景咬牙切齿道，甚至他们一起去了澳洲后，筱颖都从未主动跟他说过大哥的事情。
<br/>
<br/>        在他眼里，筱颖和大哥之间，除了跟他一样的亲情之外，就没有任何交点了。
<br/>
<br/>        傅夜沉不禁疾言厉色地反驳：“那是因为，筱颖一直以为，我误把苏漫雪当成了那晚救我的人，她为了你，从未想过要主动来跟我解释说我弄错了救命恩人。她从未在意过我！更不屑当傅家的大少奶奶！”
