　　虚空海，并不在北邙学院之中。
<br/>　　而是在偌大北域的一处极为神秘的空间之中，寻常人想要进入虚空海，只怕都找不到其中的道路。
<br/>　　不过好在，在北邙学院中，有着记载虚空海的地图，再加上白萱的指引，两人便是在北域中飞行，足足是连续飞行了一日时间后，才来到了一处平静的荒原之中。
<br/>　　在这片平原荒原之中，二人打破空间，便是来到了一处截然不同的虚空之中
<br/>　　谁能想到，这看似平静的荒原背后，居然便是隐藏了北域最为神秘的地方之一，虚空海。
<br/>　　不同于寻常进入虚空，寻常进入虚空，其中的空间的大小，便是以武者灵力的大小缔造的。
<br/>　　但是眼前的这虚空海，一眼望去，却是看不见尽头，庞大无比，根本便是天然缔造而出的一片空间。
<br/>　　其中的空间乱流，也要强大的多，没有寻常巅峰圣者的实力，在这虚空海中，甚至就连空间乱流的无法抵抗。
<br/>　　只有实力达到寻常巅峰圣者之上，才能够在这虚空海中探索。
<br/>　　“在这虚空海中，危险无比，无数虚空乱流，必须要远远不断的利用灵力来抵挡，我们还是先找个地方休息一下再说。”
<br/>　　来到虚空海，白萱的表情变得凝重许多，张望了一眼，便是带着楚风眠，向着一个方向飞去。
<br/>　　虚空海中，虽然有着无数奇遇，但是其中的危险，也要远远超越北域，稍微不注意，就算是生死境强者，在其中，都会陨落。
<br/>　　楚风眠跟随者白萱，一同飞去，对着虚空海，楚风眠还是初来乍到，所以还是跟随白萱比较好。
<br/>　　两人在虚空海中，小心翼翼的飞行了一阵，在楚风眠的面前，便是一座巨大无比的岩石，突然出现了。
<br/>　　在这岩石上面，居然是建立了不少客栈的样子，一眼望去，其中上面至少有着上千名武者的存在。
<br/>　　“这里是？”
<br/>　　楚风眠有些惊讶的开口道。
<br/>　　他也想不到，在这危险无比的虚空海中，居然还有着客栈，这样休息的地方。
<br/>　　在其中，无数武者在其中休息这，大吃大喝，甚至是都有些店铺，来让武者进行买卖，俨然也是一座繁华小城了。
<br/>　　“这是虚空海中的无数聚集地之一，星天峰，这块陨石，乃是虚空海中的特产，足以抵抗虚空海中的虚空乱流。”
<br/>　　“一些武者，便是借助这块陨石，在其中搭建客栈，茶楼，甚至是在其中做起了生意。”
<br/>　　白萱看着楚风眠惊讶的表情，笑着解释道。
<br/>　　若是没有来过虚空海的人，都会有些不相信，在这样危险的地方，还有可以休息的庇护之地，白萱第一次前来的时候，也是一样惊讶。
<br/>　　“这虚空海越是危险，就越是需要这样的庇护之地，在这里面，可以休养生息，确保安全，我们先过去。”
<br/>　　白萱说着，便是直接向着那星天峰落了下去，楚风眠紧跟其后。
<br/>　　“来者止步！”
<br/>　　刚刚降落在了那星天峰上，两尊武者，便是突然飞了过来，这两尊武者的实力，也比起寻常巅峰圣者，要强大不少，至少跟那拓跋鼎足以抗衡。
<br/>　　在这虚空海中，再弱小的存在，都是有着巅峰圣者的实力。
<br/>　　看到这两尊武者的突然出现，白萱倒是随意笑了笑，显然是习惯了，她手心一动，一颗金色拳头大小的奇物，便是直接递了过去。
<br/>　　这奇物，名为金沙石，在诸多奇物中，虽然只能算是中下等，但是放在外面，卖几百万九鼎丹还是可以的。
<br/>　　没想到在这虚空海中，最为简单的交易，便是需要奇物。
<br/>　　“这虚空海中的武者，最弱的都是圣者巅峰，实力达到生死境的强者，都是比比皆是，甚至一些生死境高阶的大人物，都会出现在虚空海中。”
<br/>　　“对这样的强者来说，寻常的九鼎丹等，自然没有任何的意义了。”
<br/>　　“所以在这其中最为简单的交易，便是需要靠奇物了。”
<br/>　　“能够实力达到巅峰圣者的地步，谁没有几个不用的奇物，也可以一物换物，换取需要的东西。”
<br/>　　白萱出言解释道，楚风眠听到，也是点了点头。
<br/>　　他也是第一次来到这虚空海中，对着其中的无数规则不知道，白萱的解释，倒是让他明白了许多。
<br/>　　的确，九鼎丹这种丹药，对楚风眠来说，也已经没有任何的帮助了，这些九鼎丹，楚风眠便是准备去换取其他对楚风眠有用的奇物了。
<br/>　　“原来是北邙学院的白萱姑娘。”
<br/>　　那两尊武者，似乎是认得白萱，打了个招呼。
<br/>　　“这星天峰的规矩再次，我等二人也是奉命行事，还请白萱姑娘不要见怪。”
<br/>　　“自然，星天峰的规矩我懂，这一共有着两块金沙石，就算是我们二人的费用好了。”
<br/>　　白萱一连拿出两块金沙石，递了过去，同时对着楚风眠小声说道。
<br/>　　“楚师弟刚刚加入到北邙学院中，这一次就由我来付了。”
<br/>　　“恩？”
<br/>　　那两尊武者，看了一眼楚风眠这个陌生面孔，打量了一眼楚风眠的境界，眼神中都是闪过一丝不屑。
<br/>　　神力境四重的境界，就来闯这虚空海，根本就是在找死，不过人是白萱带的，只要是缴纳奇物，就算是一个没有修为的普通人前来，都没有关系。
<br/>　　只是死在虚空海中，却是跟他们星天峰没有关系了。
<br/>　　“白萱师姐已经帮了我不少次了，这一次哪里需要劳烦白萱师姐。”
<br/>　　楚风眠突然伸出手来，拿出了一块黑色金属，抢在白萱前面，递给了那两尊武者。
<br/>　　“这是黑玄金，价值已经超过了两块金沙石，就算是我们二人的费用吧。”
<br/>　　“黑玄金？”
<br/>　　看着楚风眠递过来的一块黑色金属，那两尊武者的脸上，都微微露出几分惊讶。
<br/>　　黑玄金的价值，可要比起金沙石高上不少，而且比起金沙石来说，黑玄金还有着一大好处，便是通用。
<br/>　　许多奇物虽然价值奇高，但是却并不是人人需要的，就算是有着价值，也很难碰到合适的买家。
<br/>　　但是这黑玄金在炼制无数灵器的时候，都可以加入其中，所以根本不愁销路。
<br/>　　论起价值，绝对是要比起金沙石高的多。
