        &lt;h3&gt;第三卷第四百零一章  好消息不断&lt;/h3&gt;
<br/>
<br/>        江俞轩在西城高新的项目招标众多的方案中脱颖而出，这个是大项目，又关系着整个高新区后期的整体规划，江俞轩他们的方案被众多专家看好，以最高评分标准拿下了高新的的项目。
<br/>
<br/>        从招标现场出来，江俞轩对姐姐江俞琪说：“这个项目真的是让我们费了一番心思，放眼之前的项目，我们姐弟哪有同时出现在招标现场啊。”
<br/>
<br/>        “轩轩，说实话，我也害怕你最近不能好好工作，看来我的担心还是多余的。”江俞琪怕江俞轩为情所困，会耽误工作。
<br/>
<br/>        “你想多了，我只有让自己越来越强大，才能得到自己想要的，工作对我来说就是变得强大的唯一通道。”
<br/>
<br/>        江俞琪不知道的是，因为陈子昂的邮件安慰了江俞轩的心，让江俞轩在听到了她的一番话之后更加坚定了想要和陈子昂在一起的决心，就像他自己说的，他要越来越强大才能得到他想要的，陈子昂急匆匆的离开，让江俞轩猜到了陈子昂的处境，江俞轩决定，从现在起，他要团结他所有的力量，努力成长，在不久的将来能给陈子昂一个安全的避风港。
<br/>
<br/>        在陈子昂离开的这一段时间里，他既不联系张倩楠也没有联系陈子昂，他沉下心来研究当前的局势，研究当前的经济。
<br/>
<br/>        他似乎超然事外，和他们任何一方都不进行联系，包括郝景文来西城，他都一门心思的扑在江家的工作上，在这一段时间里，静下心来将高新的方案过了一遍又一遍，而陈子霖和易倾秋也知道江俞轩在忙，所以大家都没有打扰他。
<br/>
<br/>        从招标现场出来的时候，杨东给他打电话：“俞轩，怎么样？”
<br/>
<br/>        “我感觉结果应该和我预期的一样，老头子，你放心吧。我一定会让你满意的。”江俞轩抬头看了一下天，天气晴朗，阳光灿烂，微风和煦。
<br/>
<br/>        “啊，那我就好，那就好！”杨东提着的一颗心放下来了，他有些疲惫的揉了揉眼睛，最近越来越觉得累了。
<br/>
<br/>        江俞轩的方案其实也是凝聚了他的心血，在前期他们做方案的时候，他也是进行研讨，为了一个小的布局，他们争得面红耳赤，经过大家的集思广益，多少次的磨合才有那样一份可以堪称完美的作品。
<br/>
<br/>        “放心吧，你下午有没有事情，我想和你一起吃顿饭，我最近感觉我都没有吃饱过。”
<br/>
<br/>        “下午恐怕不行，北京来了一个小朋友，我要和他一起吃饭。”杨东笑着说。
<br/>
<br/>        “唉，我又被抛弃了，你不爱我了！”江俞轩调皮的说。
<br/>
<br/>        “得了，下午我带着你，刚好也介绍你们认识一下，也许以后对你有好处。”
<br/>
<br/>        “算了吧，感觉我在利用你。不去了！我自己去吃去。”江俞轩看了一眼身旁的姐姐，下午还是和姐姐一起出去庆祝一下。
<br/>
<br/>        “不行，你晚上得开车，我想和小朋友喝几盅，唉，可惜陈子昂不在西城，要不然还有个酒友可以聊聊天。”
<br/>
<br/>        “我怎么感觉就这么扎心呢，我就是一个司机，陈子昂就可以聊天，你说陈子昂是你的好朋友还是我是你的好朋友？”
<br/>
<br/>        “别贫嘴了，下午来接我。”
<br/>
<br/>        “不接，你让陈子昂来接你。”
<br/>
<br/>        “你这个臭小子，你明明知道陈子昂不在西城，你让陈子昂来接我？”
<br/>
<br/>        “哇，我不晓得陈子昂不在西城，也不晓得陈子昂在哪里啊。”
<br/>
<br/>        “哇，那可真是太扎心了，陈子昂昨晚还和我们视频了。”杨东的心情似乎很好。
<br/>
<br/>        “没爱了，我下午不去接你，糟老头子你欺负我。”江俞轩有一搭没一搭的和杨东调侃着。
<br/>
<br/>        “来来来，晚上我和陈子昂视频让你也见见陈子昂，哈哈哈哈！”杨东开怀大笑。
<br/>
<br/>        “那嫂子晚上也在吗？”
<br/>
<br/>        “嗯，今晚上一起吃饭的人是你嫂子的外甥，自己人。”
<br/>
<br/>        “行，我给嫂子一个面子，勉为其难和你一起吃饭，给你当司机。”
<br/>
<br/>        “行了行了，你心里那点小九九我还不知道，晚上见，我这来人了，挂了。”杨东挂了电话。
<br/>
<br/>        “杨总？”江俞琪一直静静的听着江俞轩和杨东的聊天。
<br/>
<br/>        “嗯。”
<br/>
<br/>        “杨总的身体怎么样？”
<br/>
<br/>        “好像还是之前那样的，只是我后来去问了西医的那个教授，那个教授反而不肯多说什么了，说是病人的隐私，不方便透露，只是让我们多陪陪杨总，让他注意营养，注意休息。”
<br/>
<br/>        “杨总是个好人呢，最主要的是他没有私心，这样的人你和他打交道真的是很省心，而我们也没有什么后顾之忧。”
<br/>
<br/>        “姐，原本我想晚上和你一起吃饭，杨总让我晚上陪他一起，晚上你就自行解决了。”
<br/>
<br/>        “我那一次不是自行解决的？好像离开你我没有饭吃。”
<br/>
<br/>        “嘿嘿，姐，忙过这一段时间，我想吃你做的饭，好久没有你吃你做的饭了。”
<br/>
<br/>        “行了，明天晚上给你做饭吃。”
<br/>
<br/>        “真的？”
<br/>
<br/>        “我什么时候骗过你，对了，爸明天有可能会飞过来，你晚上问问杨总，看他有没有时间我们叫上他和爸一起吃顿饭。”
<br/>
<br/>        “好嘞！”
<br/>
<br/>        项目虽然还没有结果，但是江俞轩等于也是知道了结果，他们只等着发布招标公告就好。
<br/>
<br/>        好消息接连不断，江俞轩又接到了易倾秋的电话；“江总，招标结束了吧？有几成的把握？”
<br/>
<br/>        “一半。你们的结果呢？”
<br/>
<br/>        “你猜！”
<br/>
<br/>        “哈哈哈，听你这口气，肯定是成了，要不然你肯定会哭唧唧。”
<br/>
<br/>        “恭喜江总，招标公告已经下来了，我们评分标准第一，许弋澄第二，宋长生第三。”易倾秋的喜悦透过电话传染给江俞轩。
<br/>
<br/>        “太好了，赶紧给你陈总发个邮件，让她也开心开心。”
<br/>
<br/>        “嗨，我们觉得这个邮件由你来发比较好。”
<br/>
<br/>        “你们发，我最近没有负责这个项目你们陈总是知道的，我要等一段时间给她发我的这个项目。”陈子昂为江俞轩所做的一切江俞轩都知道，这个高新的项目下来了他一定要给她发邮件。
<br/>
<br/>        “俞轩，我们的项目你还要给陈子昂汇报？”江俞琪有点不开心。
<br/>
<br/>        “姐，陈子昂认识刘翼飞！”江俞轩看了一眼江俞琪，从他给江俞琪说了陈子昂的事情之后，江俞琪好像不怎么待见陈子昂。
<br/>
<br/>        “陈子昂好复杂啊。”
<br/>
<br/>        “其实她非常简单，只是被复杂的人给复杂了。”
