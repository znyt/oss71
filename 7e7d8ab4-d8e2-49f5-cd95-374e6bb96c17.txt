　　普加乔夫眼镜蛇机动？Zero-Knot？
<br/>　　对于前一个世界著名的战机机动动作，副机长还是有所了解的；但是至于zero-knot，副机长却是一脸懵，如果有人愿意回答他的话，他真想好好问清楚，这是什么玩意儿！？
<br/>　　驾驶舱内，机长对着林晓阳伸出手，说道：“大名鼎鼎的蜂鸟，早有耳闻。原空军第七军，中校唐震。”
<br/>　　林晓阳伸手跟唐震握了握，笑着说道：“没想到我的名字这么响亮。”
<br/>　　“不响亮也不行啊。”唐震有些疑惑的对林晓阳问道：“你这样的人才怎么会跑去陆军特种部队？”
<br/>　　“当年出了点事儿，不能飞了。”林晓阳坐到副驾驶上，指了指坐在观察员位置上的罗昊，说道：“如果不是他帮忙，我可能连飞机都不敢坐了。”
<br/>　　“我们换个位置？”唐震对林晓阳说道。
<br/>　　林晓阳连连摆手，说道：“我只会开战斗机，民航客机还是您熟悉。”
<br/>　　林晓阳这个王牌飞行员在加入龙隐部队后，每个月还会去空军基地，驾驶战斗机进行飞行，为的就是保持他的状态，不至于完全疏忽了他的“本职”，就连很多飞行员都没有摸过的国产第五代隐形战斗机，林晓阳也不止飞过一次了。
<br/>　　驾驶民航客机，那就更不在话下了，更何况，他现在只是作为副机长，协助机长驾驶。
<br/>　　林晓阳接手副驾驶后不久，那架从A340客机左侧桶滚到右侧的F15J，又是一个桶滚回到了左侧；另外一架F15J更是加速冲到了A340前方，试图以这种方式胁迫A340降落。
<br/>　　驾驶舱的公共频道中，传来F15J上飞行员蹩脚的英文，叽里咕噜的说了一堆，最后的目的只有一个，要求A340降落在冲绳军事基地的军用机场，接受危险品检查。
<br/>　　“滚！”
<br/>　　林晓阳毫不客气的通过公共频道进行了简单粗暴的回应，并且关闭了公共频道，说道：“就这二流水平的驾驶技术也好意思在我面前秀，现在如果驾驶的是战斗机，爷分分钟教你们做人！”
<br/>　　查看了一下雷达后，林晓阳跟罗昊说道：“昊子，跟空军进行联系，说明现在的情况，并且告诉他们，我们大约在三十分钟后进入我国防空识别区，要求他们现在立刻起飞，准备为我们护航。”
<br/>　　罗昊点了点头，立刻通过卫星电话进行联系。
<br/>　　……
<br/>　　某沿海空军基地，一个空军大校气愤的拍打着桌子。
<br/>　　“无法无天！简直就是无法无天！”大校怒极反笑，说道：“他们想干什么，他们到底想干什么！？还想要上演当年‘银河号’事件？我看他们是昏了头！”
<br/>　　“命令！”大校对站在办公室里的通讯员命令道：“第一、第三战斗编队立刻载弹起飞对跟踪的F15J驱逐！另外，第二、第四战斗编队，第一轰炸机编队载弹起飞，去霓虹国海‘逛逛’，来而不往非礼也！”
<br/>　　“是！”
<br/>　　通讯员立正，对着大校敬礼，然后转身离开。
<br/>　　这时，办公桌边一名空军中校皱眉对大校说道：“参谋长，这么做恐怕不妥吧，这样做怕是会引起外交争端啊。”
<br/>　　“外交争端关我什么事？”大校眨巴了一下眼睛，说道：“他要有本事就把我的战斗机编队和轰炸机编队打下来，没本事就闭嘴！就算有外交抗议，那也是外交部的事情，再说了，以前我们抗议了那么多，嘴巴都说干了有用吗？没用！人家还不是照样飞，照样在你头顶上晃悠？还是那句话，弱国无外交！对付流氓，能用棒子就绝对不用拳头！”
<br/>　　中校张了张嘴，不说话了。
<br/>　　……
<br/>　　两架霓虹国航空自卫队的F15J依旧围绕着龙隐部队所搭乘的这架A340客机转悠，并且不时做出一些危险的机动动作，但唐震对此却是视而不见，驾驶着飞机径直往东海防空识别区飞去，意思很明白，想让老子降落那是不可能的，你丫的有本事就直接把我打下来，没本事就一边待着玩杂耍去。
<br/>　　那两架F15J的飞行员还真不敢把这架A340打下来，尽管这两架F15J是载弹起飞，但是从始至终，他们连武器系统的保险都没有打开过，没有他们“爸爸”的点头，霓虹国航空自卫队司令可不敢私自下令开火。
<br/>　　别说是霓虹国航空自卫队司令，就是詹姆斯都不敢下达开火的命令，谁知道现在共和国导弹部队的巡航导弹发射井是不是处于开启状态，车厢中藏着洲际巡航导弹的“死亡列车”是不是正在铁轨上欢快的奔跑？
<br/>　　要是一道命令下去，紧接着找来数颗多弹头的洲际巡航导弹，他们找谁说理去？找谁哭去？
<br/>　　在距离东海航空识别区还有二十多公里的时候，数架苏-30出现在天空中，并且正快速朝着A340的方向飞来。
<br/>　　两架F15J的飞行员看到数量是自己这边数倍的苏-30，顿时有些慌了，不过他们并没有放弃跟随，相反跟还比之前更加紧了，这是他们最后的机会，一旦这架A340进入防空识别区他们就彻底失去机会了，因为他们认为那数架苏-30是不敢飞出防空识别区的。
<br/>　　两个飞行员第一次打开了火控系统的保险。
<br/>　　只是，这一次他们的“认为”错了。这数架从沿海空军基地起飞的苏-30战斗机在发现A340后，立刻就迎了上去，对两架一路尾随A340的F15J进行驱逐。
<br/>　　通过公共频道进行两次警告后，苏-30打开了火控雷达，察觉到苏-30的动作，两架F15J立即释放了一串干扰弹，掉头就跑。
<br/>　　两个F15J的飞行员清楚，如果自己这边继续紧跟着不放，对面那几架苏-30战机是真有可能对自己开火的。
<br/>　　两架F15J离开后，立刻有两架苏-30飞到了A340客机两侧，通讯频道中出来苏-30战机飞行员的声音，“你们已经进入东海防空识别区，现在由我们为你护航，欢迎回家！”
<br/>　　直到这个时候，驾驶舱和客舱中的人才重重的松了一口气，悬着的心彻底放了下来，林晓阳对着飞在自己旁边的苏-30战机竖起一个大拇指。
