　　随着双方的距离越来越近，李世民突然开始后悔自己当初的某一个决定。
<br/>　　如果不是他自己规定了贞观炮的发射距离不得超过六千步，在眼下这个距离他已经可以发起攻击，完全不用再继续向前推进。
<br/>　　但现在的情况却是他必须让舰队继续向前，因为中间隔着自己的舰队，所以不得不等到前面舰队开始接上火之后，他后面的铁甲战舰才能发挥出应有的能力来。
<br/>　　可敌人也不是傻子，数次交手之后，那些大食人也学的精明了不少，他们努力保持着与大唐舰队的距离，只是不断向水中投下渔网，试图用这个东西来阻止大唐舰的前行。
<br/>　　毕竟螺旋桨驱动的战船最怕的就是渔网这个东西，螺旋桨万一被缠住，几乎没有任何办法摆脱，要么继续转动桨叶，赌一把看看能不能将渔网搅碎，如果搅不动就只能等着桨叶与传动轴被缠住，然后停摆。
<br/>　　独孤青云很清楚这些渔网的‘威力’，所以在前面的那些木质战船上面，数支长长的木杆被戳进水里，无数水手在拼了命的搅动，试图把水中的那些渔网都打捞起来。
<br/>　　便是这样，大唐的舰队与大食人的舰队僵持了整整一个上午，直到中午时分，大食人派出去的探子回来了，向阿里·伊本报告了东南方有一座大岛的消息。
<br/>　　“准备迎战吧，绝对不能上唐人登上那座岛！”阿里几乎没有任何犹豫便下达了攻击的命令。
<br/>　　“将军，与大唐舰队硬碰硬我们没有任何胜算！”阿里的副手听到他的命令之后吓了一跳，连忙出来提醒。
<br/>　　“我们不必拼太长时间，唐人显然已经快要支撑不住了，就算是他们有足够的食物，淡水却是无法解决的大问题，所以围住他们，用不了几天，他们就会因为缺少而失去战斗力，到了那个时候，我们就胜利了。”阿里斩钉截铁的说道。
<br/>　　他一点都没有怀疑过自己的判断，因为就在他与大唐舰队僵持的这段时间，他已经补充了三次淡水，可是大唐的舰队却一次都没有补充过。
<br/>　　大海里是没有淡水的，虽然大唐的战舰庞大，但是所存的淡水也依旧有限，他相信一个半月的时间应该足够那些唐人自己将淡水消耗光。
<br/>　　不得不说，他的判断是正确的，李世民的舰队的确是没有水了，但同样，他的判断也是错误的，因为他过高的估计了大食人舰队的实力。
<br/>　　当他们组织到一起，向大唐的舰队发起攻击的时候，阿里就知道，自己再一次失算了，他们根本无法拦住那位大唐的太上皇。
<br/>　　大唐的木质战舰在与大食人的舰队即将接触的时候首先发起攻击，一波火箭弹打下去，立刻有十余艘大食人的战船燃起熊熊大火，无数大食人的水手开始跳船逃生。
<br/>　　接着，大唐后面的铁甲战船开火了，无数攻击距离远超大食人想象的炮弹划空而至，砸到大食人的舰队中间，将一艘艘不大的木船炸成碎片。
<br/>　　那种惊天动地的场面看的无数人一颗心从嗓子眼儿一直砸到屁股眼儿，恐惧开始不由自主的在大食人和其也异族人中间蔓延。
<br/>　　而且大唐的战船似乎也没有停下来的意思，就那么不断的向前突进，几十艘运输船全都是靠风帆为动力，根本不为渔网所困扰，在它们的搅动之下，大量的渔网被拖到了船上，为后面的钢铁战船开出了一条并不大，但却足以通过的水道。
<br/>　　疯了，真是疯了，阿里万万没想到自己的舰队在正面战场上竟然如此不堪一击，面对大唐舰队猛烈的炮火，一个个跑的比岸上的兔子还快，几乎没用多少时间就将前面的通路让了出来。
<br/>　　“堵上，把他们给我堵回去，唐人根本没有多少物资了，那种炮弹他们也没有多少，不要怕他们。”阿里站在自己的旗舰上拼了拿的喊着，但是因为事先缺少沟通，几乎没有多少人会听他的。
<br/>　　而那些听了命令的战船虽然顶上去了，但是很快又会被摧毁，这一情况在那些逃离的大食人看来，无疑是在证明阿里的判断是错误的。
<br/>　　而就在这个时候，似乎老天也在帮助大唐，随着越来越多的大食人和异族人落水，大量的血腥味使海面上开始有大量的鲨鱼聚集过来。
<br/>　　那些鲨鱼拼命的抢夺着这份从降美食，而与此同时，那些被丢进水中，没有被大唐运输秽破坏掉的渔网则在鲨鱼的搅动下纷纷被带离了原本的位置，使得铁甲战舰在这一片海域变的畅通无阻。
<br/>　　独孤青云迅速抓住战机，十二艘钢铁战船瞬间加速，越过那些木质的运输舰，杀入大食人的船队中间，贞观炮，火箭弹，床弩之类的远距离武器不要钱一样砸了出去，迅速将胜利的局面扩大。
<br/>　　仗打到这个时候，阿里知道这一次自己已经无力再阻拦唐人离开，不过好在他们只是想要去那座岛，并不是突围反回大唐，从这一点来说，他以将来还是有机会可以将那位大唐的太上皇抓住的。
<br/>　　想到这里，阿里下达了撤退的命令，无数的大食战船疯了一下向着远离大唐舰队的方向瞬间逃开，连落入水中正在挣扎求生的同伴也不管了，任凭他们在大海中被那些鲨鱼生生拖进无底的深渊。
<br/>　　只有阿里远远的看着逐渐远去，已经停止攻击的大唐舰队，陷入一种沉思的状态，一个此前被他忽略的念头在心中隐隐升起，喃喃自语道：“唐人已经没有能力再战了，他们的武器应该快要消耗光了吧！”
<br/>　　“什么？”阿里的副手并没有听清楚他说的是什么，于是好奇的问道。
<br/>　　阿里并没有回答副手的问道，反而在看了他一眼之后，对他问道：“如果你有很多大唐的那种武器，像刚刚那样的情况你会怎么做？”
<br/>　　“当然是斩尽杀绝！”阿里的副手想都没想便回答道。
