        童筱颖嘴角微扬，眯起漂亮的眸子，娇俏的脸蛋上是甜甜的笑容：“我画了一只青蛙和一只天鹅。”
<br/>
<br/>        她的话语一落，傅夜沉蓦然身心微怔。
<br/>
<br/>        就算她忘了全世界，也不会忘记他的脸，不会忘记对他的呼唤，不会忘记曾经与他一起拥有的那份甜蜜。
<br/>
<br/>        童筱颖不知道自己为什么会突然兴致阑珊地画这两种动物，只不过，脑海里一想到这两种动物的时候，心里总会觉得很甜。
<br/>
<br/>        傅夜沉能看得出，筱颖的潜意识里，还保留着与他有关的记忆。
<br/>
<br/>        “大哥。”
<br/>
<br/>        “嗯。”
<br/>
<br/>        “明天，我顺便去做个检查。”童筱颖忽然语重心长地说道。
<br/>
<br/>        傅夜沉顿时紧张了起来：“是不是哪里不舒服？要不现在就叫医生来给你看看。”
<br/>
<br/>        “不是啊！而是婆婆说的，我到底能不能生孩子的这件事情。”童筱颖微微一笑。
<br/>
<br/>        傅夜沉剑眉微蹙：“我说过，你不用去在意这件事情。”
<br/>
<br/>        童筱颖却一本正经道：“如果我真的如婆婆所说的那样，不能生孩子，那我们就离婚吧！”
<br/>
<br/>        听到她说这句话时，傅夜沉其实有一肚子的火气，但是，他忍了下去，没把火气发出来，而是耐着性子，心平气和地说道：“我们其实可以去领养一个。”
<br/>
<br/>        “大哥，你真的不介意吗？”童筱颖叹了口气，从凳子上起身，蹲到傅夜沉的眼前，匍匐在他的病床边缘，眼睛里是一片澄澈的眸光。
<br/>
<br/>        傅夜沉抬起手来，宠溺地揉了揉童筱颖的头顶：“如果你跟我离婚了，我会死的。”
<br/>
<br/>        “我不要你死。”童筱颖眨巴着眼睛，天真无邪地说道，“我舍不得。”
<br/>
<br/>        这个男人，是全世界对她最好的男人了。
<br/>
<br/>        “那你还要跟我离婚吗？”傅夜沉嘴角微扬，挑了挑好看的剑眉。
<br/>
<br/>        童筱颖犹如拨浪鼓般，努力摇头拒绝的样子很是可爱。
<br/>
<br/>        连华生已经问过他父亲老连，并转告过他傅夜沉，说她童筱颖恢复记忆的可能性为零，要他傅夜沉做好心理准备。
<br/>
<br/>        他愿意为她放弃杀父之仇，放弃这么多年来堵在心里的执念，所以，面对拥有小孩子习性的筱颖，他能全盘接受。
<br/>
<br/>        并不是他斗不过夏阳枭，而是他终于明白，他的继续挣扎，压根就给不了筱颖想要的幸福。
<br/>
<br/>        他可以一直活在阴暗里，但是筱颖不可以。
<br/>
<br/>        “大哥，你怎么长得这么好看？”突然间，童筱颖无厘头地说道。
<br/>
<br/>        傅夜沉怔了一下，爽朗地笑了起来，指腹轻轻地刮了刮童筱颖的鼻梁：“你也是，我的宝贝，怎么长得这么好看。”
<br/>
<br/>        或许、大概，是因为情人眼里出西施。
<br/>
<br/>        “吧唧”一声，童筱颖凑近傅夜沉的脸庞，在他的脸颊上亲了一下。
<br/>
<br/>        傅夜沉没料到童筱颖会突然来这招，整个人瞬间一脸懵。
<br/>
<br/>        童筱颖凝视着他的眼睛，笑容像阳光那般格外灿烂，又像蜜糖般看起来很甜蜜。
<br/>
<br/>        他骨节分明的手，忍不住在她柔软的唇瓣上摩挲起来。
<br/>
<br/>        傅夜沉的俊脸慢慢地朝童筱颖靠近，当四片唇瓣快要碰到一块时，一个护士叩响了房门。
<br/>
<br/>        童筱颖“噌”地一下站起身来，红着脸，远离了傅夜沉。
<br/>
<br/>        护士似乎没注意到刚刚房内的情况，而是自顾自地走到病床边，给傅夜沉换吊针上的吊瓶。
<br/>
<br/>        在医院里照顾傅夜沉的这两个星期里，童筱颖觉得自己简直不要太聪明了。
<br/>
<br/>        例如，她以为自己不会下厨，但是真到为傅夜沉洗手作羹汤的时候，动作娴熟又麻利。
<br/>
<br/>        再例如，傅夜沉需要在笔记本上修改合同，但是他趴在床上，仰着头看太久脖子会痛，所以，她童筱颖还能帮他把合同里的内容念出来，问题是她念的还是英文合同，然后他说改哪里，她便帮他改哪里。
<br/>
<br/>        傅夜沉发现，他的这个小女人其实一直都没有变，如果没有那么多伤心的经历，她会是一只快乐自在的小鸟。
<br/>
<br/>        他为什么没有早点意识到这点？
<br/>
<br/>        如果，早点能意识到……
<br/>
<br/>        “大哥，你在想什么呢？”童筱颖正在整理东西的时候，见傅夜沉突然望着自己发呆，于是放下手里的活，蹦到他跟前，抬起手来，在他眼前晃了晃。
<br/>
<br/>        今天出院后，他们回了傅家大庄园。
<br/>
<br/>        为了不让冷夫人和傅老太爷担心，傅夜沉在医院里住了两个星期这事，家里的这两位长辈并不知道。
<br/>
<br/>        “我明天要去一趟滨江城，你是跟我一起走，还是留在这里等我回来？”傅夜沉回过神来后，目光温和地看着童筱颖问道。
