　　妙心接着说：“靠风水术想当皇帝，其实是非常难的，因为风水学中，百密一疏，只要有一处不对，整个风水盘也就毁了，达不到最好的效果。
<br/>　　而人算总有穷尽，就算是我们地师家族，也不能保证做到，就算做到，一些天灾地动、甚至一只钻进古墓的穿山甲或者老鼠，都能毁坏整个风水盘，再像禳补就不可能了。”
<br/>　　叶少阳缓缓点头，这些知识，大部分大本来是懂的，只是因为妙心刚给他说了太多新奇的东西，一时间有点晕，问了傻问题，当下说道：“其实，这也算是一种天命吧，有时候机关算尽，但是并不一定就能成功。”
<br/>　　妙心点点头，表示赞同他的意见。
<br/>　　叶少阳再看那些用来凑阵的手指，想到他们当初是被人从一百个人手上生生斩下来的，顿时有点头皮发麻。
<br/>　　大伙的目光，不约而同落在那副棺材上。
<br/>　　叶少阳道：“你之前说，这里卖的是大宋皇族的后人？可是，为什么会用西域的邪术？而且我们之前看到的就是。为什么会这样？”
<br/>　　妙心想了一下，说道：“也许当时修墓的人，找不到地师，而身边又有一个西域的了不起的法师，于是就借助了西域巫术，反正结果都是一样的，用什么法术去做到，倒没什么区别，也算是就地取材吧。”
<br/>　　这种猜测，虽然没有任何证据，不过叶少阳觉得可以接受。
<br/>　　当下上上下下打量着这座坛城，说道：“我怎么有一种九层妖塔的即视感？”
<br/>　　妙心愣了一下道：“你知道九层妖塔？”
<br/>　　这一问，倒是把叶少阳给问住了。“额……这个，鬼吹灯里看到过。”
<br/>　　“鬼吹灯是什么？”
<br/>　　叶少阳耸了耸肩，这个就没法解释了，总不能告诉他这是百年之后有人写的一本盗墓小说吧？
<br/>　　“你怎么也知道九层妖塔？”叶少阳好奇地问道。百年前可没有《鬼吹灯》这本书……
<br/>　　“知道，这是地师家族的一个传说，长白山下，有一座九层妖塔，里面关着强大的邪物……我们眼前这座坛城，其实也是一座妖塔，只不过是用西域的邪术塑造的，供奉的应该就是赵禥的后人。”
<br/>　　叶少阳抬头望着棺材，说道：“会不会里头有一个千年老粽子。”
<br/>　　“不太可能有粽子，不过，这个也说不准。”妙心瞥了他一眼，说道：“你们法师管僵尸也叫粽子吗，我好像没听说过。”
<br/>　　“不是，我身边有一个朋友，是摸金校尉，这些名词都是他告诉我的。”
<br/>　　妙心一听，有些动容，说道：“真正的摸金校尉吗，那可真是罕见，如果有机会，你可以带我认识一下，我有一些问题，很想跟摸金校尉探讨。”
<br/>　　摸金倒斗，跟地师虽然是不同职业，不过因为做的都是跟古墓相关的事，有很多地方有相通之处，也不怪妙心感兴趣。
<br/>　　“我们要不要这就把棺材打开看看？”梧桐早就好奇得不行，总算等到他们，立刻提出了这个想法。
<br/>　　妙心也很好奇，当然同意。
<br/>　　“那就先把这些人骨去掉，破了坛城的法阵再说。”这些法阵，都是用来加持风水的，对古墓本身不会有什么影响，因此梧桐也是毫不客气地把那些摆在坛城上的人骨清理掉，法阵果然立刻就被破除了，阴冷的空气一下子就消散无踪了。
<br/>　　叶少阳招呼毛小方一起，爬到棺材旁边去，开棺之前先上下检查了一下，这才看到棺盖上雕饰着凤凰图案，惟妙惟肖，一看就工艺不凡，最重要是，就算是叶少阳，也能一眼看出这是汉人的手艺。
<br/>　　在汉族的民间习俗中，凤与凰本来是一对，凤为雄，凰为雌，后来却成了龙凤相对，在古代，凤凰也是有极高的地位，能用凤凰来比拟象征的，只有皇后一人，连一般的妃子都不行。
<br/>　　叶少阳看到棺材上的凤凰雕纹，顿时有点吃惊，本来以为这里会是什么赵禥的子孙，但是看到这棺材上的雕纹，就知道肯定不是了。
<br/>　　帝王所用的东西，一定是龙形，而不是凤凰，难道这棺材里面是皇后？
<br/>　　既然棺材就在面前，乱想这些没意义，里面到底是谁，开棺就知道了。叶少阳找了一个稳妥的位置站好，跟毛小方一起在棺盖下面摸索起来，这种金属质地的棺材，其实比普通棺材还要好开的多，因为普通棺材都要钉钉，金属棺材是没法钉钉的，都是直接扣上去，开启起来也不是那么费事，而且银子也不重。
<br/>　　两人找好位置之后，一人站在一头，抠住棺盖，调整了好几次，最后找到平衡点，一起把棺盖掀了起来，直接扔下去。
<br/>　　叶少阳低头去看棺材里面，并没有直接看到尸骨或者僵尸，而是一个浑身被银色丝线缠绕的形状，从头到尾，全身都被裹紧，就好像木乃伊一样，一动不动得平躺在棺材里。
<br/>　　两人当场就震住了。呆了好半天，叶少阳伸手摸了一下尸体的脚，有金属的感觉，看来这尸体身上穿的多半是真是银丝编成的衣服。
<br/>　　“里面有什么，说话呀！”陈晓宇等人在下面大叫。
<br/>　　因为坛城太高，周围墙壁上的冷光照得不清不楚，叶少阳找妙心要来一盏鲛油灯，对着尸体打量起来。
<br/>　　尸体的身材娇小，因为银丝衣紧紧紧身，像是穿了一件紧身衣，身材也纤毫毕现，看上去……身材还真挺火辣的，嗯，关键是有胸，至少是……这得是D罩杯？
<br/>　　叶少阳吐了吐舌头，中止了自己的胡思乱想，再打量尸体的四周，发现她身下垫着的是软软的黄布，这一点倒是符合汉人的习惯，而且是皇室习惯。
<br/>　　看来没错，这一副尸体，应该就是所谓的皇后了——至少是按照皇后的标准下葬的，不过……自己没听说过古代有把人用银丝包裹了再下葬的习惯啊？
<br/>　　“少阳子，你看她的手。”毛小方在对面发现了什么，招呼叶少阳去看。
