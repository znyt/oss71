<br/>    七天的时间过去了，陈明依旧是没有出来接旨，接任天庭大祭司之位。
<br/>    
<br/>        瑶池确实有旨意到了，这个时候，瑶池终于是按捺不住心中的悸动，在这一天，瑶池的旨意，同样是降临到了陈明的院前，“帝有言：请燃罪上师执掌瑶池大祭司权杖！”
<br/>    
<br/>        并没有那么多的废话，就是一句话，简单明了。
<br/>    
<br/>        在陈明的院外，瑶池的千秋亲王举着旨意和大祭司权杖，在二太子的身旁站好，静静等待着陈明地出现。
<br/>    
<br/>        二太子扫了这瑶池的千秋亲王一眼，也是有着一抹怒色，你们瑶池，也想要请燃罪上师？
<br/>    
<br/>        还是在我们天庭的地盘上请？
<br/>    
<br/>        可是再次过了三天，燃罪上师依旧是没有出现，院门依旧是封闭，这个时候，更多人看出来了，看来瑶池也不是燃罪上师的选择！
<br/>    
<br/>        就在第三天刚过，这一天黎明之前，太虚的苍空亲王拿着太虚的旨意来到了此处，“帝有言：请燃罪上师执掌太虚大祭司权杖！”
<br/>    
<br/>        苍空亲王的话语不过刚刚落下，就听见另外一声中气十足的大喝，那是一位老妪，月宫的伴月婆婆，帝主的身边人，“帝有言：请燃罪上师执掌月宫大祭司权杖！”
<br/>    
<br/>        不到一个时辰的时间，陈明吃过了早饭，终于是微微一笑，嗯，差不多了。
<br/>    
<br/>        陈明打开院门，院门打开，众多的修行者都是齐齐看向陈明，白衣圣僧，佛光万丈。
<br/>    
<br/>        陈明走到二太子的身前，伸手，接过了二太子手中的大祭司权杖。
<br/>    
<br/>        所有人的一颗心终于是落下，燃罪上师，最后还是选择了天庭。
<br/>    
<br/>        瑶池，月宫，太虚的人则是心中一阵失望，燃罪上师，最后还是选择了天庭。
<br/>    
<br/>        这个时候，伴月婆婆却是见到，陈明走到她的身前，一把接过了月宫的大祭司权杖。
<br/>    
<br/>        众人都是安静了下来，不太明白陈明的意思。
<br/>    
<br/>        却是见到陈明再次走到苍空亲王之前，伸手接过了太虚的大祭司权杖，同时走到千秋亲王之前，接过了瑶池的大祭司权杖。
<br/>    
<br/>        陈明将四根大祭司权杖插在身前
<br/>    
<br/>        ，双手合十道：“我不入地狱谁入地狱，愿为光明，祛除世间一切黑暗！”
<br/>    
<br/>        “承蒙帝主不弃，小僧，愿为四国大祭司！”
<br/>    
<br/>        一言道出，众人都是嘴巴张大，不知道说什么，这燃罪上师，准备接任四国大祭司！
<br/>    
<br/>        这四位帝主会同意吗？
<br/>    
<br/>        燃罪上师，想要以一人之躯，掌四大仙国的大权！
<br/>    
<br/>        天庭大祭司在人群之中遥遥地看向陈明，“这妙手是有了，这局你也破了，可你可知道，这样做的代价，可是不得好死啊！”
<br/>    
<br/>        历代君不见的大祭司，少有能够善终的，就好比瑶池的那位九千岁，没有什么大错，还不是因为调戏公主关在了瑶池之中，永世不得出来。
<br/>    
<br/>        调戏公主，在大祭司的眼里，简直不值一提，别说一个公主了，睡个四五个都没事啊！
<br/>    
<br/>        那位九千岁，权柄太重，已经得到了帝主的忌惮，只能在牢狱之中孤独终老了。
<br/>    
<br/>        君不见，必定权柄极大，一言决定天下事，但是同样的，这同样是因为帝主有所忌惮，帝主甚至不敢去见这种大祭司，因为他们的一举一动，都会暴露自己的想法，然后就会成为大祭司的棋子，故而帝主压根不见大祭司。
<br/>    
<br/>        不过天庭大祭司可是听到了陈明的话的，他不入地狱，谁入地狱？
<br/>    
<br/>        这是抱着必死之心，也要执掌四大仙国啊！
<br/>    
<br/>        若是能够胜过无冕者议会，这未必，不是一件能够名垂千古的事。
<br/>    
<br/>        陈明抱着四根象征着权柄的权杖，来到伴月婆婆的面前，然后道：“去月宫！”
<br/>    
<br/>        伴月婆婆现在也是有点发蒙，这算怎么回事啊？
<br/>    
<br/>        难道真的要让燃罪上师一人成为四大仙国的大祭司？一人独掌天下？
<br/>    
<br/>        但是现在，伴月婆婆也不好说什么，心道：这燃罪上师，应该蹦不了多久，四位帝主，绝对不会同意的。
<br/>    
<br/>        现在嘛，先看看燃罪上师到底准备干嘛吧。
<br/>    
<br/>        伴月婆婆很是尴尬地看了二太子，还有两位亲王一眼，挥手道：“请燃罪上师上车，去月宫！”
<br/>    
<br/>        陈明登上马车，去月宫自然是有着他的打算，他在这儿这么多天，也不是白呆的，月宫和妖族，在长璇星系争斗得很凶，既然想要请君入瓮，自然要给这四大仙国一点甜头。
<br/>    
<br/>        辗转了三月的时间，陈明终于是到了月宫，遥遥地看向月宫的那一颗星辰，却是惊呆了，好一颗大星，这是和苍龙星差不多大小的一颗星辰，这颗星辰，准确的说，实际上便是月亮。
<br/>    
<br/>        这颗星辰，还有一个名字，叫做太阴星。
<br/>    
<br/>        如今办伴月婆婆已经没有什么放肆的态度了，对陈明恭敬有加，原因便是因为四大帝主，竟然对于陈明执掌四大仙国大祭司，没有什么反应。
<br/>    
<br/>        没有反应，便是反应，这是默许了！
<br/>    
<br/>        看来现在这一位，是真的得罪不起啊，数万年以来，第一位能够执掌整个仙国大权的一位大祭司！
<br/>    
<br/>        陈明来到太阴星的第一天，月宫太子便是亲自前来拜见，月宫太子生的倒是一番好样貌，若不是早就知道这月宫太子是男的，陈明还以为这家伙是女扮男装，是一个阴柔之极的男子。
<br/>    
<br/>        陈明居住在以前的月宫大祭司的宫殿之中，月宫太子很快便前来拜见，在后院之中，月宫太子朝着陈明恭敬道：“见过大祭司！”
<br/>    
<br/>        陈明微微点头，示意自己知道了。
<br/>    
<br/>        月宫太子也是不敢有丝毫大意，眼前这一位，可是真正地权势大到惊天的地步。
<br/>    
<br/>        月宫太子问道：“大祭司，不知道有什么安排？”
<br/>    
<br/>        陈明道：“所谓修行，争得便是一个天命，小僧此来，便是要先替月宫定鼎天命！”
<br/>    
<br/>        月宫太子了然，然后道：“不错，这气运一说，虽说虚无缥缈，但是在上师眼中，必定能够清楚地看出来的。只是不知道，上师准备从哪里入手？”
<br/>    
<br/>        陈明笑道：“殿下可知，月宫能够立足世间的凭借是什么？”
<br/>    
<br/>        月宫太子道：“自然是我月宫的英灵树！太阴金桂！”
<br/>    
<br/>        陈明却是道：“历代大祭司想必对于太阴金桂的位置都有着考量，必定是最好的位置，我要说的，是另外一点，无字碑！”
