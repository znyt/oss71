　　不管是古代还是现代，主帅从来都是士兵心中的天，主帅战死，那就等于是天塌了一样。
<br/>　　所以阿里·伊本这一死，刚刚还杀红了眼的大食人瞬间就没了斗志，‘哗’的一下便从山坡上退了下去，一个个拼了命的向四外开始逃窜。
<br/>　　而就在此时，刚刚完成登陆的北海舰队也杀了上来，他们这一波人与李世民率领的‘海狼’可不一样，清一色的新式步枪，隔着老远就对着那些大食人开了火。
<br/>　　另外还有那些负责火箭弹的士兵，借着大食人的乱劲儿，不管三七二十一，哪里人多就往哪里打，一颗火箭弹下去，立马就是死伤一片。
<br/>　　大食人在此之前死了主帅原本就是军心大乱，这个时候再被北海舰队这么一打，顿时彻底乱了阵脚，不管那些还活着的基层指挥官如何去喊，也无法阻止溃散的发生。
<br/>　　仗打到这个份上，已经到了可以尾声，无数的大食人丢掉手中的武器跪地投降，也有无数的大食人跳海逃生。
<br/>　　不过在这个远离大陆的海岛之上，那些逃走的人最后唯一的下场就是喂鲨鱼，或者被那些土著抓住，变成一堆白骨。
<br/>　　而在一片狼藉的战场之上，最吸引人注意的还是投出长枪杀死阿里·伊本的秦琼。
<br/>　　战斗虽然已经结束，但他却依旧保持着投出长枪的姿态，一手提着马缰，一手做投掷状，呼雷豹也是一副纵跃的姿态，一只前蹄抬起，头颅高高的仰着。
<br/>　　这样的一副画面很快引起了众人的注意，尉迟恭气喘吁吁的催马来到他的身边，只看了一眼整个人便僵在当场，尉迟宝琪赶到的时候，同样发出一声惊呼：“秦，秦叔叔……他……他。”
<br/>　　直到此时，众人才发现，一代猛将，秦琼秦叔宝，在投出自己手中长枪的同时，就已经停止了呼吸！而他的战马呼雷豹亦同时随他而去。
<br/>　　……
<br/>　　李世民万万没想到，一场大战之后，第一个收到的消息并不是大获全胜，反而是自己的心腹重臣殡天。
<br/>　　看着前来通知自己的长孙无忌，李世民踉踉跄跄的退了好几步，嘴角不断抽搐：“这，这不可能，叔宝，叔宝不是得一季阳寿么？怎，怎么地会……会……。”
<br/>　　“陛下，今年，正是一季！”长孙无忌扶着李二，语气有些沉声的说道：“翼国公，时间已经到了。”
<br/>　　“不，朕不信，无忌，你一定是在骗朕对不对？你是在与朕开玩笑，对不对？！”秦琼的死对于李世民的打击太大了，他完全无法接受数个时辰前还在与自己谈笑风生的心腹爱将就这么离自己而去的事实。
<br/>　　“陛下，翼国公的确是去了，请您节哀！”方老太监此时已经从外面核实情况回来，压低了声音对李世民劝道。
<br/>　　“不，不可能，叔宝早上还与朕说此战大胜之后要一醉方休，怎地会如此不守承诺，怎地就这样去了！”李世民一把扯住老太监的衣襟，大声喝道：“你去找承乾，让承乾再布七星阵，将叔宝给朕救回来！”
<br/>　　老太监虽有一身不俗的武技，但被李世民提在手里却没有丝毫的反抗，只是哀声对已经失去理智的老李劝道：“陛下，我，我们是在南海，距离长安数万里之遥，来，来不及啊！”
<br/>　　李世民在老太监的提醒似乎终于想起自己刚刚正在打仗，松开抓住方老太监的手，喃喃自语道：“南海？大食人？！”
<br/>　　老太监得以脱身之后，躬身言道：“是的陛下，大食人已经被打退了，五殿下带着北海舰队过来了，翼国公是战死的，死前亲手了解了大食人的主帅阿里·伊本。”
<br/>　　“李佑？李佑来了？”李世民听到五殿下几个字，皱了皱眉头。
<br/>　　“是的！眼下五殿下正在追击溃军，一会儿应该就会过来。”
<br/>　　“早不来，晚不来……”李世民依旧没有从秦琼离世的悲痛中恢复过来，喃喃嘀咕了一句之后像是突然想起什么，沉声言道：“齐王李佑救驾来迟，命人拖下去仗三十！”
<br/>　　“陛下不可！”长孙无忌一直在边上没有说话，此时一听李世民竟然有乱命的意思，连忙阻止道：“陛下，北海舰队已经尽了最大的努力了，责其主将实为赏罚不明，为明君所不取！”
<br/>　　“这也不行，那也不行，难道朕的翼国公就这么白白的死了？朕……朕……”李世民说到这里，只觉得心里一阵堵的慌，手按心口，‘噗’的一口黑血喷了出来，人在一瞬间便倒了下去。
<br/>　　如此一幕正被从外面赶来的长孙皇后看在看中，顿时吓的一声惊叫：“陛下！”
<br/>　　……
<br/>　　一场血战，其结果是秦琼战死，李世民昏迷不醒，大食人被杀的落花流水。
<br/>　　这一切说到底其实还是错在老李，如果他不是处处小心，在战斗开始的时候就直接让部队使用步枪战斗，根本不会出现眼下这样的情况。
<br/>　　但是现在说什么都晚了，冥冥之中天意弄人，实非人力所能阻止。
<br/>　　贞观十年的时候，李慕云用七星阵法救回秦琼，使其得以延寿一季，现如今时间到了，就算是李世民没有做出这样的决定，老秦该走还是要走，就算没有死在战场上，也会死在其他地方。
<br/>　　而且从眼下的情况来看，老秦战死沙场其实也算是成全了他，毕竟对于一个英雄来说，老死病榻之上才是最大的悲哀，战死沙场才是最好的归宿。
<br/>　　李佑在战斗结束之后不久也收到了秦琼战死的消息，震惊之余免不了也是幽幽叹息了一声。
<br/>　　不过，他现在并没有时间去管老秦的事情，山腰一场血战‘海狼’战千余人，伤者无数，大食人战死一万有余，伤者三万，另外还有两万余的俘虏。
<br/>　　这些都需要有人来负责处理，所以李佑只是匆匆探望了一下昏迷中的老头子，便又开始忙着处理战后的事务，收敛将士遗骨，安扶伤者，重要的是把老秦的遗体带回船上。
