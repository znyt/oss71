<br/>    星光笼罩，如同一道道光柱落下，落到陈明的身旁，魔宗七劫剩下的五人终于是察觉到了不对的地方，“这是天劫，他本就是大君之下无敌，现在，度过了这天劫，就是大君了吧？”
<br/>    
<br/>        “先看看情况，又不是每一个人，都能安全渡劫的！”
<br/>    
<br/>        “要是他渡劫失败，或是渡劫成了重伤，我等可是有着斩杀大君的机会！”
<br/>    
<br/>        一想想有斩杀大君的机会，魔宗七劫剩下的五人也是心头火热，“斩杀大君，就在今朝！”
<br/>    
<br/>        紫霞仙子和雪自悟看向这奇奇怪怪却又威势不下超脱劫的天劫，“宗师度过此劫，应该就是大君的实力了！”
<br/>    
<br/>        陈明脚踏剑龙，手握万道天竹，看向众多的星光，魔王们此刻早就已经离开了此处，大君的天劫，他们在天劫之下怕是瞬间变成飞灰。
<br/>    
<br/>        陈明意念一动，灵剑推着卓青瑶来到雪自悟和紫霞仙子的身旁，“看好她！”
<br/>    
<br/>        他渡劫，卓青瑶可顶不住，还是让雪自悟和紫霞仙子照顾一番为好。
<br/>    
<br/>        紫霞仙子伸手，将卓青瑶拉了过来，卓青瑶见是紫霞仙子，也没有怎么在意，乖乖地站在紫霞仙子的身前，总得来说，这个小姐姐还是很不错的！
<br/>    
<br/>        至少，能够让自己打的很爽！
<br/>    
<br/>        陈明伸手摸了摸身旁的星光，这就是我的天劫吗？
<br/>    
<br/>        上方忽然一阵变化，凝结出一朵方圆十里的黑云，黑云之下，所有的水，都被天劫之力压得瞬间一空，朝着周围蔓延去，陈明如同站在黑色的海面上，陈明看向头顶之上的黑云，这应该就是劫云了，居然有着十里方圆！
<br/>    
<br/>        黑云之中一阵翻滚，忽然有着一道星光落下，然后，便是八十七道星光破开黑云，从其中照耀而下，一道道光柱，分布在陈明的四面八方。
<br/>    
<br/>        自光柱之中，忽然有着一物落下，陈明仔细看向那一物，居然，是一颗并不饱满，还有着些许褶皱的豆子！
<br/>    
<br/>        这什么鬼啊？
<br/>    
<br/>        天上开始下豆子了........
<br/>    
<br/>        难不成，天劫想要用豆子撑死我？
<br/>    
<br/>        问题是，我不喜欢吃豆子啊！
<br/>    
<br/>        雪自悟也是愣了愣，“怎么天空之中开始下豆子了？紫霞仙子，你是群星仙门的，宗师渡劫倒是有点像你们群星仙门的风格，你看出什么了吗？”
<br/>    
<br/>        紫霞仙子仔细看向这些豆子，忽然惊叹道：“这好像是，撒豆成兵！”
<br/>    
<br/>        雪自悟听到紫霞仙子的话，也是惊叹道：“撒豆成兵，这是仙术！”
<br/>    
<br/>        紫霞仙子道：“还不仅仅是仙术，撒豆成兵恐怕只是第一步。”
<br/>    
<br/>        就在这时，豆子落在海面上，一颗豆子忽然升起一阵云雾，云雾散开之后，一个浑身如同群星构成的天兵出现，手中握着一柄巨剑。
<br/>    
<br/>        周围的豆子瞬间开始变化，变成一个个手握不同兵器的天兵，这些天兵，体内有着星辰之光，如同是一颗颗星辰构筑而成，星辰相连便是骨骼，璀璨的星光之中，这些天兵忽然开始动了起来，他们朝着陈明的头顶上挥手打出，便是一道星光凝聚在陈明的头顶。
<br/>    
<br/>        紫霞仙子继续道：“果然没错，这是周天星辰大阵，传闻这世上有着三千大道本源星辰，便是三千大道，这周天星辰大阵，传言可以磨灭一切！而现在，宗师经受的，便是部分的周天星辰大阵。”
<br/>    
<br/>        一瞬之间，海面之上，野草狂长，树木横生，一株株大树拔地而起，劫云之中，一滴滴雨水落下。
<br/>    
<br/>        陈明看得到脚下有着野草在狂长，树木的枝丫从自己的头顶蔓延而过，从树叶的缝隙之间，一滴滴雨水落下。
<br/>    
<br/>        这些都是星光形成的，陈明可以轻易看出来，自己的天劫，到底是什么鬼？
<br/>    
<br/>        刹那间，地面上的一草一木，皆是兵器，朝着陈明了过去，无数的兵器朝着陈明飞去，如同一阵狂风暴雨，长刀森寒，利剑锋锐，长枪如林，斧钺勾镰，十八般兵器紧接着飞来！
<br/>    
<br/>        陈明心念一动，抬头看向头顶之上的黑云，脚下的剑龙随着他的心意飞向天空之上的劫云之中，这一刻，陈明踏龙，手握万道天竹，在斑驳的海面上乘龙而起！
<br/>    
<br/>        万道天竹挥动，陈明打向天空之上的劫云，一道青风飞过，朝着劫云飞去，自那劫云之中，射出一道道星光，朝着青风打去，星光碎裂，化作灵气。
<br/>    
<br/>        陈明看向劫云，“果然，这天劫不像之前万道天竹的天劫那么简单，它根本不会轻易给我机会让这劫云消散！”
<br/>    
<br/>        紫霞仙子看向陈明在狂风暴雨之中的身影，“他手中的万道天竹，我没记错的话，应该是天道不动之器！”
<br/>    
<br/>        雪自悟道：“你的意思是，那柄拂尘，是仙器？”
<br/>    
<br/>        紫霞仙子道：“还不是仙器，只是具有了仙器的部分威能，不过这万道天竹的第一个威能，叫做天道不动之器。”
<br/>    
<br/>        雪自悟道：“这样的仙器，若是成为真正的仙器，威能会更加可怕，只怕神兵榜第一的位置就要变了。”
<br/>    
<br/>        紫霞仙子道：“天劫不会那么容易让他碎掉这劫云的！”
<br/>    
<br/>        浓厚的黑云之中，忽然有一片幽蓝色的鳞片出现，然后是一只爪子露了出来，一根龙须自黑云之中，一条龙尾，不时从劫云之中扫出。
<br/>    
<br/>        紫霞仙子急道：“天劫变了，这分明是一条星光龙！”
<br/>    
<br/>        雪自悟道：“你说的是，超脱劫之中才会遇到的龙？”
<br/>    
<br/>        紫霞仙子点头道：“不错，超脱又可称之为化龙，因为超脱劫之中，会出现一条大道之龙，代表着天道，而苦修，达到超脱，能够拥有一条龙的力量，故而，又称之为化龙。”
<br/>    
<br/>        黑云之中，星光龙终于是露出了一个龙首。
<br/>    
<br/>        而此时，陈明的脚底，已经万兵来袭。
<br/>    
<br/>        陈明脚下的剑龙一变，在陈明的身后凝聚成双翼，长剑为羽，陈明操控着背后的双翼将自己包裹，无数兵器击打在陈明的双翼之上，大雨之中，无数的火花迸溅，陈明的嘴角，也是有着一缕鲜血溢出。
<br/>    
<br/>        这些兵器之中，蕴含着苍天大势！
