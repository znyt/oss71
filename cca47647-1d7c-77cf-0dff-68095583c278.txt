　　　　大魔老的速度极快，如同一颗黑色的流星，风驰电逝。
<br/>　　“当本王不存在吗？”东海龙王见此，顿时冷喝一声。
<br/>　　昂！
<br/>　　随即，一声巨大的龙吟声响彻天地，而后，只见东海龙王的身躯，瞬间化为了万丈蛟龙之躯。
<br/>　　嗖！
<br/>　　龙尾一摆，巨大的蛟龙之躯，瞬息千里，直接就从上空超越了大魔老。
<br/>　　昂！
<br/>　　一声巨吼震天动地，东海龙王巨大的龙口一张，一股粗大的水柱从龙口中喷涌而出。
<br/>　　这股水柱中，蕴含恐怖到极致的水属性力量，犹如咆哮的大海，又犹如万里江河。
<br/>　　水柱犹如是一柄巨大的擎天神剑，破碎虚空，以无与伦比的速度，向大魔老袭杀而去。
<br/>　　“吼！”
<br/>　　大魔老的口中，发出了如野兽一般的嘶吼声，一拳狠狠的轰击而出。
<br/>　　这一拳没有滚滚尸气，浩大的伟力全部凝聚在了一只拳头之上。
<br/>　　大魔老的拳头干枯瘦弱，其上长着浓密的紫色毛发，显得异常的妖异。
<br/>　　轰！
<br/>　　霎时之间，紫色的拳头轰击在了水柱之上，爆出惊天巨响。
<br/>　　嘭！
<br/>　　粗大的水柱瞬间炸开，化为漫天水珠，激.射八方。
<br/>　　霎时间，方圆数十里之内，仿佛下起了大雨。
<br/>　　咚！咚！咚！！
<br/>　　水柱中蕴含的惊天力量，顿时让得大魔老身形不断的后退，每退一步都将脚下的空间踩的炸裂开来。
<br/>　　东海龙王得不给大魔老丝毫脱身的机会，一击之后巨大的龙尾一甩，向大魔老抽击而去。
<br/>　　而他的巨口中，则再次爆射出一道水柱，向远处的二魔老袭杀而去。
<br/>　　显然，东海龙王此举，是要牵制住大魔老和二魔老，让得苏莫有足够的机会击杀上官昊。
<br/>　　而此刻，苏莫已经距离上官昊极近了，不知三千里，完全可以轻易的袭杀上官了。
<br/>　　“死吧！”苏莫一声怒喝，帝皇剑猛然一挥，一道犀利的三色剑气，斩破虚空，如同一道浩大的三色闪电一般，向上官昊袭杀而去。
<br/>　　“糟糕！”上官昊见此顿时大惊失色，剑气上裹挟的杀机，几乎让他生不起反抗的力量。
<br/>　　不过，他绝对不能死，电光火石之间，他极力向一旁躲避。
<br/>　　嗤！
<br/>　　剑气如同死亡之刃，从耳畔划过，急速斩向远处。
<br/>　　呼！
<br/>　　上官昊见此长长的出了口气，好在苏莫距离他较远，他勉强还有躲避的机会。
<br/>　　而后，上官昊爆发出生平最快的速度，加速飞逃。
<br/>　　他知道自己千万不能被苏莫追上，不然必死无疑。
<br/>　　他目前的实力，与苏莫相比，差距不是一星半点，而是如同婴儿与成年人的差距一般。
<br/>　　“上官昊，你的死期到了！”苏莫大声冷喝，虽然刚才一击没有杀死对方，但他机会多的是。
<br/>　　嗖！
<br/>　　苏莫御剑急行，速度比上官昊要快的多，他一边追击，一边分出一缕心神控制另外八柄灵剑。
<br/>　　另外八柄灵剑，虽然完全不是殷棕的对手，但从四面八方不断的攻击却能纠.缠住对方。
<br/>　　咻！
<br/>　　苏莫整个人都如同是一道剑光，快速的和上官昊拉近距离。
<br/>　　“哈哈！苏宫主，杀了那个小畜生！”远处，海明江哈哈大笑着说道。
<br/>　　此刻，海明江完全被阳鼎天压在了下风，不过，后者的实力并不比他强多少，想打败他也不容易。
<br/>　　“苏莫，你也算是帝玄宫的弟子，就代我清理门户吧！”帝释的声音紧接着响起。
<br/>　　帝释的实力和北堂平华相差无几，隐隐还要略强半分，两人的战斗非常激烈。
<br/>　　注意到苏莫在追杀上官昊，帝释顿时大吼了一声。
<br/>　　苏莫没有理会海明江和帝释，他已经追逐到了上官昊身后两千里之内，完全可以轻易斩杀对方，不会再有任何悬念了。
<br/>　　因为上官昊的实力，根本不可能挡住他一招半式。
<br/>　　“上官昊，结束了！”
<br/>　　苏莫轻喝一声，手中剑光不断闪烁，一道道犀利的剑气，形成了一股浩大的剑气风暴，向上官昊绞杀而去。
<br/>　　远处的大魔老，见到上官昊遭遇如此攻击，口中发出一声声如野兽般的怒吼，但他根本摆脱不了强大东海龙王。
<br/>　　轰隆隆！！
<br/>　　瞬息之间，浩大的剑气风暴绞杀一切，裹挟惊天巨响，急速临近了上官昊。
<br/>　　“不……我不想死……！”
<br/>　　在上官昊惊恐的吼声中，浩大的剑气风暴轰然来临，将他彻底淹没。
<br/>　　在上官昊的身体，被剑气风暴淹没的那一刻，其目光中充满了无尽的恐惧、不甘和悔恨。
<br/>　　嗤嗤哧！！
<br/>　　随着身体不断的被撕裂，上官昊的意识仿佛回到了过去。
<br/>　　那是一个弹丸小城，城主府的比武台上，一名土著少年，伫立在他的身前，声音铿锵有力：“我只需五年时间，必定会超越你，五年之后，我会向你发出挑战！”
<br/>　　这段话语，在上官昊当时看来，是多么的可笑！多么的无知！
<br/>　　但现在，上官昊无比的悔恨，早知今日，他当初无论如何，也要将这个叫做苏莫的土著杀死。
<br/>　　但现在，一切都完了，他将要死在这名土著的手中。
<br/>　　轰隆隆！！
<br/>　　剑气风暴扫荡而过，原地只留下一堆细碎的残渣，上官昊彻底的死亡了。
<br/>　　呼！
<br/>　　望着变成了一堆残渣的上官昊，苏莫微微叹了口气。
<br/>　　结束了！

<br/>　　数年前初临林阳城的上官昊，嚣张不可一世，种下了因，现在则自食其果。

<br/>　　少倾，苏莫催动了体内的吞噬武魂，上官昊残碎的尸体中，一缕缕血气涌向苏莫。

<br/>　　远处，阴阳神宫的弟子，以及黄泉魔宗、殷家和北堂世家的强者，眼见苏莫斩杀了上官昊，顿时吓得急速逃窜，再也不敢留下来观战。
