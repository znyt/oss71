　　天色逐渐明亮起来，疾驰在昏暗的森林中，雨歇风止，清晨的第一丝光亮，透过天空中浓浓的云层和死泽中飘荡的雾气，倾洒而下。
<br/>　　森林中，四周一片寂静，随着一丝光亮，渐渐的开始喧闹起来，不知名的地方，莫名的响起一声鸟鸣。
<br/>　　顿时，随着枝叶间倾洒下的一道道光辉，整个森林像是从夜的沉眠中苏醒了过来，或远或近，到处是晨风里的欢叫声，仿佛在迎接新一天的到来。
<br/>　　听到鸟鸣声，慕清霄目光一凝，剑眉皱在一起，回想起剧情中的一句话‘金芒现世，黄鸟必出’，而这所谓的黄鸟，乃是上古神兽——九天灵鸟！
<br/>　　此刻的森林间，到处都飘荡着薄如白纱的雾气，与前几日死亡沼泽的气息明显不同，疾驰在期中，一路下来，毒虫野兽也不见了踪影。
<br/>　　估计是因为先前的惊天异象，想必都吓跑了吧。
<br/>　　前方数十仗的地方，就是昨晚那道金色光柱的方向，慕清霄搂着陆雪琪飘然落地，脚步也放慢了下来，因为前方雾气变得浓厚起来，迷迷茫茫，让人看不真切。
<br/>　　只是这雾气却和内泽边缘的瘴气有所不同，颜色并非灰色，而是纯白的模样，放眼望去，只见雾气在林间轻轻飘荡，一丝一缕，层层叠叠。
<br/>　　慕清霄也不说话，牵着陆雪琪向内部走去，很快两人就被纯白色雾气吞没。
<br/>　　走在林间，慕清霄一边寻找天帝宝库的位置，一边观察这林间的颗颗巨木，倒也并非是什么罕见罕闻的奇树，其中有橡树，枫树，槐树等等。
<br/>　　哪怕换做死亡沼泽外的普通山间，这些树也数不胜数。
<br/>　　但奇就奇在这里的各种树木都特别巨大，寻常的只要有它们一半大小，便已经令人惊愕，更何况这么多巨树全部聚集在一起。
<br/>　　更奇怪的是，还是这些巨树所在的位置，本应该是生机盎然，但这片迷雾之下，如今非但看不到一只动物，连刚进来是还偶尔能见到的荆棘灌木，也全都不见了踪影。
<br/>　　甚至周围地面上，除了偶尔露出地面的巨树树根，就是结实而微黄的泥土，竟然连青草也没有一株，实在诡异的紧。
<br/>　　在如此诡异的领域内，感受到慕清霄手心的温度，陆雪琪顿时觉得安全感十足。
<br/>　　慕清霄面色平淡，历经沧桑的他，对待这种情况，已经能够保持镇定，心中连一丝波澜起伏都没有。
<br/>　　停下脚步，沉吟许久，环顾四周，只见颗颗巨树参天，笔直高耸，他们两人在林间漫步，仿佛是陷入了一个巨大的迷宫中，他脚尖轻点，搂着陆雪琪就腾空而起，继续深入其中。
<br/>　　既然来了，就没有空手而归的道理，管他前方有何魑魅魍魉，挡我者皆一剑斩之！
<br/>　　前进速度明显比在外围的时候慢上许多，毕竟天帝宝库很有可能就在附近，慕清霄并没有飞到森林上空，而是距离地面六尺，御空而行的同时，也不忘仔细搜索。
<br/>　　在这诡异的森林中，慕清霄可不敢随意释放神识，神识是一个人的神魂力量，若是神识遭到重创，他本人也会因此受伤。
<br/>　　毕竟，他根本不清楚其中的情况，死亡沼泽里诡异程度无法揣测，若是随意将神识释放出去，岂不是送上去给别的东西攻击。
<br/>　　触碰到禁制，引动阵法，后果会更糟糕！
<br/>　　而且，各门各派都有专门对付精神力量的法宝，哪怕在外界，他都不敢随意将神识全部释放出去，更何况天帝宝库所在重地，稍有不慎，都有可能葬身于此。
<br/>　　所以说，神识有利也有弊，运用的好，绝对像外挂一样，运用的不好，死的就早！
<br/>　　时间悄悄过去，树林里依然是一片寂静，陆雪琪趴在慕清霄怀中，沉默不语，手腕上的合欢铃，偶尔响起几声清脆悦耳的铃声。
<br/>　　树林中的树木，随着慕清霄的渐渐深入，躯干越来越是巨大，此刻映入两人眼中的，多半已经是不可思议，粗壮到六七个粗壮汉字才能合抱的古木。
<br/>　　不想也能知道，这里的树木，怕都已经有了上千年的寿命！
<br/>　　在一片越来越是浓厚而诡异的气氛中，慕清霄搂着陆雪琪悄无声息的落在枝干上。
<br/>　　此刻已经天亮了许久，在这里的迷雾，却似乎根本没有散去的迹象，仿佛从亘古以来，这诡异的迷雾就和这片森林共存一般。
<br/>　　站在树林的最深处，两人立于细小的枝干上，向前望去，挡在面前的，居然是一道耸立着的墙壁，是一片木墙！
<br/>　　粗糙的树木纹理，坚硬而带着一丝丝裂痕，从迷雾深处突然深处，高达三丈的一道木墙，如虬龙一般强健有力横在巨木林中，深深如入泥土之中。
<br/>　　慕清霄仔仔细细的注视着眼前的木墙，陆雪琪也好奇的打量着眼前的一幕，片刻后，他嘴角狠狠的抽搐几下，身形已经来到木墙旁边，并且尝试用手抚摸着墙壁。
<br/>　　触碰的一霎那，传来的是温和而粗糙的感觉，慕清霄心中隐隐想到了什么，眼中弥漫着浓郁的不可置信之色，收回手掌，搂着陆雪琪顺着木墙向前飘去。
<br/>　　“雪琪，在传闻中，宝库居于亿年神木之上，我本不信，现在看来，无风不起浪，传闻也并非全部虚假。”
<br/>　　陆雪琪美眸望了他一眼，有些不明所以。
<br/>　　白色的雾气在面前渐渐散开，又在身后慢慢凝结，面洽前的这道木墙越来越高，也越来越是粗壮，渐渐变成了圆形形状，而高度也在缓缓上升。
<br/>　　终于，在高度达到六丈的时候，两人终于来到这道木墙的终点，慕清霄也轻飘飘的落在其上。
<br/>　　然而，饶是慕清霄心性沉稳而坚定，却依旧无法压下心中的震撼，原本波澜不惊的眼眸中也泛起一阵涟漪，在不可仰止的心跳中，眼前的一切，穿过了迷雾，终于呈现在他的眼前。
<br/>　　陆雪琪抬起小脑袋，向上望去，纤纤玉手捂着粉唇，简直不敢相信眼前的一幕，冲击性实在太大，两人立于木墙上，久久无言。
