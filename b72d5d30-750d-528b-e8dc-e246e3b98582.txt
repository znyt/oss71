　　杭山市，近郊别墅区。
<br/>　　这里虽然不是杭山市市区，但是这里的房价可丝毫不比市区的便宜。
<br/>　　价格不便宜，可是销售情况却很好，一期开盘的时候，不到一个礼拜的时间，就销售一空，而且来这里买房的，很多都是其他城市来这里或是投资、或是买来度假用的。
<br/>　　从这个别墅区到市区有五十多公里路，而且附近没有地铁和直达公交，但这些都不叫事。能在这里买下一套上千万别墅用来投资或者度假的人，谁还差一辆车啊。
<br/>　　李霄力在这里就有一套别墅。李霄力对这套别墅做了很大的改动，包括偷挖了逃跑密道，做了暗间用来藏枪械等等。李霄力平时不在外面做事情的时候，就会回到这里居住，而且跟三儿不同，李霄力对女人不是很感兴趣，不是那种恨不得整天趴在女人肚子上不动弹的人。
<br/>　　女人，对于李霄力来说并不是一个必需品。即便是需要，他也是带着女人在外面办事，不会带到这栋别墅里来。
<br/>　　李霄力手中把玩着手机，思考过后，从抽屉中拿出一部卫星电话，拨通了一个号码。
<br/>　　……
<br/>　　南美洲，“cocaine”王国，布克尔特所控制的cocaine原料产地村庄。
<br/>　　这个村庄里唯一的农作物就是cocaine，田地里除了cocaine外，再也没有其他农作物。这个村庄的田地里只能种植cocaine，也只允许种植cocaine！
<br/>　　村庄西南角，那里有几栋破损的房屋和一个靶场。
<br/>　　布克尔特是职业军人出身，所以他对于自己手下人也是采取准军事化管理。他不介意自己手下那些人玩女人，但是严令禁止他们强奸，只要被发现，会被立刻枪决，绝不姑息。
<br/>　　而且布克尔特对手下武装人员的训练也十分严苛，这几栋破损的房屋就是布克尔特所设计的“简化版恐怖屋”，供手下武装人员进行CQB训练。
<br/>　　因为布克尔特军事化的管理和训练，他手下的武装势力跟南美洲其他贩毒组织的武装势力相比，更像是一支训练有素的军队，这也使得布克尔特在南美洲这片辽阔无比的土地上，有着很大的名气，没有人愿意随意招惹他。
<br/>　　靶场上，布克尔特拿着一支M4A1战术步枪对着竖在靶场上的钢靶进行射击，子弹打在钢靶上发出“当当当”的声响，弹头撞击钢靶，火星四溅。
<br/>　　一个弹匣只打了一半，布克尔特就把枪放下，卸掉弹匣，退出枪膛中的子弹，看了眼站在自己身后的手下，问道：“这把枪是谁校准的？”
<br/>　　站在靶场外的武装人员沉默不语。
<br/>　　布克尔特提高嗓音，再次问道：“我再问一遍，这把枪是谁校准的！？”
<br/>　　隔了几秒钟，一名武装人员从跨出一步，走出队伍，说道：“是我校准的，sir！”
<br/>　　“你是第一次校枪吗？”布克尔特问道。
<br/>　　“No，sir！”
<br/>　　“你会校枪吗？”布克尔特再问。
<br/>　　“Yes，sir！”
<br/>　　布克尔特把手里的M4A1战术步枪仍在他面前，走上前，一脚把那名武装分子踹倒在地，怒声道：“会校枪就是这样的？就是一个从没摸过枪的孩子，我教他一次都比你做的好！这是你第一次，也是最后一次，下次如果再被我发现工作不认真，我会把你挂在杆子上当枪靶，到时候，你最好希望我手里的枪是你校准的，这样你还能再多活几秒钟。现在，立刻背着负重背囊五十公里武装越野，跑不完不准吃饭，不准休息，不准喝水！”
<br/>　　“Yes，sir！”
<br/>　　那名武装分子从地上爬起来，连身上的尘土都没拍，拎起放在旁边的负重背囊，离开了靶场。
<br/>　　“我说过很多遍，在战场上，你们手里的武器就是你们第二条生命，对手里的武器不负责就是对你们自己的生命不负责！”布克尔特如鹰一般犀利的眼睛扫过站在靶场外的手下，厉声说道：“同样的事情，不想再看到第二次！”
<br/>　　“Yes，sir！”
<br/>　　在训练过程，布克尔特就是他那些手下的教官，回答问题也只有“yes，sir”和“no，sir”两种。
<br/>　　“Boss！”
<br/>　　黑人武装毒贩Tim拿着卫星电话跑向布克尔特，把电话递给他，说道：“李霄力来的电话。”
<br/>　　“李霄力？”
<br/>　　布克尔特皱了皱眉，拿过Tim手里的电话，接听起来，“喂？”
<br/>　　“布克尔特先生，我这里有一则消息，我想你会非常感兴趣。”李霄力开门见山的说道：“你让我留意的那个女人佟千雪，下个月会去T国旅行。”
<br/>　　“你确定？”布克尔特有些怀疑的说道。
<br/>　　“我不确定，这是我手下给我的消息。”李霄力对布克尔特说道：“在T国，你应该有你的人手，只要留意一下应该就能知道是真是假。”
<br/>　　“很好，十公斤高纯度cocaine我会以上次的方法送到，到时候会有人跟你联系。”布克尔特说道。
<br/>　　“布克尔特先生果然信守承诺，跟你这样的人做生意是件很愉快的事情。”李霄力捧了布克尔特一句。
<br/>　　花花轿子人人抬，好听的话都会说。
<br/>　　“李先生，听说你最近在T国活动很频繁。”布克尔特对着李霄力说道：“你不觉得自己的手伸得有些长吗？”
<br/>　　果然来了！
<br/>　　李霄力在打这个电话之前，他就料想到布克尔特会质问自己，自己的动作对T国的市场造成了一定冲击，普提查没道理不告诉布克尔特。
<br/>　　“要说手伸的长，布克尔特先生的手伸的可要比我长多了。”李霄力手里把玩着一个全球限量版的zippo打火机，说道：“你可是在南美洲，而我在亚洲，谁的手伸得更长应该是很显然的事情。”
<br/>　　“李先生，做事情讲究先来后到，你这么做不符合规矩。”布克尔特语气中多了一些怒意。
<br/>　　“布克尔特先生，蛋糕一共就这么大，要想吃多吃饱，只能去抢别人的，这个道理你不应该不知道吧？抢市场这样的事情，我相信你也没有少做，不然你现在掌控的那些市场又是哪里来的？”李霄力对着布克尔特反问道。
