        &lt;h3&gt;第三卷第一百四十六章  三个女人一台戏&lt;/h3&gt;
<br/>
<br/>        陈子昂和陈子霖回西城去了，这就忙坏了办公室主任宁雅。
<br/>
<br/>        陈子昂走之前交代了要她时刻关注着耗材和信息部，这一段时间陈子昂都不出差了，在公司盯着这两个部门加班加点。
<br/>
<br/>        拼命三娘的名号不是白捡的，陈疯子也不是别人给她强加的，每天基本都和员工一起，尤其是同赵绾绾他们一起，和他们探讨了软件怎么写，思路怎么进行。
<br/>
<br/>        宁雅都不知道陈子昂怎么会那么多东西，她对那些东西是一窍不通的。
<br/>
<br/>        让她去给他们传达一个信息她是可以的，对他们所说的那些术语，她是一个也不懂的，所以她对赵绾绾充满了好奇。
<br/>
<br/>        去信息部的时候，路过耗材部，她顺道去看了看耗材的情况，看见李凌菲和倪小美因为运营方案在争执。
<br/>
<br/>        俩个人的思路大致相同，但是在小细节上有所不同，于是俩个人就细节问题展开了争论，俩个人都有道理，谁也说服不了谁。
<br/>
<br/>        宁雅说：“既然你们俩个都有道理，方案提交给我，我上会研究，看看大家同意那种方案吧。”
<br/>
<br/>        “好吧。”俩个人都没有说什么，但是看出来，还是有点别扭。
<br/>
<br/>        “你们俩人是耗材部的领导，如果你们俩人的意见不统一，你们部门人的思路都会被你们所左右。所以你们以后思路一定要统一。”宁雅看着他们严肃的说。
<br/>
<br/>        “好的，宁主任，我知道了。”李凌菲因为从西城来不久，所以还没有完全融入到这个圈子里。
<br/>
<br/>        倪小美干脆就不说话。
<br/>
<br/>        宁雅看着气氛微微有些尴尬，于是对李凌菲说：“我去看赵绾绾他们，你要去吗?”
<br/>
<br/>        “好，我也很想过去看看呢。”李凌菲知道宁雅是给她找台阶下，便和宁雅一起去看赵绾绾。
<br/>
<br/>        “宁主任，谢谢你，我这才来不久还需要和同事们多沟通。”李凌菲有些难过，毕竟来这里时间不长，而且耗材对她来说完全是一个陌生的领域。
<br/>
<br/>        “其实我们公司的氛围还是好的，你和倪小美之间只是因为工作的原因，这个其实是很好沟通的，有的公司各种斗，什么样的事情都有。”
<br/>
<br/>        “这个我是知道的，只是我对耗材也不是很了解，工作起来还是多少有些困难的，毕竟跨行了。”
<br/>
<br/>        “凌菲，其实我们公司开始谁都对耗材不懂，陈总跑了一趟南方回来，就让皇甫双双、倪小美他们负责，他们当时也是不懂的，干到这么长时间也算是熟悉了，人说活到老，学到老，谁是天生下来就会的？”宁雅安慰着李凌菲。
<br/>
<br/>        “我来公司其实只有两年，听同事们说陈总白手起家，你看看我们公司现在的规模，在这个公司上班都是骄傲呢，好好干，陈总可是想将耗材推出海外的，你和倪小美担负着重要的责任呢。”
<br/>
<br/>        “其实我也没有什么，可能和倪大哥的思路还没有统一吧。”李凌菲心里还是很感动的。
<br/>
<br/>        “我们公司无论是男同事还是女同事都被陈总感染着，每一个人工作都一丝不苟，兢兢业业，你再呆一段时间你就知道了。”
<br/>
<br/>        “我已经感觉到了呢，在西城其实我们要求也很严格的，要不然也不会一个项目一个项目的拿。”李凌菲提起西城，好怀念在西城的时光。
<br/>
<br/>        “你和陈副总一起调过来，证明陈总对你们很看重，她明年要去外地拓展业务，这个已经不是秘密了，大家都在努力，想等陈总走之前让陈总安心的去外地拓展业务，所以最近一段时间大家都鼓着一股子劲干活。
<br/>
<br/>        倪小美是公司里的老员工了，据说陈总开公司的时候就来了，你想想他对公司是一种什么样的感情存在，他不容许有一点点的差池也是正常的，还有我带你去见的赵绾绾，工作是拼了命的，前一段时间生病住院了，都没有怎么休息就开始上班，这一段时间陈总也是陪着她加班加点的。”
<br/>
<br/>        “嗯，我都看到了呢，公司上上下下一条心，真好！”
<br/>
<br/>        “诺，前面就是信息研发中心了。”俩人走到一栋大楼前。
<br/>
<br/>        李凌菲还是第一次到研发中心，这是一栋15层的大楼，门都是指纹识别的，没有权限的人是进不去的。
<br/>
<br/>        宁雅有指纹，因为宁雅会经常往来各个部门之间，宁雅刷了指纹，对李凌菲说：“信息研发中心是要保密的，他们基本都在这个大楼里工作，也有宿舍，你都不知道，他们写程序有时候都是没日没夜的。”
<br/>
<br/>        进了大楼之后，是一面双层玻璃门，宁雅站着，玻璃上出现了一个人：“宁主任，你带的是谁？”
<br/>
<br/>        “耗材运营部的李部长，我和她一起来看看赵部长。”宁雅看着大屏幕说。
<br/>
<br/>        “你稍等！”大屏幕上的人不见了。
<br/>
<br/>        屏幕上出现了李凌菲的信息和照片后门开了。
<br/>
<br/>        “这就是高科技，每次来信息中心我就感觉置身在科幻世界一样。”宁雅笑着说。
<br/>
<br/>        进了玻璃门，宁雅在门口的鞋柜上拿了一双鞋套，对李凌菲说：“换上鞋套，防静电。”
<br/>
<br/>        俩个人轻轻的走过好几个房间，到了一个办公室门口，宁雅按响了门铃。
<br/>
<br/>        “宁雅，你怎么来了？”赵绾绾看见宁雅，又看了一眼宁雅身边的李凌菲。
<br/>
<br/>        “这个是耗材部的李凌菲。”宁雅介绍着。
<br/>
<br/>        “来，凌菲，进来吧。”赵绾绾笑着说。
<br/>
<br/>        “哇哦，你们认识？”宁雅惊叫着。
<br/>
<br/>        “你傻了，最近是不是失恋了？记忆力不好了？听说失恋的人总是丢三落四。”赵绾绾笑着说。
<br/>
<br/>        “我们开会的时候见了好几次。”李凌菲也笑着说。
<br/>
<br/>        “哎呀，绾绾，你好讨厌，被你说中了，来，借你的肩膀让我靠靠。”宁雅一副被揭开了心思伤心的模样。
<br/>
<br/>        “算了，我最近身体素质不好，你还是靠凌菲吧，凌菲比我强壮一点。”赵绾绾故意嫌弃的看了一眼宁雅。
<br/>
<br/>        “你讨厌，我哪里强壮了？”李凌菲站在赵绾绾的身边比了一下，“唉，是比你强壮一点，我也想娇小玲珑。”
<br/>
<br/>        三个人数宁雅个子高，一米七二。
<br/>
<br/>        “唉，我们两个你都靠不住，你还是靠你的男神去吧。”李凌菲看着宁雅的身高，她才齐宁雅的耳朵。
<br/>
<br/>        “哪有男神？男神都飞走了。我的男神已经飞走了，别的那呀呦别的那呀呦，我的男神已经飞走不回来。”宁雅笑着改了歌词。
<br/>
<br/>        “看她嘚瑟的样子，这叫飞走了？”赵绾绾笑着说“我们单身我们快乐，狗粮吃到饱，心情也很好。”
<br/>
<br/>        “就是嘚瑟！”李凌菲那一点点郁闷一扫而空。
<br/>
<br/>        “绾绾，我们就是来看看你，我们走了，不打扰你工作了。”宁雅拉着李凌菲离开了。
<br/>
<br/>        她心中真的有男神，也是真的飞走了。
<br/>
<br/>        幸福就在她的身边溜达了一圈，然后就离开了，让她猝不及防，狼狈得来不及收场。
