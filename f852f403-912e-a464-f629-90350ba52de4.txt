　　这一股灵力的突然打出，没有任何人察觉得到，无声无息之间的出手了，向着楚风眠压迫而来。
<br/>　　随着这一股灵力从天而降，一股雄厚到极致的天地之力，突然降临了。
<br/>　　在这天地之力的面前，哪怕是九劫古帝，仿佛是都成为了蝼蚁一般。
<br/>　　轰隆隆！
<br/>　　这一股灵力，仿佛是不知道从哪一方虚空深处之中，突然爆发而出的一样，撕裂了无数空间，向着楚风眠轰击而来。
<br/>　　天地震动，剑道门之中的空间，本都是被特殊凝炼过的，坚固无比，可依然是在这一股灵力之下，被震的粉碎。
<br/>　　这种力量，已经是超逸了帝尊境的范畴。
<br/>　　有半步天人境的强者出手了。
<br/>　　属于七大宗门一方的半步天人境强者。
<br/>　　楚风眠早就有着猜测，以七大宗门的底蕴，其中不可能是会没有半步天人境的强者了。
<br/>　　不过就像是圣龙秘境之中许多半步天人境强者，都是闭关不出一样，只怕是七大宗门之中的半步天人境强者，平日里也都在闭关，参悟天人境。
<br/>　　不到万不得已的时刻，不可能会轻易动身的，毕竟这等强者，许多人的寿命，都已经是达到了无比极限的地步。
<br/>　　一旦是出手，有所受伤，可能终生都无法再度踏入天人境了。
<br/>　　真正踏入天人境的难度，比起踏入半步天人境的时候，还要难上百倍，千倍。
<br/>　　在如今大罗天星域之中，满打满算，半步天人境的强者，也至少有着十位。
<br/>　　可真正的天人境，却是一位都没有，哪怕是号称最有希望踏入天人境的少帝，甚至是云尊，都没有真正踏入天人境。
<br/>　　以少帝的妖孽资质，都没有能够踏入天人境，被卡在半步天人境的巅峰，无法再进一步，便是足以知道这跨越天人境，到底是多么困难的一件事。
<br/>　　为了踏入天人境，许多半步天人境强者，都是躲藏了起来，不打算冒任何一点点的风险。
<br/>　　这一次居然是有着半步天人境的强者，前来出手，对付楚风眠了。
<br/>　　看来上一次楚风眠出手，灭杀了三大宗门强者的事，已经是被七大宗门所知道了。
<br/>　　所以这一次，为了毁灭剑道门，不止是派出了足足十五位帝尊无敌的人物，就连半步天人境的强者，也随之而来了。
<br/>　　不过看样子，这一位半步天人境强者，本是不愿意出手的，只有是看着其他人，不是楚风眠敌手的时候，才选择出手。
<br/>　　这一次看来，七大宗门为了对付楚风眠，也是花费了血本，就连半步天人境强者，都给请出山了。
<br/>　　这一位半步天人境强者，既然是选择出手了，便是一出手，就是绝对的杀招，向着楚风眠轰击而来。
<br/>　　这一股灵力打出的一刻，爆发出来的力量，却是纯粹的蛮力。
<br/>　　这种蛮力，看似愚蠢，可实际上却是最为厉害的，因为一切手段，都难以抵挡住这灵力，想要抵挡，唯有以力碰力这一条路。
<br/>　　七大宗门的强者，也知晓了楚风眠的手段众多，想要借助手段对付楚风眠，几乎是不可能的事情，倒不如直接出手，以雷霆之势，灭杀了楚风眠。
<br/>　　“荒古大戟！”
<br/>　　这一股灵力，在逼近到楚风眠头顶之上的一刻，突然间的凝聚了起来，随即化为了一根长戟，猛然对着楚风眠劈砍而下。
<br/>　　荒古大戟！荒古门的强者！
<br/>　　这荒古大戟一出，这一尊半步天人境强者，到底是属于哪一方势力的，如今便是一清二楚的。
<br/>　　的确是在七大宗门之中，对楚风眠最为痛恨的，当属荒古门了。
<br/>　　毕竟楚风眠可是曾经偷偷潜入到荒古门的神树秘境之中去，将神树盗取走。
<br/>　　可以说现在楚风眠的成就，跟神树有着极为密切的关系，没有神树，只怕是楚风眠一路以来，都死了无数次了。
<br/>　　这样一件至宝，本该是属于荒古门的，现在却是落入到了楚风眠的手中，也怪不得荒古门对楚风眠如此痛恨。
<br/>　　荒古大戟一出，这一尊荒古门半步天人境强者，也现身出来，加持灵力在荒古大戟之上，对准楚风眠斩杀而去。
<br/>　　要这一击之下，将楚风眠所斩杀。
<br/>　　要是换做之前，楚风眠碰到这半步天人境的强者突然出手，便是只有退避这一条路。
<br/>　　可是这一次，楚风眠已经踏入了九劫古帝。
<br/>　　就算是半步天人境的强者，楚风眠也没有丝毫的惧怕。
<br/>　　楚风眠抬起头，看向那落下的荒古大戟，冷笑一声，却是不闪不避，以手中剑锋，迎了上去！
<br/>　　轰！
<br/>　　从楚风眠的剑锋之上，爆发出惊天剑意来，冲天而起，跟那荒古大戟，碰撞在了一起，爆发出无尽的威能。
<br/>　　这其中碰撞的力量，简直是跟两尊半步天人境强者交手，没有丝毫的区别。
<br/>　　一旦是被这力量卷入其中，哪怕是九劫古帝，甚至是帝尊无敌的人物，都可能被余波活活震死。
<br/>　　所有人都是急忙退后，在剑道门之中，留出了一片空场来，谁也不敢向前踏足一步。
<br/>　　不少剑道门弟子，都是抬起头，看着天空之上的战斗，眼神之中都是无比担心。
<br/>　　楚风眠的实力强横，他们自然是知道。

<br/>　　半步天人境，也终究是一只脚，迈入了天人境的存在，跟帝尊境，已经是截然不同了。
<br/>　　纵然是楚风眠的实力再强，可面对半步天人境的强者，无数人还是心中有些惊慌的。

<br/>　　不过他们也知道，这种层次的战斗，已经不是他们可以插手的了的了。
<br/>　　他们出手，对楚风眠来说，只是碍手碍脚罢了，只能够是联手之下，死死的护住剑道门之中的无数宫殿，免得被这余波之力所打破。
<br/>　　“荒古门的功法，果然强横。”
<br/>　　一招碰撞，伯仲之间。
<br/>　　楚风眠也清晰的感觉得出，眼前这一尊荒古门半步天人境强者的实力，绝对是比起那五行刀圣，还要强横几分。
