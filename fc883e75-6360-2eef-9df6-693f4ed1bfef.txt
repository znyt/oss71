　　　　对于现在的陈高阳来说，他所谓的女朋友就宛如昙花一现，没了就没了，好像不曾出现过。
<br/>　　多年相处的女友，留给他的就一个名字，简直滑天下之大稽。
<br/>　　这件事情很奇葩但同时也很蹊跷。
<br/>　　冥冥之中，涂小安嗅到了一丝诡谲之处。
<br/>　　“我可以去你家做客吗”
<br/>　　忽然，涂小安说出这样一句话。
<br/>　　“我家？”陈高阳一怔，没有拒绝的理由，对方帮了他两回，点头答应。
<br/>　　涂小安疑虑的问：“你这样，找的到回家的路吗”
<br/>　　陈高阳嘴角一抹弧线：“不是有这样一句话吗，瞎子眼瞎，但心不瞎，回家的路，怎会找不到”
<br/>　　此话一出，涂小安深深的看了他一眼，对他这句话深以为然。
<br/>　　多少人眼睛不瞎，但心早就瞎了。
<br/>　　他不禁想起了一个恶趣味的笑话。
<br/>　　一个傻子偷了乞丐的钱包，被瞎子看到了，哑巴大吼一声，把聋子吓了一跳，驼子挺身而出，瘸子飞起一脚，疯子说：大家要理智，通辑犯要拉他去公共安全专家局，麻子说，看我的面子算了。
<br/>　　这是一个很莫名其妙的笑话，但要用心去听。
<br/>　　随后，陈高阳带路，涂小安没去扶他，没做声，就默不吭声的跟着。
<br/>　　他跌跌撞撞，时常四处碰壁，可令人诧异的事，他脚下的路非常的鲜明，他们两人的步行很慢，慢极了。
<br/>　　但最后，陈高阳将涂小安带到了一片老旧的住宅区，经过一道幽深的小胡同后，一栋看上去几乎有数百年历史的四合院出现。
<br/>　　“这便是我的家了”
<br/>　　陈高阳脚步戛然而止，涂小安心悦诚服。
<br/>　　他什么都看不到，但其实什么都印在了自己的心间，不多不少，不偏不倚，他恰好站在了自家门前。
<br/>　　“请进吧，这个家很久很久没有来客人了”
<br/>　　陈高阳做了一个请，门槛在前，他一脚抬起迈了过去。
<br/>　　涂小安没有第一时间跟进去，站在外头原地不动，望着眼前精致而质朴的四合院，那黝黑的宅门，那锃亮的门钹，像是明清建筑。
<br/>　　在当代，这样的古式四合院几乎不可见了。
<br/>　　这片住宅区，跟外面繁华的景象形成了一个鲜明的反差，转身放眼望去，老槐树下扯闲篇儿的老人，追逐顽戏的孩子，还有那由远而近，略带沙哑的买卖吆喝声...
<br/>　　好美，好梦幻，悄然的，他以为穿梭了时间岁月。
<br/>　　没想到温城之中，还保留着这样一块古朴的地方。
<br/>　　看了几眼后，涂小安迈过了高高的门槛，一进其中，四四方方的院落刻满风蚀的残痕，古朴却又生生不息地沿着中轴线延展开去。
<br/>　　它宛如香奁宝匣一般散发着馥郁的历史气息，在砖缝瓦隙里留住了旧日的光阴。
<br/>　　能现在还住在这样的四合院，比那些豪华的别墅要让人心旷神怡的多。
<br/>　　“我家这四合院，传到我这一代已经是第八代了”陈高阳的声音幽幽的响起：“这四合院在明清就早就存在了”
<br/>　　涂小安好好欣赏几眼：“能拥有这样的四合院，那想必你家曾经也是大户人家吧”
<br/>　　不过，这句话说出，涂小安觉得有点突兀，因为现在偌大的四合院，既然空无一人。
<br/>　　没有一点人的气息。
<br/>　　“富不过三代穷不过五服”陈高阳淡淡的说：“有兴必有衰，曾经我奶奶说，这栋四合院住着陈家三十多口的人，可到九十年代，就只剩下奶奶跟小的我”
<br/>　　“等我长大了，奶奶也走了，就独独剩下我了”
<br/>　　“不，是只有我跟我女朋友了”陈高阳最后补充了一句。
<br/>　　他墨镜下英挺的眉和鼻梁都似乎透着哀伤，还好他的眼睛看不到，不然一个人住在这里，太大了，太空荡了。
<br/>　　涂小安听着他的话，也没有做什么回复，陈高阳像是很久没找到说话的人。
<br/>　　现在慢慢悠悠的讲，讲这个四合院的故事，涂小安静静的听。
<br/>　　这种建筑，对于他是非常的新鲜的，以前好像只是电视里见过。
<br/>　　古老的四合院，就好像是石砌的史书，记载了不知道多少人的点点滴滴。
<br/>　　那些人都不在了，它为众人保存记忆。
<br/>　　欣赏着，听着，忽然，涂小安一阵蹙眉起来，慢慢的闭上眼睛，高挺的鼻子冲着空气嗅了嗅，嗅到的是散发着馥郁的历史气息，还有一股非比寻常的...精怪气息。
<br/>　　对！
<br/>　　涂小安猛然掀开眼，妖治的眼眸精光四射。
<br/>　　他在这栋充满历史的四合院中察觉到了精怪的气息。
<br/>　　那是觉醒生物残留下，久而不散的浓郁气息。
<br/>　　陈高阳说自己的女朋友一夜之间从他的世界消失，担心她是不是出了什么意外。
<br/>　　如今，这栋四合院中有觉醒生物的气息，而且这种气息还很强。
<br/>　　应该属于二阶层次的觉醒生物了。
<br/>　　莫非是他女朋友遭到了觉醒生物的毒手？
<br/>　　那么陈高阳为什么好端端的。
<br/>　　不，涂小安经历了不少匪夷所思的事情，他想到了另外一个光怪陆离的可能性。
<br/>　　眼睛骤然看向陈高阳，他握拐杖，杵在院落发呆。
<br/>　　不知何时，他没有说话了，转为发呆，带着墨镜的他，很难让人看出他是不是在发呆。
<br/>　　但涂小安一眼便觉得他就是在发呆，宛如一块木头一样，一动不动，陷入某种回忆
<br/>　　涂小安悄然的走到了陈高阳的身边，没有出声，而是伸出一根手指，轻轻的点在了他墨镜上的额头上。
<br/>　　“给我意识笼罩，画面呈现”
<br/>　　这一招，曾经涂小安在蛇胎女人舒瑶的身上就动用过，可以看到当事人此时想之所想。
<br/>　　与此同时，涂小安也闭上了眼睛，去感同身受。
<br/>　　霍然...
<br/>　　一片漆黑如墨，他仿佛一下子陷入了黑暗的深渊中，涂小安微微的蹙眉，什么都没看到，什么画面也没有。
<br/>　　此刻，涂小安才恍然大悟，对于一个天生的瞎子，他何来的画面。
<br/>　　他所有的一切，只能去感知，只能去自我想象。
<br/>　　据说天生的瞎子哪怕做梦的时候也是黑暗的，但是可以梦见听见的声音。
<br/>　　正常人很难体会一个一出生就是瞎子的人，他什么都看不见，那么他脑海中的世界是什么样的，那些人与事，那些动植物，那些房子汽车等等...
<br/>　　想必全靠自我想象。
<br/>　　在瞎子的世界上，没有炫彩霓虹，没有红黄蓝绿青蓝紫，他们只有一种颜色，那么便是黑，漆黑如墨的黑。
<br/>　　无论这个世界何等的美丽多彩，陪伴他们的只有黑。
