　　　　陆雪琪娇躯微微一颤，脸颊涌上一丝红晕，赶忙将房门关上，却没有挣开慕清霄的怀抱，任由他抱着，轻轻的靠在他的胸膛上，声音幽怨。
<br/>　　“白天的绿衫少女，是不是很漂亮？”
<br/>　　慕清霄闻言，嘴角勾起一丝笑意，哪里还不知道她是在吃醋，搂在她腰上的手紧了紧，嗅了嗅她身上散发的幽香，道：“确实很漂亮，可在漂亮也不及你万一。”
<br/>　　慕清霄说的可是实话，美貌的女子他见的多了，碧瑶虽然漂亮，气质也相当出众，但比起陆雪琪来，还是差上了几丝。
<br/>　　听到如此动人的情话，饶是陆雪琪心性沉稳，小心肝也是颤了颤，甜蜜不已，语气温柔，道：“那你还一直盯着她看？”
<br/>　　“雪琪，以后就看你一个好不好？”
<br/>　　“油嘴滑舌。”
<br/>　　慕清霄俯下身子，嘴唇含住她冰凉的粉唇，两人紧紧相拥，缠绵在一起。
<br/>　　“唔唔...！”
<br/>　　足足在一起亲密了半个多时辰，陆雪琪气喘吁吁，美眸迷离，慕清霄恋恋不舍的离开，现在嘴上和手上占点便宜已经是极限了，明天要还要赶往空桑山，初夜还是留到以后吧。
<br/>　　......
<br/>　　......
<br/>　　竖日清晨，旭日东升，青云门五人相继起床，梳洗过后，在西苑门口齐聚，开始商议今日的路程。
<br/>　　“空桑山在东方三千里之远，路程不近，魔教妖人最近动作越来越明显，我们还是赶路要紧。”林惊羽话语刚落，众人附议。
<br/>　　慕清霄对此也没有什么意见，他的目标只有一个，那就是天书，至于从何处进入万蝠古窟都一样，渝州城的碧波亭并不是唯一的入口。
<br/>　　山海苑的老板与曾书书有说有笑的交谈，对青云门果然心存敬仰，原本昂贵的房钱居然打了个对折，几乎与普通客房的费用相同。
<br/>　　慕清霄在四周环视一圈，并没有看到碧瑶等鬼王宗的人，于是就收回了目光。
<br/>　　他们五人御空而行，这三千里的路程足足用了一周的时间。
<br/>　　由于张小凡几人修为都不是很高，长久时间的御空而行对他们来说负荷太大，所以只能赶一段路，休息一些时间，对于在青云门山上生活的他们来说，在荒郊野外过夜并非难事。
<br/>　　半日后，众人终于抵达空桑山，落在一座山巅上，视线在周围环绕一圈，皆是一惊。
<br/>　　只见方圆百里之内，一座大山险峻高耸，但岩石居多，草木稀少，山下更是荒无人烟，视线之内一片荒凉。
<br/>　　此时已近黄昏，日头西沉，夕阳的余晖照射在空桑山上，仿佛带了几分萧索，也有几分可怖。
<br/>　　慕清霄几人都在山脚下收起仙剑，看了看天色，道：“看周围恐怕也没有借宿的地方，来的时候已经耽搁了不少时间，黑夜对我们来说并不碍事，倒不如休息片刻就上山寻找万蝠古窟，你们觉得如何？”
<br/>　　张小凡点头道：“慕师弟说的有道理，我们休息片刻就上山吧。”
<br/>　　林惊羽见张小凡答应，自己也没有什么意见，曾书书倒是收敛起平日嬉笑的模样，脸色变得严肃起来，陆雪琪依旧面无表情的跟在慕清霄身后。
<br/>　　片刻后，众人喝了点水，吃了点干粮，神色警惕的向山顶走去，空桑山虽比不了青云山通天峰那般高的夸张，但也不低。
<br/>　　加上偏僻险峻，无路可寻，他们也只能从山脚下往上走，只是走到山腰处，天色便已经完全黑了下来。
<br/>　　看了看周围的景象，慕清霄开口道：“张师兄，你将掌门赐予的六合镜拿出来......”
<br/>　　众人皆是一愣，不明白他的用意，张小凡没有丝毫犹豫，直接照做，六合镜本就是慕清霄的东西，而且后者神秘无比，说的话自然有他的用意。
<br/>　　随着慕清霄的指示，张小凡将六合镜拿在手中，口中低声咏读了几句咒文，原本暗淡无光的六合镜就有了感应，逐渐亮了起来。
<br/>　　光芒越来越亮，最后从张小凡手中飘了起来，停留在他头顶三尺处，光芒渐盛，形成淡黄色的光圈，将他们周围六尺左右的园地笼罩其中。
<br/>　　慕清霄可没有忘记，当初在渝州城碧波亭看到的蝠妖，夜晚就是蝠妖狩猎的时间，他自己倒是没什么，最主要的还是担心陆雪琪。
<br/>　　有六合镜在，倒也省事。
<br/>　　“空桑山在八百年前，乃是魔教聚集之地，而且此山荒凉诡异，只怕有许多山野精怪，六合镜功能护住，我们也好防范于未然。”
<br/>　　慕清霄话语刚落，众人觉得有道理，张小凡看着漂浮在自己头顶的六合镜，这还是他头一次使用，只见那面小镜貌不惊人，但古拙中隐藏着瑞气，不可小觑。
<br/>　　陡然，就在他们准备再次赶路的时候，远处忽然传来一声巨响，紧接着响起‘噼啪噼啪’的声响，声音逐渐密集，距离此地也越来越近。
<br/>　　从声音中能听出，估计是群居的飞行野兽，只听见‘吱吱’尖锐的声音回响在这荒山野岭，慕清霄嘴角抽了抽，当真诸事不顺，听这声音，明显是蝙蝠妖啊！
<br/>　　靠着六合镜所发出的光芒，众人隐约能看到远处山背后，霍然腾起的黑色云气，在这黑夜中显得诡异无比。
<br/>　　众人骤然色变，慕清霄语气平淡道：“不必担心，六合镜护主功能颇为不俗。”
<br/>　　话语刚落，那片在空中越来越近的黑云仿佛感觉到了什么，快速向这里飞来，随着一阵刺耳的呼啸声，黑云已来到附近。
<br/>　　霎时间，原本星光闪亮的黑夜变得漆黑一片，头顶除了六合镜闪烁着光芒，四周都变成漆黑一片。
<br/>　　下一秒，尖锐刺耳的声音，令人作呕的腥臭味充斥着四周，张小凡几人未曾下山过，哪里见过这种场面，无一不是大惊失色。
<br/>　　慕清霄脸色依旧平淡无比，安慰道：“切记，不要离开六合镜护主的范围，否则凶险难测，它们无法入侵进来，过一阵子就会离开此地。”
