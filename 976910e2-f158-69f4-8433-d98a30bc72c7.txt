        &lt;h3&gt;第三卷第三百二十二章  送烟&lt;/h3&gt;
<br/>
<br/>        凌辰和董亚子在西城的暗访以失败告终，因为他们的事情还给陈子昂的公司带来了一点麻烦，让凌辰和董亚子心里有些愧疚，俩个人都有些气馁。
<br/>
<br/>        董亚子准备返回贵州，将这件事情告诉她的父亲，他们现在知道了假货的源头，却苦无证据。
<br/>
<br/>        “我要回贵州，你看看你去哪里？”董亚子问凌辰。
<br/>
<br/>        “我等杭州的消息，如果杭州有动静了，刘诚之那边应该就轻松了，原本想在西城搞个大事情，没有到被老鹰给啄了，特么的，我不将他们一网打尽我就不姓凌！”凌辰一拳头砸在办公桌上，疼得他自己呲了一下牙。
<br/>
<br/>        “杭州有什么事情？”董亚子不明所以。
<br/>
<br/>        “等明天看看有没有消息，我先送你回贵州，然后我也去甸城，我就不信何少华他们有通天的本事！”凌辰心里很憋屈，这样被人摆一道确实让他感觉很恶心。
<br/>
<br/>        不过这一次确实是他们太大意了，都没有想到会碰巧被顾恩赐看到。
<br/>
<br/>        “我已经买好了回贵州的机票，你要是去了我就给你将票买了。”董亚子看了一眼凌辰，现在生气也起不了什么作用。
<br/>
<br/>        “行，你给我买机票，我给刘诚之发个邮件。”凌辰坐下来打开电脑给刘诚之发了一个邮件：“三月幼苗西城现，杭城水雾碧连天，要问源头从何起，有兽有鱼有暗潭！”
<br/>
<br/>        发完邮件，凌辰拿起一支烟，点着，深深的吸了一口，才想起来要给小头目李祺送烟，不行了找个人送过去，后来想了一想对董亚子说：“我要出去一趟，给人送两条烟。”
<br/>
<br/>        “收货那个部门？”
<br/>
<br/>        “亚子真是人美又聪明！”
<br/>
<br/>        “你还厉害的不行，人家会收吗？”
<br/>
<br/>        “两条烟而已，我明天就走了，他不会多想的。机票买好了吗？”凌辰找出来一个手提袋，将烟装进手提袋里，掂了掂。
<br/>
<br/>        “快去吧，明天的。”
<br/>
<br/>        “唉，明天是陈子昂他们公司全店联动，应该是一个盛况。”凌辰叹了口气。
<br/>
<br/>        “我买的是下午，能看到的。”董亚子微微一笑。
<br/>
<br/>        “我就想看看我家小雅，我好久没有见到小雅了，这几天忙的都没有顾上和我家小雅聊天。唉，什么时候美人在怀，事业不用我操心就好了。”凌辰摇了摇头。
<br/>
<br/>        “美得你，不努力你那里来的事业？不努力美人也会飞走的。”董亚子像看白痴一样的看着沉浸在爱情幻想之中的凌辰。
<br/>
<br/>        “唉，我挺羡慕刘诚之那个憨憨，就随我去了一趟贵州，不但美人在怀，还有家产可得，你们这叫强强联手啊，靠北，你说那个憨憨为什么命那么好？”凌辰摇了摇头。
<br/>
<br/>        “嗯哼？那我现在给你一次机会，你再重新选择一下？”董亚子微笑着看着凌辰，那笑让凌辰一个哆嗦。
<br/>
<br/>        “我错了，亚子，刘诚之那个憨憨命好，我的命也不赖，我家小雅没得说，你们都是女中豪杰。”凌辰急忙认错。
<br/>
<br/>        “哼，凌渣男，我告诉你，我原来喜欢你就是喜欢你认真的对待感情，要是你三心二意你就死定了，记住了，以后要好好对待我小雅姐姐，要不然......哼!”董亚子点着头。
<br/>
<br/>        “亚子，你晓不晓得，你凌哥哥我也很欣赏你这火爆脾气、敢爱敢恨的性子，我要是有你一半的性格，我就将你小雅姐姐追到手了。”凌辰笑了一下，还好醒悟得早，要不然以后真的会很后悔。
<br/>
<br/>        “别在这里灌迷魂汤了，赶紧去送东西。”董亚子挥了挥小手，将赶凌辰走。
<br/>
<br/>        凌辰出门碰到了陈子霖：“凌总，你这是要去哪里？”
<br/>
<br/>        “我去给那个李祺送两条烟。”凌辰摔了摔手中的手提袋。
<br/>
<br/>        “哈哈哈，你这无孔不入啊，什么人都能被你搞定？”陈子霖哈哈大笑。
<br/>
<br/>        “这个人以后可能会见面的，还没有坏到骨子里，可以拉一把！”
<br/>
<br/>        “要我送你吗？”陈子霖也准备出门，手中拿着车钥匙。
<br/>
<br/>        “行，我们一起去看看。”
<br/>
<br/>        “这一次是我疏忽了，让大家的心血功亏一篑，唉，竟然将顾恩赐这货给忘记了，我们之前有过合作，这人吧，太善变了。”
<br/>
<br/>        “不，是我给你们添麻烦了，我们其实应该还有别的方法，俗话说吃一堑长一智，这一次也是给我提了个教训，后期我会更加小心的。”
<br/>
<br/>        “顾恩赐的老婆其实权利还挺大的，所以这一次能顺利的将货送回来，江俞轩是花了一番功夫吧？”陈子霖不知道江俞轩和凌辰去的情况。
<br/>
<br/>        “江总就说了几句话，顾恩赐估计权衡利弊之下就放弃再折腾下去了。”
<br/>
<br/>        “哈，江俞轩又发挥他那舌战群雄的潜质了？很好奇！”
<br/>
<br/>        “其实他就给顾恩赐分析了一次情况！”
<br/>
<br/>        “我们公司有句话：俞轩出马，一个顶俩！这人说话能直击人心，让你还找不到毛病。”
<br/>
<br/>        “你这大舅哥竟然如此的夸自己的妹婿，看来江俞轩的工作做得不错啊，你们家里除了陈子昂之外都被他搞定了吧？”
<br/>
<br/>        “说什么话呢，江俞轩都已经结婚了，可别乱说，我妹妹还要嫁人的。”陈子霖皱了一下眉头。
<br/>
<br/>        “啊，你说什么？”这么多年，凌辰一直以为江俞轩和陈子昂是一对，陈子霖爆了这么大一个料。
<br/>
<br/>        “你是怎么搞定那个小头目的？”陈子霖岔开了话题。
<br/>
<br/>        “我也就说了几句话，我说我和他是同一种人，都是被人踩压着的，身不由己！”
<br/>
<br/>        “哈哈哈哈，你那里被人踩压了？”
<br/>
<br/>        “不是你吗？哈哈哈！”
<br/>
<br/>        “他肯定说：我才不会和你是同一种人呢，奸商！”
<br/>
<br/>        “你怎么知道？”
<br/>
<br/>        “因为他对我说过啊！”
<br/>
<br/>        俩个人说说笑笑的开着车到了某部门，敲开李祺的办公室，李祺正皱着眉头打电话，看见凌辰和陈子霖显出了吃惊的样子。
<br/>
<br/>        凌辰和陈子霖又退回到门口，等李祺打完电话才进去。
<br/>
<br/>        “你们又来干什么？货不是已经送回去了吗？”李祺不知道凌辰他们来干啥。
<br/>
<br/>        “我们是来感谢你将货给我们送回去了，谢谢你！”陈子霖说。
<br/>
<br/>        “是啊，我明天就要离开这里了，我给你拿了两条烟，来向你辞行了。”凌辰拿出了烟。
<br/>
<br/>        李祺的脸上露出了一丝愧疚：“不是，我什么都没有做，你们不用感谢我，我还.....将你们的货收了！”
<br/>
<br/>        “你心怀正义，明辨是非，只是身不由己而已，我们不怪你，相反还很感谢你，要不是你我们还不知道会怎么样呢。”
<br/>
<br/>        李祺的脸一红，这些人其实很善良，而且很宽容，如果以后有机会他坚决不助纣为虐！
