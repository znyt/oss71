
<br/>　　<br/>　　    王惊蛰把未来老丈人喝懵逼了之后，这饭自然也就吃完了，茅清水搀着李兰荣回房睡觉去了，小草自己收拾着残局，王惊蛰在一旁喝着茶水醒醒酒。<br/>　　<br/>　　    半个多小时后，小草摘掉围裙，收拾利索了，说道：“洗脸，刷牙，睡觉……”<br/>　　<br/>　　    “唉，好叻！”王惊蛰干脆利索的回了一句。<br/>　　<br/>　　    洗漱完了，在李家大宅的后院，王惊蛰骚了吧唧的跟在小草身后，问道：“啊，那个什么，我睡哪啊？”<br/>　　<br/>　　    “多好笑啊，睡房间里呗，你难不成还想睡外面嘛？”<br/>　　<br/>　　    “我是想问，睡哪个房间”王惊蛰喷着酒气，眼珠子直冒火星的说道。<br/>　　<br/>　　    茅小草盯盯的看着他，忽然伸手掐着他的脸蛋子，拉到自己身前，眯眯着眼睛问道：“盆友，你什么想法啊？”<br/>　　<br/>　　    王惊蛰舔了舔嘴唇：“想法肯定没有，主要是因为我怕黑！”<br/>　　<br/>　　    “哥们，我是不是太长时间没有和你谈过关于家教的问题，导致你自己有点分不清轻重了？”<br/>　　<br/>　　    “咱俩一别多时，每到夜间我都饱受相思之苦，特别是看到天边那一轮残月的时候，我都……”王惊蛰正神情告白的时候，对面一间房里，茅清水突然推开房门，说道：“你俩还不睡觉在那秉烛夜谈呢？要说话回房间里说去，小草你的房间我已经给你收拾出来，全换新的了，早点去休息吧！”<br/>　　<br/>　　    王惊蛰“唰，唰”的眨了眨眼睛，哎呀我去了一声，这丈母娘对他好的比上一次明显是来了个弯道超车啊，幸福出现的太让人措手不及了。<br/>　　<br/>　　    小草脸“腾”的一下就红了，咬着嘴唇说道：“没正形！”<br/>　　<br/>　　    片刻后，小草的卧室里，她换完一身睡衣的时候，王惊蛰已经麻溜的爬到了床上露出一双玉腿，上面盖着被子，冲着她一顿勾魂眼。<br/>　　<br/>　　    茅小草咬了咬牙，心里跳的“嘭，嘭”的，迈着小莲步挪蹭到了床边，心里天人交隔的挣扎着两个念头：“矜持，必须得矜持，不能便宜了这个小傻鸟……反正早晚都有那一天，无所谓吧？”<br/>　　<br/>　　    “唰”王惊蛰忽然抬手，一把将小草拉到了床上，同时胳膊反手一按“啪”的一下把床头灯给关了，动作行云流水，一气呵成，屋内顿时漆黑一片。<br/>　　<br/>　　    小草躺在床上闭着眼睛，呼吸渐浓，王惊蛰拉起被子盖在了她的身上，已经快要深秋了，陇西的天还是很凉的。<br/>　　<br/>　　    窗外秋风徐徐，被窝里暖人心脾。<br/>　　<br/>　　    小草在心底叹了口气：“好吧，好吧”<br/>　　<br/>　　    王惊蛰侧过身子，胳膊环绕过她的身体，然后紧紧的搂着小草，拥入到怀中，小草身子哆嗦了一下，脸碰上了脸。<br/>　　<br/>　　    良久之后，王惊蛰依旧就这么搂着她，动也没动。<br/>　　<br/>　　    又过了许久，他还是这个姿势。略微有点懵逼的小草，轻声问道：“你干嘛呢”<br/>　　<br/>　　    “碎觉”王惊蛰嘴里含混不清的嘟囔了一句。<br/>　　<br/>　　    小草有点小惊讶的问道：“就这么睡了？”<br/>　　<br/>　　    王惊蛰：“……”<br/>　　<br/>　　    几分钟后，王惊蛰忽然在小草耳边，呢喃了一句：“我没想别的，就想这么搂着你，就行了”<br/>　　<br/>　　    小草讶然，忍着羞涩说道：“反正都这样了，你想别的也行”<br/>　　<br/>　　    王惊蛰：“……”<br/>　　<br/>　　    片刻后，王惊蛰咽了口唾沫，说道：“喝多了……不好使了”<br/>　　<br/>　　    小草瞬间无语！<br/>　　<br/>　　    深秋的陇西有点冷，但深秋的渭河景色真是不错，虽然凉意徐徐，但是泛舟于河上的话，欣赏着岸边的两排垂柳和河中的落叶，也别有一番情致。<br/>　　<br/>　　    这两天里，小草和王惊蛰就在河上回忆起了两人的青春，虽然他俩之间的青春还不到一年。<br/>　　<br/>　　    小草撑着竹筏，王惊蛰坐在上面翻看着从凉绣那里坑来的墨家机关术，这本书里面记载的主要是两部分，上半部分记载的是如何制作各类机关，诸如墓葬机关，城防机关系统等等，下半部分正好是讲解的破解之道，机关术数看起来非常复杂，但是如果要破的话，其中也有个死穴，因为墨家在创造机关的时候基本都会留下一手暗门，以防自己以后会撞上了导致自己给自己挖了个坑。<br/>　　<br/>　　    这本墨家机关术，对于王惊蛰作用来讲，是达不到锦上添花的，顶多也就是丰富一下他的阅历，因为他既不需要动用机关术去伤人，也没想过去盗那些个有机关的墓葬，更不至于靠这玩意儿生存了，所以对于他来说，看看这本机关术就是多一些理论上的知识，总体来讲屁用没有。<br/>　　<br/>　　    但世事难料，多一门技术傍身，用不着的时候体现不出什么效果来，真要是用上了的话，救人于水火之中也是没准的。<br/>　　<br/>　　    “呀，咬钩了”正在钓鱼的茅小草突然叫了一声，起身一拉鱼竿就拽上来一条潍河鲤鱼。<br/>　　<br/>　　    片刻后，在岸边，茅小草把鱼下锅，没过多久一股鱼香味就飘了出来。<br/>　　<br/>　　    王惊蛰蹲在地上“吧嗒，吧嗒”的裹着烟嘴，抬头问道：“鱼是好鱼，心是不是好心呢？”<br/>　　<br/>　　    “嘛意思？”<br/>　　<br/>　　    王惊蛰挠了下鼻子，说道：“是不在这个岸边，凉绣那傻姑娘被你给暗坑了一把，鱼汤里下了蛊？”<br/>　　<br/>　　    “你怕了？”小草用铲子轻轻的翻着鲤鱼，淡淡的说道：“女人对我心怀叵测，我肯定以牙还牙，至于你么？王惊蛰，你要是敢背着我搞破鞋的话，我到不至于给你下蛊”<br/>　　<br/>　　    “那也不能白白放过我吧？”王惊蛰诧异的问道。<br/>　　<br/>　　    小草举起手，手起刀落，道：“你想多了，我直接断了你的根就是了”<br/>　　<br/>　　    王惊蛰顿时感觉心里一阵冰凉。<br/>　　<br/>　　    几天后，已是深秋的尾巴了，来到陇西已经有十来天左右了，王惊蛰算是修身养性了一段时间，过着没心没肺的日子。<br/>　　<br/>　　    十二月初的时候，王惊蛰就跟小草告辞了，说自己要进京一趟，有件事答应了人家，最近也该到办的时候了。
