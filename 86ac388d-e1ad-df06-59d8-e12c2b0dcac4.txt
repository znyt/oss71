　　一切都结束了。
<br/>　　芮冷玉内心一片惨然，反而冷静了下来，转头望着四周，发现了身下的雕花大床，还有面前的玻璃茶几和沙发。茶几上还摆着一束假花。
<br/>　　冰雪聪明的她立刻想到了什么，问道：“这些东西从人间来的？”
<br/>　　“是我想办法弄来的，我想给你人间的感觉，虽然我能做的事很少。”
<br/>　　后卿凝视着她，“我对你，冷玉，你是知道的。”
<br/>　　说完一步从玉石盘外面跨了进去，直接穿过了水幕，从长袍里取出了一堆书本，不是古籍，都是崭新的那种在书店里能够买到的书。
<br/>　　芮冷玉看了一眼，都是一些流行的小说，有文学名著，也有通俗小说，《三体》、《追风筝的人》等等，还有一本青子写的《女生宿舍》，看封面就挺恐怖的，应该是恐怖小说。此外还有一些漫画，拿出来之后有一小堆。很难想像后卿是怎么把这么多书塞进袍子下面的。
<br/>　　“这些都是我从人间买的，我不知道你喜欢看什么样的书，所以看流行的各种买了一点，希望能帮你解闷，纸笔也有，你可以写点东西，如果你喜欢看什么书，或者想要什么东西，我可以帮你买来。”
<br/>　　芮冷玉道：“人间的东西，是怎么样带到这里来的？”
<br/>　　“我当然有我的办法。不过……你现在是灵身状态，你不需要喝水吃饭，你只要……在这里呆着，目前来说，只有这次才是最安全的。”
<br/>　　芮冷玉冷哼一声，道：“说这么多，还不是用我做筹码，你何必这么假惺惺的。”
<br/>　　后卿深情地望着她，轻轻摇头，缓缓说道：“我之前说的那些理由，实际是说给女魃和赢勾听的……其实，我只是想救你而已，就像当初你救我一样。我的心，你难道不明白？”
<br/>　　芮冷玉望着那些他从人间带来的家具，还有刚扔下来的那些书，心念转动，说道：“救命之恩，你之前也算是报了，我们两不相欠。”
<br/>　　后卿道：“我对你，不止是为了报恩，我……”
<br/>　　芮冷玉知道他想说什么，摇头道：“我不信，你是尸王，不知道存在了多少年，我跟你才相处了几天，你怎么可能对我有什么情分，你想要利用我就直说，根本不用这些谋划，没用的。”
<br/>　　后卿眼中现出痛苦的神色。
<br/>　　“我是存活了千年，不过一直都在这个苦寒之地，无止境的战斗，杀戮，没有温情……这样的日子，千年也不过如同一日，所以我才会去人间历劫，没想到我轻信于人，被人封印，痛苦不堪……你救好了我，照顾我，跟你在一起的那段时间，是我从来没有过的体验，冷玉，我不相信任何人类，除了你……”
<br/>　　后卿从来没有这样卸下防备，说出这么多掏心窝的话。
<br/>　　“现在，空界，阴司，人间法术界，全世界都在追杀你，只有我能保护你！纵使天地覆灭，我也要保护你的周全！冷玉！相信我！”
<br/>　　芮冷玉真的震惊了，傻傻地看着他，半天说不出话来。
<br/>　　“你……要我怎么做？”许久之后，芮冷玉吐出这句话。
<br/>　　后卿向她面前走了一步，一字一顿地说道：“忘了叶少阳，忘了人间，让我把你变成尸族，随我征战空界，永生不灭！”
<br/>　　芮冷玉心中的震撼，无以复加。
<br/>　　“后卿，你疯了？”
<br/>　　后卿笑了笑，“我没疯。你如果变成尸族，成了我们自己人，女魃会对你放心，不会再防备你。”
<br/>　　“不可能，你也说了，我是转世鬼童，尸气对我没用，我做不了僵尸。”
<br/>　　“我有办法的。”后卿显得胸有成足，“我主将臣，尚存血肉，他的血，是世上至浊之物，能污染一切生灵，就算你是转世鬼童，一旦被将臣的血气侵蚀，也会成为僵尸，只要你愿意。”
<br/>　　芮冷玉倒吸一口气，说道：“我若不愿意，你会用强吗？”
<br/>　　“不会，我永远都不会强迫你做任何事，所以才会等你醒来之后跟你商量……”
<br/>　　“那我很明确地告诉你，我不要变成僵尸！永远都不会。”
<br/>　　后卿看着他，沉默片刻说道：“可是你回不去了，你跟少阳，也不可能了。”
<br/>　　“我知道……”芮冷玉哀伤地说道。
<br/>　　“如果你坚持这样，你只能被关在这里，女魃和赢勾，绝不许你踏出这里。”
<br/>　　等了半天，后卿见芮冷玉没有改变心意的意思，转身朝外面走去，说道：“你好好想想，下次我再来告诉你，我军务在身，要先走了。”
<br/>　　后卿从水幕下直接穿了过去。芮冷玉急忙跟上去，却再次被挡住。情急中大叫了一声：“阿呆。”
<br/>　　后卿心神一颤，回过头来，满怀期待地望着她。
<br/>　　芮冷玉趴在水幕上，一脸决然地望着他，说道：“阿呆，看在过去的情分上，我求你帮我一件事。”
<br/>　　后卿眼神灰暗下去，说道：“如果你求我放你走，我做不到，你一旦离开零界，不管遇到什么人，都必死无疑。”
<br/>　　“不。”芮冷玉咬着嘴唇，用异常坚定的语气说道：“我求你一件事：杀了我！”
<br/>　　什么！
<br/>　　后卿震惊地看着她。
<br/>　　在水幕结界之中，任何气息的运转都是被禁止的，芮冷玉的法力也无法运用，她现在又是灵身状态，想要自毁灵身都做不到。
<br/>　　“我不要当什么转世鬼童，既然我的存在，对人类是一个威胁，那么，我想要毁灭自己。”
<br/>　　在确定自己就是转世鬼童的那一刻，芮冷玉就陷入了深深的绝望，虽然她对于太阴山，对于无极鬼王，没有任何一丁点记忆，她从人间来，十二年来，她都觉得自己是个人类，但是……证据是确凿的，而且自己也伤了叶少阳，差一点杀了他。
<br/>　　她非常害怕自己突然又变成那个自己不认识的样子，去伤害叶少阳。
<br/>　　退一万步说，就算自己能控制不变成那样，但是全天下都知道她是转世鬼童，她不能让叶少阳为了自己，得罪整个天下。
<br/>　　为了叶少阳，她只能放弃自己的生命……
<br/>(本章完)
