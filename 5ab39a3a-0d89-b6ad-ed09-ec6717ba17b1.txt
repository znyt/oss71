        离、离婚？！
<br/>
<br/>        童筱颖一听到这两个字眼，不禁倒吸了口冷气，心口顿时闷得发慌，瞬间有些语无伦次地支吾道：“为、为什么？我其实刚刚有出去要给你送手机的，但是……我……我是不是做错了什么？让你不高兴呢？”
<br/>
<br/>        傅昼景无奈地瞥向童筱颖，看着她娇俏的脸蛋因为紧张而脸色惨白，一双漂亮的大眼睛里含着委屈的泪光，又于心不忍了。
<br/>
<br/>        “不，不是筱颖做错了什么，而是我，已经无法给筱颖幸福了。”傅昼景抬起手来，温柔地摸了摸童筱颖的头顶，就像一个大哥哥一样安抚着自己年幼的妹妹。
<br/>
<br/>        童筱颖有些难以置信地对上傅昼景的目光，抬起双手抹掉眼角的泪花，坚定地回答道：“我不离婚！”
<br/>
<br/>        “筱颖，不要闹了。我现在……欠了银行的巨额贷款。每个月要偿还……”傅昼景欲言又止。
<br/>
<br/>        他和她说这些做什么，他说了她也不懂啊！
<br/>
<br/>        “为什么会欠债呢？每个月要还多少？我可以出去工作的！”童筱颖一脸笃定地说。
<br/>
<br/>        傅昼景伸手将童筱颖揽入怀中，只是紧紧地抱着，什么话也不再说了。
<br/>
<br/>        大学毕业那年，因为自己是私生子又与季思妍分手的缘故，不想继续留在傅家，所以他其实是与傅家闹翻了，才独自从傅家搬了出来的。
<br/>
<br/>        然而，爷爷并未对他完全冷酷无情，虽然关系闹僵了，但是爷爷不仅安排他进了傅氏集团旗下的地利置业工作，还将地利公寓顶楼的这豪华单身公寓套房子送给了他，并过户在他的名下。
<br/>
<br/>        所以，他拿这套房子的房产证作为抵押和朋友一起找了银行里的熟人贷款，白手起家，开了一家装饰装修公司，他只作为股东之一，年底拿分红，公司完全交由他朋友打理。
<br/>
<br/>        他觉得，自己的实力与能力完全不输大哥傅夜沉和堂叔冷岩峰。
<br/>
<br/>        这两年来，他也利用了自己在傅氏集团旗下地利置业的职位便利，给他和朋友的小公司拉了不少业务。
<br/>
<br/>        但是，他朋友的胃口大，不满足于只是当一个小小的装饰装修公司的老板，还瞒着他拿公司作抵押，另外搞投资生意。
<br/>
<br/>        结果，他那朋友做生意不仅亏本，还欠了一屁股债跑路了。
<br/>
<br/>        无奈之下，银行里那两位熟人，不得不亲自上门来找他还债。
<br/>
<br/>        这些事情，傅昼景只能烂在肚子里。
<br/>
<br/>        他好面子，也有极强的自尊心。
<br/>
<br/>        这些都来源于他是“私生子”的身份，让他比任何人都要活得卑微。
<br/>
<br/>        童筱颖见傅昼景不吭声，身子往傅昼景的怀里钻了钻，双手紧紧地抱着他的腰畔，眼泪湿润了眼睛，声音有点儿哽咽地说道：“昼景，你不要不要我了好不好？我不要跟你离婚！我不管你发生了什么事情，我都不会离开你。求你……”
<br/>
<br/>        是啊！
<br/>
<br/>        她在求他……
<br/>
<br/>        除了他，她就再也没有其他家人了。
<br/>
<br/>        诺爸诺母不要她，童爸童妈也不要她，只有他——傅昼景，是她唯一的依靠，也是她内心深处唯一的温暖。
<br/>
<br/>        童筱颖不想离开这一丝温暖，只想好好地珍惜。
<br/>
<br/>        不论贫穷还是富贵、健康还是疾病，一生一世忠于他，尊敬他，陪伴他。
<br/>
<br/>        爱可以让一个人变得很卑微……
<br/>
<br/>        她的心思，他能懂吗？
<br/>
<br/>        她的身体在微微颤抖，就像一只无助的小猫。
<br/>
<br/>        傅昼景将童筱颖搂得更紧了些，微微偏头，忍不住在她的刘海上落下一个轻柔的吻。
<br/>
<br/>        “我只是不忍心……你跟着我吃苦……”
<br/>
<br/>        他的声音低沉、微弱，其实是来自内心深处最真实的想法。
<br/>
<br/>        “筱颖……”
<br/>
<br/>        “嗯？”
<br/>
<br/>        “吃过午饭后，我们得收拾一下行李，然后搬家。”傅昼景无奈地笑了笑，微微低头看着怀里的童筱颖。
<br/>
<br/>        童筱颖怔愣了一下，抬起头来与傅昼景的眼睛对视时，突然意识到了什么，立即破涕为笑，道：“好！”
<br/>
<br/>        吃过午饭后，傅昼景就一直在跟助理赵斌打电话，像是在为了租房子的事情而犯愁。
<br/>
<br/>        此外，童筱颖还无意间偷听到，傅昼景在跟赵斌交底说他朋友跑路了，因为是他的名义贷的款，所以把房子抵押出去后，还欠了银行六百万，贷款期限为十年，等额本息还款，他每个月得月供银行近四万元的样子。
<br/>
<br/>        赵斌觉得，六百万对傅大Boss傅夜沉来说，只是个小数目而已，让傅昼景找自己的大哥解燃眉之急。
<br/>
<br/>        这样，就不用带着筱颖搬家了。
<br/>
<br/>        话虽如此，但是傅昼景不愿意向大哥傅夜沉开这个口。
<br/>
<br/>        亲兄弟明算账，虽然他知道，他开了口，大哥肯定会给，而且也不需要他还，但是，他心里的那道坎过不去。
<br/>
<br/>        颜面比金钱更重要！
<br/>
<br/>        傅昼景不想欠大哥的人情，更何况，他要从大哥手中一次性拿走六百万，爷爷必定会知道这件事情。
<br/>
<br/>        本身爷爷就瞧不起他，如今他还……
<br/>
<br/>        傅昼景心思深沉了起来。
<br/>
<br/>        他心里有苦衷，没有人能理解。
<br/>
<br/>        从小到大，不管是傅家的人，还是外人，都认为大哥是最优秀的。
<br/>
<br/>        他一直想努力证明自己，可为什么他一步步走得如此艰辛？
<br/>
<br/>        最终，他还是带着童筱颖搬家了。
<br/>
<br/>        不过，他们并未搬去其他地方，而只是从地利公寓的顶楼豪华套房搬到了一楼最便宜的单间公寓里罢了。
<br/>
<br/>        他们现在住的这个出租屋里，可以说，一进门就是卧室。
<br/>
<br/>        一室一厨一卫一阳台，没有客厅，没有餐厅，更别说分主次卧了。
<br/>
<br/>        地利公寓真的可以说是看楼层来区分户主的身份，越有钱，住的楼层就越高。
<br/>
<br/>        童筱颖此刻终于明白，当初哥哥诺司峻和嫂嫂唐倩兮为什么想住楼上去了。
<br/>
<br/>        地利公寓楼上的套房，真的比一楼的套房隔间配置要好太多。
<br/>
<br/>        “以后，你睡床上，我睡沙发。”傅昼景微笑着说。
<br/>
<br/>        童筱颖一点都不介意自己住这样的房子，站在傅昼景面前，她反倒有点儿娇羞地说道：“我们俩都已经是夫妻了……所以……”
<br/>
<br/>        睡在一起没什么不妥的……
