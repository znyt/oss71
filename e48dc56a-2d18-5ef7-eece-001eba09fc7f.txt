　　不周之巅，三清眼看七大先天葫芦就快要成熟了，然而，就在此时，却迎来了六位不速之客。
<br/>　　“咦，我道是谁呢？原来是两只扁毛雀啊，如今来此，是为了送死吗？”
<br/>　　就在这时，通天见来人之中有帝俊与太一，想起之前在紫霄宫宫前恩怨，他顿时就不爽了，大声骂道。
<br/>　　“通天，你找死！”
<br/>　　太一听了通天的话，顿时大怒，多次被人称作扁毛雀，他心中怒火早已难抑，当下凭空托出一口古老大钟，想要上前与通天大战一场。
<br/>　　“且慢，二弟。”这时，帝俊上前拉住太一，让他且慢动手。
<br/>　　“大哥，我们怕他们干什么。”太一大怒，责问帝俊。
<br/>　　“先看看是什么宝物，这里有如此多人，我们还会怕他们吗？”
<br/>　　“好，就先听大哥所言。哼！”太一这才作罢。
<br/>　　元始脸上露出不屑的神情，帝俊打什么注意，他一眼便知。他元始三人只有三人，而对方却又六人，帝俊不就是想要在待会儿夺宝之中，引发六人围攻他们三人吗。哼，六人又如何，你以为我元始会怕吗。
<br/>　　“六位到此所为何事？若是无事，本座还是劝各位早早离去。”
<br/>　　元始淡淡地看了六人一样，便不再看他们，而是将目光放在眼前的七个葫芦身上，就好像是六人还没有这葫芦好看，不屑之意，毫不掩饰地流露出来。
<br/>　　“呃——”镇元子与红云见如此气氛，有点尴尬，不知从何说起。
<br/>　　“哼！怎会有如此霸道之人。”倒是女娲见此，娇哼一声。
<br/>　　“道友，贫道这妹妹被贫道惯坏了，三位请不要在意。”伏羲有点尴尬，对着元始三人解释道。
<br/>　　不过，他的话说完后，谁也没有答话。见此，伏羲更加尴尬了。
<br/>　　“咳！”伏羲咳嗽一声，以此来掩饰自己的尴尬，接着说道：
<br/>　　“道友，我等感到自身机缘所至，故此来到此处，此处有七个葫芦，想来是我等与这葫芦有缘。”
<br/>　　“你这话就说错了，有缘？本座还说吾与那口大钟有缘呢？怎么不见他将那口大钟送到本座面前？真是可笑！”
<br/>　　元始冷笑一声，不屑道。有缘？骗鬼去吧！一句有缘就想要宝贝，天下哪有那么好的事？天下宝物，还不是靠自身本事，本事过硬，没缘也可以变成有缘，若是没有本事，有缘又如何？哼！
<br/>　　“你——”太一大怒。
<br/>　　“道友说笑了。”伏羲尴尬一笑。
<br/>　　“说笑了？我看未必！也罢，本座也不屑与尔等废话。本座今天就把话说明白了，这先天葫芦藤归本座所有，尔等一个葫芦都别想要，若是尔等有意见，本座奉陪到底。”
<br/>　　元始也懒得与他们废话，直接将话挑明。
<br/>　　轰！
<br/>　　顿时，其他人不乐意了。什么？都是你的，这怎么可能？我们来此不就是为了一个葫芦吗，都是你的，那我们怎么办？
<br/>　　至于帝俊，心中更是冷笑，大骂元始傻子，竟然傻到将在场六人都全部得罪了。他之前还想着用什么办法来让元始引起众怒，这样他们就有机会群起而攻之了。谁曾想到，他还没有行动呢，这元始自己就把众人都得罪了。
<br/>　　“什么？道友，你这话就过了，此处有七个葫芦，你们只有三人，剩下四个，我们六人分，已经是最大的让步了，你还要得寸进尺？”镇元子道。
<br/>　　轰隆隆！
<br/>　　就在元始要答话之时，伴随着一阵轰响，七个葫芦吸收完灵气与神水，已经成熟了。灵气散开，露出里面的葫芦。
<br/>　　众人的眼光立刻就被这七个葫芦吸引过来，只见七个葫芦颜色各异，威能也各不相同。
<br/>　　第一个为紫金葫芦，第二个为紫白色葫芦，第三个为紫红色葫芦，第四个为紫青色葫芦，第五个为紫黑色葫芦，第六个为紫黄色葫芦，第七个为紫绿色葫芦。
<br/>　　七种光芒绽放，耀眼无比。光芒闪烁之间，蕴含着莫大的威能，神光璀璨，动人心弦。
<br/>　　“道友，如今七个葫芦，我们只取四个，你们三个，而且我等让你们先选，如何？”
<br/>　　就这众人看着葫芦眼冒绿光之时，红云看着葫芦对元始道。
<br/>　　“不如何！”元始淡淡的声音传来，“七个葫芦本座全要了。”
<br/>　　说完，不等众人再说话，他对着七个葫芦就是一道混沌色的光幕刷去。只见光幕一闪即没，随之消失的，还有七个葫芦。
<br/>　　显然，七个葫芦已经被元始刷走了，他并不想要留下给别人。
<br/>　　不好！众人见元始那动作，便知道元始不打算分配葫芦，而是想要独吞，可是众人刚想要阻止，却已经来不及了，因为葫芦已经被元始走走了。
<br/>　　“元始，将葫芦交了出来！”帝俊望着元始，冷笑一声，大声喝道。
<br/>　　但元始看都懒得看他一眼，而是对着众人淡然说道：
<br/>　　“葫芦本座已经收了，交出来却是不可能的，尔等要如何？”
<br/>　　“哎！罢了罢了，看来贫道缘分还不够啊，贫道先走一步。”
<br/>　　红云道人看了看元始三人，之后又看了看帝俊等人，既然葫芦已经没了，他也没有必要在此恶了别人，就此告辞是他的选择。
<br/>　　“贫道也先走一步。”
<br/>　　镇元子苦笑一声，也告辞离开。他之前并没有感受到什么机缘，而是只有女娲、红云与太一感受到，他只是陪着他好朋友红云来的。如今正主都已经走了，他还留下来干什么。
<br/>　　“道友，何必着急先走，我等有六人，又不是没有机会夺取先天葫芦？”帝俊拦截住镇元子，劝阻道。
<br/>　　“此宝与贫道无缘，贫道还是不趟这趟浑水了。就此告辞！”
<br/>　　镇元子淡淡地看了一眼帝俊，帝俊打什么注意，他早已看出来，他却是不想被帝俊利用。
<br/>　　“喏，已经走了两个，你们呢？”
<br/>　　元始轻笑一声，眼中不屑之意更浓了，对着眼前四人淡声问道。说来也是巧了，四人都是妖族之人，看来他们是想要联合了。
<br/>　　四人不语，各自取出法宝，隐隐想要将元始三人围住，不让他们逃走。他们的动作就已经表明了他们的想法。
<br/>　　“元始，你太霸道了，紫黄色葫芦与贫道有缘。”女娲娇声喝道。
<br/>　　这四人中，女娲是唯一的女子，她之前的确感受到那葫芦与自己有缘，想要取走，却被元始抢先了一步，俗话说，女人是最记仇的生物，如今看来，果然如此。至于伏羲，他则是为了他妹妹女娲。
<br/>　　更何况，之前在来的路上，帝俊就提出想要将两方妖族势力合聚的想法，他也有些赞同。现在帝俊与太一要对付三清，那么他觉得自己也应该帮忙。
<br/>　　至于帝俊与太一，两人的想法就更简单，之前与三清结仇，他们之间已有恩怨，只是想单纯的破坏三清机缘，顺便得到几个葫芦，就这么简单。
<br/>　　元始淡漠地扫了四人一眼，眼光如水，仿佛能将四人的想法看透一般，之后，轻声道：
<br/>　　“唔，看来，还是有人不识趣啊！”
<br/>(本章完)
