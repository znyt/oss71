        &lt;h3&gt;第三卷第一百五十六章  张函母子&lt;/h3&gt;
<br/>
<br/>        弥青云心情不佳的回到了家里。
<br/>
<br/>        张沛尧最近太忙，回来的很晚，张函也是基本不在家里，她想到陈子昂都有孩子了，张函和陆玉还两地分居，这两年都没有怎么见面，再这样下去，他们家怕真是没有孩子了。
<br/>
<br/>        家里的保姆回家了，就弥青云一个人，她连灯也没有开，就坐在客厅的沙发上，想着怎么和张倩楠开口说这件事。
<br/>
<br/>        张函从陈子昂的家里出来，他是开心的快乐的也是心酸的难过的，他的心里是实实在在的打翻了五味瓶。
<br/>
<br/>        陈子昂能让他在家里吃饭，是不是证明陈子昂已经原谅他了？陈子昂前几次对他的态度不是很好，对他冷冰冰的，今天竟然同意他在她的家里吃饭了。
<br/>
<br/>        如果没有陆玉，那么他们一家人应该是团团圆圆在一起的，虽然他还能去看孩子，但也是偷偷摸摸的，不能让别人知道。
<br/>
<br/>        什么时候他才能光明正大的去看他的孩子啊。
<br/>
<br/>        张函怀着纠结的心情打开了家里的灯，看见他的妈妈坐在沙发上，微微的吃了一惊。
<br/>
<br/>        “小函，你回来了？”弥青云也有很久没有看见张函了，张函最近老是出差。
<br/>
<br/>        “妈，你怎么不开灯？”张函感觉有些奇怪。
<br/>
<br/>        “你过来，我有事情要和你说。”弥青云还是决定给张函说一说陈子昂的事情，毕竟关系到张倩楠，张倩楠虽然不是她生的，她也当成自己的孩子看，要不然她也不会将冯睿送给她的那个手镯送给张倩楠。
<br/>
<br/>        张函刚好也有事情要和弥青云说，便走过去坐在沙发上。
<br/>
<br/>        “你喝酒了？”弥青云皱着眉头问。
<br/>
<br/>        “喝了一点。”张函喝的也不是很多，他的心情到现在都是激动的，脸红红的。
<br/>
<br/>        “你还是少喝酒。”弥青云担心自己儿子的身体，这几年张函看起来和正常人一样，但是弥青云还是很担心。
<br/>
<br/>        “你有什么事情吗？”张函问弥青云。
<br/>
<br/>        “陈子昂你还记得吗？”弥青云直接问张函。
<br/>
<br/>        “你就说是什么事情。”张函皱着眉头，帅气的脸上瞬间蒙上了一层寒冰。
<br/>
<br/>        “陈子昂和江俞轩生了一对双胞胎。”弥青云说。
<br/>
<br/>        “你听谁说的？”张函心里一惊，江俞轩告诉过他，冯睿和夏颜都知道陈子昂有孩子了，果然，冯睿会告诉弥青云和蔡少芬。
<br/>
<br/>        “冯睿说的，你得想办法告诉倩楠，这个陈子昂真的不是个东西，怎么就和我们张家断不了瓜葛呢？”弥青云气愤极了。
<br/>
<br/>        “这么多年了，你心里就一点愧疚都没有吗？”张函冷冷的问弥青云。
<br/>
<br/>        “我有什么愧疚？”弥青云奇怪的问张函。
<br/>
<br/>        “你们为了攀上高枝拆散了我和陈子昂，你们就没有愧疚过？”张函的语气使弥青云打了一个寒颤。
<br/>
<br/>        “你不是说你都已经忘记了吗？”弥青云瞪大了眼睛。
<br/>
<br/>        “你都没有忘记，经常在冯睿哪里探听消息，我又怎么能忘记呢？”张函语气依然冰冷：“我不妨告诉你，这么多年我就是为陈子昂母子而活的。谁要是敢打他们的主意就算是你，我也不会放过。”
<br/>
<br/>        “小函，为了那样的一个女人，你值得吗？”弥青云也生气了。
<br/>
<br/>        “什么样的女人？她为了我生了俩个可爱的孩子，她一个人辛苦的带着俩个孩子，要不是落叔叔的帮助，我的孩子和我的爱人因为你们会沦落街头。”张函不知道冯睿是怎么告诉弥青云孩子是江俞轩的，他不能让江俞轩背锅。
<br/>
<br/>        江俞轩和张倩楠谈了很多年，虽然知道他们之间的关系，也知道孩子的事情，但是如果这件事情发酵了，或者是散发出去了，对陈子昂和江俞轩都不利，因为他们现在都已经是公众人物，有可能会影响江家的股市。
<br/>
<br/>        上一次江俞轩给他提醒了，他一直在寻找对策，这件事情他必须得和陈子昂好好商量，孩子现在的身份并没有完全曝光，只是在他们几个人之间，可能陈子昂只忙着工作，没有想到别的事情上去，有些事情他必须得考虑到，而且他现在的身份也不能有任何差池。
<br/>
<br/>        他的父亲和母亲在他婚姻之上做了糊涂的事情，但是在工作上还是兢兢业业，在大是大非面前还是能把控住的，这几年他也在暗暗的观察和调查着他们。
<br/>
<br/>        他决定还是将这件事情坦白告诉他们，还有重要的一点，他们都注重张函的事业。
<br/>
<br/>        “你说什么？陈子昂的孩子是你的？天哪！你别骗我了。”弥青云蒙了。
<br/>
<br/>        弥青云不是不喜欢陈子昂，要是陈子昂真的一无是处，她绝对不会去关注她，正因为陈子昂太优秀了，她才想从冯睿那里听到陈子昂的消息，她也可以自己去打听，但是她又害怕自己去打听，她以为她的儿子已经忘记了陈子昂。
<br/>
<br/>        “妈，我实话告诉你吧，这几年我根本就没有和陆玉在一起，我对陆玉的好只是表面的，我希望我的爱人和孩子能平平安安的，既然你们都知道了，我也就不怕告诉你们。”
<br/>
<br/>        弥青云不是傻子，蔡少芬让她找张函调查也是别有用意的，就是想知道张函对陈子昂的态度。
<br/>
<br/>        如果陈子昂真的生了张家的孩子，那么陆玉会怎么对待陈子昂？陆家会怎么对待陈子昂母子三人？
<br/>
<br/>        按照一般普通女人的心思，怕也不会容忍这样的事情发生，何况陆玉还不普通。
<br/>
<br/>        陈子昂一个外来的女人，虽然这几年生意做得红红火火，可是她一点背景都没有，陆家想动她，就和捏死一只蚂蚁一样，随便找个借口都会让他们的公司遭受重创。
<br/>
<br/>        还有，还有她的俩个孙子，她又想起了那一次见到的那俩个可爱的孩子，心里竟然非常渴望见到陈子昂的俩个孩子。
<br/>
<br/>        “小函，这件事情是大事情，你不应该瞒着我们，这个必须得和你父亲好好商量一下。”弥青云已经不再去责怪张函了。
<br/>
<br/>        她现在的心思全都是她的孙子，虽然弥青云还没有到七老八十，但是女人的天性决定了她们的母性。张函结婚这么多年，连个孩子都没有，突然说有了俩个孩子，是让她又惊又喜。
<br/>
<br/>        怎么样认回孙子是一个问题，怎么样让陆家接受这件事情是要解决的当务之急。
<br/>
<br/>        当然，陈子昂是孩子的母亲，也是必须要保护好的。
<br/>
<br/>        “和你商量了就能解决问题吗？”张函问他的母亲。
<br/>
<br/>        “不能解决问题也得解决，现在牵扯到倩楠和江家了，总有一天这个事情会被拔出来。不如我们早点将事情解决了。”作为张家长房媳妇的弥青云也不是一无是处。
<br/>
<br/>        “我来给你父亲打电话吧，让他晚上早点回来。”弥青云拿起电话。
<br/>
<br/>        张函并没有阻止他的母亲给他的父亲打电话。
<br/>
<br/>        现在，他得寻求父亲的支持，虽然他们曾经格格不入，但是为了孩子他必须做出让步。
<br/>
<br/>        子昂，你要是还愿意和我在一起，我就永远爱你，你要是不愿意和我在一起，我就永远相思，就远远的看着你陪你一生一世，护你一生周全，哪怕失去我的生命也在所不惜。
