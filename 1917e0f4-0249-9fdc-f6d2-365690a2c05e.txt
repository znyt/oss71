        为了让这暧昧的声音，能传入门外傅昼景的耳朵里，夏凌焰特意将自己的平板电脑放在了门边的矮柜上。
<br/>
<br/>        童筱颖忍不住抬起手来抚着额头，全程不忍直视。
<br/>
<br/>        “现在，让你的前夫在门外揪心一会儿，我们接着赶设计稿，好不好？再不出稿，我的客户要杀了我了！”夏凌焰一脸哀求地看着童筱颖。
<br/>
<br/>        童筱颖放下手来，点了点头。
<br/>
<br/>        不过，她倒是佩服夏凌焰的定力。
<br/>
<br/>        那边视频里娇喘不断，他夏凌焰作为一个男人，还跟她这么一个女人，孤男寡女共处一室，他居然可以非常淡定地坐在电脑前设计图纸。
<br/>
<br/>        童筱颖也决定“两耳不闻窗外事”，一心只做别墅模型。
<br/>
<br/>        时间一分一秒地过去了……
<br/>
<br/>        童筱颖慢慢地重新进入了工作状态，但仍旧会时不时地想起，傅昼景是否还在门外守着？
<br/>
<br/>        突然……
<br/>
<br/>        “哐、啷、乓”地一声，卧室的门板从门框上脱落，重重地砸在了地面上。
<br/>
<br/>        童筱颖目瞪口呆。
<br/>
<br/>        视野里，只见傅夜沉一个华丽的收腿动作。
<br/>
<br/>        还说自己没暴力倾向！！！
<br/>
<br/>        “大哥！”
<br/>
<br/>        童筱颖还以为是傅昼景，完全没料到出现在自己面前的男人居然是傅夜沉。
<br/>
<br/>        “什么大哥？你哥哥啊？”夏凌焰从笔记本电脑后抬起头来，一副“丈二的和尚，摸不着脑袋”的表情，接着碎碎念叨，“小筱颖，你这个哥哥有暴力倾向！”
<br/>
<br/>        就连夏凌焰也这么认为……
<br/>
<br/>        而门边的平板电脑里，仍旧在播放着限制级的动作片。
<br/>
<br/>        童筱颖见傅夜沉双拳紧握，冷冽的黑眸里布满了血丝，俊脸上青筋暴起，她心里冷不丁地打了个寒战。
<br/>
<br/>        这会儿，说时迟那时快，眼见着傅夜沉踱步上前，揪住夏凌焰的衣领，将他从椅子上提起，并向他挥起了拳头。
<br/>
<br/>        童筱颖连忙扑入傅夜沉的怀中，紧紧地搂住傅夜沉的腰际，倾尽全力地阻止道：“大哥，不要啊！”
<br/>
<br/>        “对对对！大哥！您别乱来！咱们有话好好说！”夏凌焰一看傅夜沉这架势，瞬间再次成了小怂包，并畏畏缩缩地将傅夜沉揪住他衣领的手，一根手指头一根手指地掰开。
<br/>
<br/>        “你对筱颖做了什么？”傅夜沉冷冷地质问。
<br/>
<br/>        “大哥，你听我说，我和筱颖真心相爱，求你成全啊！”夏凌焰一口气说到底。
<br/>
<br/>        却把童筱颖说得哑口无言。
<br/>
<br/>        剧本不是这么写得吧！
<br/>
<br/>        不带这样临时加戏的啊！
<br/>
<br/>        “你有多爱筱颖？”傅夜沉再次将夏凌焰揪到自己跟前，犀利的眼神里透着一股杀气。
<br/>
<br/>        夏凌焰暗自庆幸这眼神不能杀人，不然他早就死了！
<br/>
<br/>        “大哥啊！我对筱颖的爱，日月可鉴！就是那种‘山无陵，江水为竭，冬雷震震，夏雨雪，天地合，乃敢与君绝！’对吧？筱颖。”夏凌焰决定豁出去了，将这戏演到底。
<br/>
<br/>        “很好！”傅夜沉黑眸微眯，睐着童筱颖，质问道，“筱颖，那你呢？”
<br/>
<br/>        童筱颖看了看傅夜沉后，又对上夏凌焰的目光，扯了扯嘴角，尴尬不已地回答道：“那个……夏工……他……就是我的正牌老公……”
<br/>
<br/>        “什么正牌老公？”夏凌焰先是一脸懵，随即立马反应了过来，“我去！他就是你跟我提起的那个‘二婚老公’啊！”
<br/>
<br/>        童筱颖抿了抿嘴，不好意思地点点头。
<br/>
<br/>        下一秒，夏凌焰立即改口了，连忙解释道：“这位大哥！误会！误会！筱颖让我给她当冒牌老公是为了去气走前夫，所以才……”
<br/>
<br/>        “啪——”
<br/>
<br/>        不等夏凌焰把话说完，傅夜沉直接揪着他的衣领顺手将他甩坐回了椅子上。
<br/>
<br/>        童筱颖回眸看向傅夜沉。
<br/>
<br/>        只见他紧抿着薄薄的唇，带着冷酷的气场，踱着步伐，脚下生风，一步一步地向她靠近。
<br/>
<br/>        “大哥……”她欲言又止，微微探身，看了一眼房门外。
<br/>
<br/>        小卫已经走了吗？
<br/>
<br/>        她不大确定。
<br/>
<br/>        傅夜沉此刻很没心情去在乎其他的事情，冷毅的薄唇轻启，话语里不带任何感情地命令道：“跟我走！”
<br/>
<br/>        “他是我的雇主，我得帮他把设计稿……”
<br/>
<br/>        “我说的话，你听不懂？嗯？”他有些冷酷无情地打断了她的话。
<br/>
<br/>        夏凌焰尴尬地坐在一旁，咳了两声，吸引了童筱颖的注意力后，苦着脸，向童筱颖推了推手，示意她赶紧走。
<br/>
<br/>        童筱颖又回眸看了傅夜沉一眼，心里估摸着或许大概也许他不大相信她和夏凌焰之间是清白的。
<br/>
<br/>        毕竟……
<br/>
<br/>        孤男寡女，共处一室……
<br/>
<br/>        而且，有种解释就是掩饰，掩饰就是确有其事的错觉。
