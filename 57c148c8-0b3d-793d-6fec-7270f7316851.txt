　　轰隆隆！！
<br/>　　劲力爆开，恐怖的冲击波如同核弹爆炸，汹涌澎湃，席卷整个平台，冲击九天十地。
<br/>　　嗖！嗖！
<br/>　　无尽气浪之中，众人只看到两道身影急速飞退，各自后退数百里，均是退到了平台的边缘，才停了下来。
<br/>　　“这是……！”
<br/>　　真圣界内外，所有观战之人看到这一幕，无不震惊，个个瞪大了眼眸。
<br/>　　这是什么情况？
<br/>　　居然势均力敌！
<br/>　　巨摩和苏莫各自后退数百里，这一击两人平分秋色，不分胜负。
<br/>　　“天啊！苏莫挡住巨摩这一拳！”
<br/>　　“何止是挡住了，平分秋色！”
<br/>　　“这怎么可能？苏莫的力量居然能比得上巨摩？”
<br/>　　“可怕！”
<br/>　　“难怪苏莫被排在真圣榜第一，这才武帝境六重啊！这要是武帝境九重还得了！”
<br/>　　众人一片哗然，议论纷纷，均是被苏莫的实力震惊了。
<br/>　　硬撼巨摩而不败，仅凭这一点，便可俯瞰古灵星河亿万武帝境武者。
<br/>　　那些巨灵族的人，更是震惊的无以复加，巨摩的力量他们最清楚不过了，在武帝境之中不敢说打遍古灵星河无敌手，但单纯的力量而言绝对是没有敌手。
<br/>　　现在，一名武帝境六重的人族武者，居然硬撼巨摩而不败，这种战力死何等的可怕。
<br/>　　但最震惊的人，莫过于巨摩本人了，他抬起有些发麻的拳头看了看，只见拳头上有一道深深的红印，拳头之上的皮肤，差点被破开。
<br/>　　“这怎么可能？”
<br/>　　见此，巨摩忍不住眉头一皱，他的一双拳头，是经过特殊淬炼的，堪比神兵利器，居然差点被破开，这是什么概念？
<br/>　　而且，苏莫刚才一剑，力量之大，让他都差点没挡住，这让他脑中有些反应不过来。
<br/>　　他自从修炼武道以来，所遇到的天才人物不计其数，还从来没有人能硬撼他的力量。
<br/>　　但现在，他差点没挡住一名武帝境六重武者的一剑，这让他感觉极不真实。
<br/>　　“巨摩，你的力量不过如此！”苏莫的声音响起，响彻四方。
<br/>　　此时，苏莫眼眸闪亮，隔空凝视着巨摩。
<br/>　　不过，他言语虽然如此，但心中还是很佩服对方的，这一击之下，他现在半个身子都麻木了，被强大的力量震得失去了直觉。
<br/>　　刚才一剑，几乎是他的巅峰战力了，却没能击败对方。
<br/>　　实际上，苏莫知道，在力量的比拼上他还是输了，因为对方还有战魂没有动用。
<br/>　　在这个世界，除了妖族与一些特殊的种族之外，都有武魂或战魂，巨摩绝对也有。
<br/>　　所以，严格意义上讲，力量比拼上苏莫还有有所不如，他动用十五种战魂，和四种特殊体质，才能比的上对方不用战魂的状态。
<br/>　　“哼！狂妄，你以为你能胜我？”巨摩闻言冷哼一声，脸色很不好看。
<br/>　　他如此强大的一拳，本以为能轻松将苏莫轰出平台，没想到居然和苏莫平分秋色。
<br/>　　咚！
<br/>　　身形一纵，巨摩向前飞出百余里，落在平台上让得平台发出一声闷响。
<br/>　　“给我败吧！”
<br/>　　巨摩再次暴喝，他的背后虚影升腾，两只庞大的战魂升腾而起。
<br/>　　这是两只兽类的战魂，一只是庞大的黑熊，身高近十丈，庞大无比，另一只是类似犀牛的异兽，同样庞大。
<br/>　　吼！吼！
<br/>　　两只战魂乍一出现，便仰天咆哮，震荡整个真圣界，而巨摩身上的气势则随之暴涨，凶煞、狂野，如同荒古巨人。
<br/>　　轰！轰！轰！
<br/>　　巨摩再次出手了，双拳齐出，不断的轰击。
<br/>　　霎时之间，一个个浩大的拳印如同一座座小山一般，碾碎虚空，铺天盖地的向苏莫轰杀而来。
<br/>　　磅礴的威势浩荡天地，恐怖的拳威震动九霄，漫天拳印遮盖一方天地。
<br/>　　巨摩已经没有了再与苏莫比拼力量的心思，彻底动用了全力，准备击败苏莫。
<br/>　　“你想多了！”苏莫淡漠一笑，挥剑迎击，剑光挥洒长空，道道剑气横贯天地。
<br/>　　这一次，他动用了吞噬战魂，强大的吞噬之力，伴随着他不算很强的吞噬奥义，席卷而出，瞬间笼罩方圆千里。
<br/>　　吞噬之力虽然不足以击败对方，但可以很大程度的影响对方，大幅削弱对方的攻击力。
<br/>　　轰！轰！轰隆隆！
<br/>　　剑气与拳印不断的相击，惊天爆响此起彼伏，恐怖的爆炸让得整个战台上变成了一片混沌的世界。
<br/>　　“这是什么力量？”
<br/>　　巨摩双眉一皱，身处吞噬之力的笼罩之中，他感觉受到了强大的拉扯之力。
<br/>　　虽然他能抵挡住这种拉扯之力，但体内玄力受到很大的影响，运转起来颇为费劲，运转的速度减缓了不少。
<br/>　　嗖！
<br/>　　苏莫的身形冲天而起，洞穿汹涌的气浪，瞬间飞临了万丈高空。
<br/>　　“巨摩，让你尝尝我剑阵的威力！”
<br/>　　苏莫朗喝一声，意念一动，他的周盛无数剑光突然闪现，出现了多达一百柄灵剑。
<br/>　　一百柄灵剑悬浮在他的周身，剑指苍穹，剑威无可匹敌。
<br/>　　“什么？本命灵剑阵？”巨摩见此面色微微一变，他没想到苏莫居然有如此多的灵剑。
<br/>　　本命灵剑阵是非常可怕的攻击手段，尤其是那种超级剑阵，堪称杀伐利器。
<br/>　　当然，因为操控本命灵剑阵，对于武者的精神力要求非常的高，所以真正强大的本命剑客极少。
<br/>　　一般的本命剑客，只能操控那些小的灵剑阵，威力并没有多强大。
<br/>　　巨摩曾经击败过一名本命剑客，但当时对方只是操控三十余柄灵剑的小剑阵，而现在苏莫居然有多达百柄灵剑，让他的脸色瞬间凝重了下来。
<br/>　　“苏莫居然还是本命剑客？”
<br/>　　“我天！百柄灵剑，他能操控得了吗？”
<br/>　　“这手段真是让人始料不及！”
<br/>　　“不过就算这样，想打败巨摩依旧很难！”
<br/>　　真圣界外，众人议论纷纷，之前苏莫与巫殇对战之时，虽然动用了灵剑，但只有三柄灵剑，众人也没有多想。
<br/>　　但现在，居然一次性动用了百柄灵剑，这就厉害了！
<br/>　　百柄灵剑的本命剑阵，若是熟练操控的话，威力在武帝境之中，堪称恐怖。
<br/>　　“有趣！”
<br/>　　逍遥剑圣见此，面上露出了一丝惊奇之色，他就是本命剑客，一生都在研习剑阵。
<br/>　　现在见到苏莫要动用剑阵，他不禁大感意外。
<br/>　　在古灵星河，能有施展本命剑阵天赋的人并不多，一般的本命剑客天赋并不强，在武帝境修为之时，能熟练操控五、六十柄本命灵剑，已经非常难得了。
<br/>　　现在苏莫要操控百柄灵剑，他也是大感兴趣，想看看苏莫能施展到什么程度！
