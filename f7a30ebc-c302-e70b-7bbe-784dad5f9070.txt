　　“共有四脉，分别是天巫星主脉和苍云星、化灵星、天冥星这三大支脉。”略胖的巫族青年说道。
<br/>　　“那你们天冥星支脉，有多强的实力？”苏莫问道。
<br/>　　“我们天冥星支脉，虽然实力远不如主脉，但也有五位武圣，几万名武尊境的尊者！”青年说道。
<br/>　　“这么强！”苏莫闻言眉头微皱，一个支脉便有五位武圣，这巫族比他想象的还要强大啊！
<br/>　　“巫天御在天冥星支脉是什么身份，平常都住在哪里？”苏莫再次问道。
<br/>　　他已经迫不及待要去寻找巫天御了，实在是他不敢拖下去了。
<br/>　　时间越久，宏青璇便越危险。
<br/>　　唯一让他比较安心的是，宏青璇修为较低，就算巫天御想拿她做炉鼎，也没有什么效果，需要很长的时间去培养。
<br/>　　但话说来，巫天御若是将宏青璇送人呢！送给境界差不多，适合双修的人。
<br/>　　亦或者，巫天御色心极重，贪恋宏青璇的美色，根本不管双修的效果，只想满足自身的欲.望，那就完了！
<br/>　　“天御大哥身份倒是不高，父母都只是巫族普通的族人，他平常住在黑耀城！”略胖青年说道。
<br/>　　“黑耀城？”苏莫闻言面露沉吟之色，看来此城是巫族的城池，说不得他要去黑耀城走一趟了。
<br/>　　不过，此事也不能太冒失，必须有详细的计划，不然只是自寻死路。
<br/>　　“苏莫，难道你想去杀天御哥？”巫娴眸中精光一闪，试探性的问道。
<br/>　　另外两名巫族青年，均是面带惊疑之色，苏莫如此询问，明显是要去找巫天御。
<br/>　　不过，苏莫却是根本没有回答巫娴的话。
<br/>　　“你们在此好好待着吧！”
<br/>　　苏莫冷冷的说了一句，身形一闪，瞬间便消失在了玉镯空间中，只剩下满脸落寞的巫娴三人。
<br/>　　巫娴三人神情怔怔，心中早已充满了绝望，他们不知道还有没有逃生的希望，但他们无力反抗，只能一直等待下去。
<br/>　　……
<br/>　　苏莫离开玉镯空间之后，便去聂乾和韩盈的洞府，将两人都喊了过来。
<br/>　　三人居住在一个山谷中，洞府相隔不足千丈。
<br/>　　“聂兄，韩姑娘，我想问你们一下，天冥星在什么地方，距古周星有多远？”苏莫向两人问道，他已经做好了去天冥星的打算。
<br/>　　本来他是准备提升到武帝境巅峰，或者是武尊之境再去的。
<br/>　　但现在他已经等不及了，武帝境修炼的如此之慢，待他提升到武帝境九重，或者是武尊之境，都不知道过去几年了。
<br/>　　那个时候，实在太晚了！
<br/>　　“天冥星？”聂乾闻言一怔，随即沉吟了一番，道：“这好像是巫族的地盘吧？”
<br/>　　“不错，就是巫族所在的天冥星！”苏莫点了点头。
<br/>　　“这个我倒是不清楚！”聂乾摇了摇头，他只是偶尔听闻过，并不了解巫族。
<br/>　　“韩姑娘，你是否了解？”苏莫又问向韩盈。
<br/>　　“我了解一些！”
<br/>　　韩盈点头，继续道：“天冥星距古周星不算远，若是一直使用星空传送大阵的话，中间差不多要中转近百次，最多四个月，就能达达天冥星。”
<br/>　　苏莫闻言顿时无语，中转近百次，四个月的时间，这还不算远？
<br/>　　“那天冥星有没有人族？”苏莫又问道，这是个很关键的问题。
<br/>　　若是天冥星全是巫族，没有人族存在的话，他去了之后未免太扎眼了。
<br/>　　“当然有了！”
<br/>　　韩盈笑了起来，她感觉苏莫就像个初出茅庐的小孩子，很多常识性的东西都不懂。
<br/>　　“在古灵星河，任何一个生命星辰之上，都有大量的人族存在，只不过，在异族掌控的星辰上，人族势力均是完全臣服与异族的！”韩盈笑着说道。
<br/>　　“原来如此！”苏莫心中恍然，这就好办了，有大量的人族存在，他就能浑水摸鱼了。
<br/>　　苏莫沉默了少倾，心中已经做出了决定，就去天冥星了。
<br/>　　虽然巫族强大无比，但他隐藏在暗中，一切倒也不是太困难。
<br/>　　“巫天御，你的死期不远了！”苏莫攥了攥双拳，心中杀机森然。
<br/>　　“苏兄，你如此打听天冥星的情况，难道是想去天冥星？”聂乾好奇的问道。
<br/>　　“不错，我要去一趟天冥星，估计要很长时间才能回来！”苏莫点了点头，他倒是没有隐瞒两人。
<br/>　　聂乾闻言，略一沉吟，道：“要不，我们三人一起去吧？我和韩盈早就想出去历练了！”
<br/>　　“对呀！苏兄，我们三人同行吧！”韩盈亦是点了点头，她的修为已经到了一个瓶颈，就差一个突破的契机了。
<br/>　　而且，聂乾和韩盈两人，之所以要离开，最主要的原因，是担心苏莫离开之后，太史轩找他们的麻烦。
<br/>　　他们目前虽然住在火元峰，但不可能一直闭门不出吧！
<br/>　　况且，他们完全不认识火元圣师，到时候有事的话，他们也很难得到庇护。
<br/>　　“你们也要去？”苏莫闻言一怔，眉头微微皱了起来，他可不是去历练，而是要干杀人放火的勾当。
<br/>　　这种事情，带着两人怕是有些不便。
<br/>　　“那个……苏兄，你若是不便，那就算了！”聂乾见苏莫的面色有些迟疑，立刻开口说道。
<br/>　　“没有什么不方便的！”
<br/>　　苏莫无奈的摇了摇头，轻笑道：“只不过，我此去有些危险，不想两位随我一起涉险而已！”
<br/>　　他说的是实话，的确是不想两人和他一起涉险，可以这么说，他此行简直就是老虎身上拔毛，危险至极。
<br/>　　“苏兄，我们可不是贪生怕死的人！”
<br/>　　韩盈闻听此言，顿时俏脸上显出佯怒之色，道：“既然你此行危险，那我们更要去了，不能让你孤身涉险！”
<br/>　　“不错！苏莫，你就让我们一起去吧！我们的实力虽然不如你，但也能帮的上忙！”聂乾亦是面色坚定的说道。
<br/>　　虽然两人和苏莫相处的时间不长，但也了解苏莫的为人，绝对是重情重义之辈，而且对他们有恩。
<br/>　　他们两人已经将苏莫当成了真正的朋友。
<br/>　　苏莫闻言无奈，沉吟一番，便点了点头，道：“那好吧！我们明日一早就动身！”
<br/>　　既然两人要去，他也不阻拦了，两人的修为实力也算不弱，到时候或许真能帮的上大忙。
<br/>　　“好！”聂乾和韩盈两人，顿时面露笑容。
<br/>　　随后，两人又和苏莫闲聊了片刻，便告辞离去，准备明早一起出发。
