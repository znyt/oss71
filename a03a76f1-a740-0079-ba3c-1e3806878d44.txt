<br/>				    中午的太阳有些刺眼，然，当她出现在这片地方时，仍感觉到一股森寒的气息传来，而且，腰间的铃铛声又响了起来。<br/>　　<br/>　　    她微皱了下眉，朝周围扫了一眼，这才迈步往前走去。果然不是寻常的鬼魅，烈日当空也敢出现，只不过，它们不敢近她身，既然如此，那便不去理它们吧！<br/>　　<br/>　　    抛出一件寻常的飞行法器代步，渐渐的将身后那跟着的鬼魅甩开，直到，远远看到一处城门时，才从飞行法器上下来。<br/>　　<br/>　　    “黑山城？”她念着这三个字，心下微讶，这是什么地方？朝前面看去，见前方的城门有数名魔修守着，而城里面看起来却好似没人一般，静悄悄的。<br/>　　<br/>　　    要不要进去？<br/>　　<br/>　　    她心下迟疑着，敛起一身气息隐藏到树后去，在那路旁的树后坐了下来，想着观察一下再做打算。然，她在这里坐了半个时辰左右，也没有人从里面出来，更别说进去的了。<br/>　　<br/>　　    就在她准备离开之时，却见一队魔修和邪修以及散修混杂的队伍从那边走来，她看了一眼，那一队约莫二十人上下，结伴而行气势嚣张，人还没走近这边，那响亮肆意的声音就已经传了过来。<br/>　　<br/>　　    “那几人真是在做梦，绝美的红衣女子？哈哈哈哈，我们在这一带活动这么久，也没见过什么绝美的女子，像这样的地方，怎么可能出现那样的人？”<br/>　　<br/>　　    “不错，依我看，他们是想借机会加入我们黑山城，成为我们的人，在这里没有依附的势力，就靠他们那些散走着，估计着不用多久不是被杀就是被抢。”<br/>　　<br/>　　    “他们也不看看，我们黑山城是这么好进的？编故事也不说个可信点的，在这样的万恶之地，女修本来就少，更谈什么穿着一袭红衣的绝美女修了。”<br/>　　<br/>　　    “不错，主子还交待我们到外面抓些女修回来，只不过，这一趟出去也没碰上有女修，就是有，也是被一些实力较强的人护着的，在这里面，找个女人不容易啊！”<br/>　　<br/>　　    “谁让被流放进这万恶之地的都是些恶人多呢？女修？呵呵，估计着女妖倒是有一些。”<br/>　　<br/>　　    那一伙人边说边笑着，朝那城里走去，将近进城门时，其中一人突然回头朝周围看了一眼。<br/>　　<br/>　　    “怎么了？”一人问着，也跟着回头看了一眼。<br/>　　<br/>　　    “我怎么感觉好像有人盯着我们一样？”那人说着，阴狠嗜血的目光朝周围再看了看，神识探出，却也没有探到其他人的气息。<br/>　　<br/>　　    “其他人？哈哈哈哈。”<br/>　　<br/>　　    旁边那人仰头一笑，拍了拍他的肩膀：“你真是多虑了，也不看看这里是什么地方？谁敢上我们这里来？再说了，我们城门前有主子布下的结界，除了佩戴我们腰牌的人之外，谁进得来？又有谁会傻傻的赶上来送死？”<br/>　　<br/>　　    听到这话，那人这才应了一声：“嗯，说得也是，是我想多了。”这才跟着他们的人一起往前走去，进入结界之后再进了那城门。<br/>　　<br/>　　    看到这一幕的凤九眸光微闪，当即悄然离开……<br/>　　<br/>
