　　　　“开始吧”桌上的年轻人看了一眼水盆中的穿山甲开口。
<br/>　　来味鲜阁，大家自然为了是吃平日吃不到的野味，这种东西，可不是想吃就能吃的到。
<br/>　　很多人因为点不到，而愤愤不平，说今日白来了。
<br/>　　如今大家的目光都盯着水盆中的穿山甲看，吃不到，也多看几眼，不少人也羡慕这桌年轻的客人。
<br/>　　涂小安看过菜单，这只穿山甲的价格，足足五万一只，非常的高，而且现在想吃还吃不到。
<br/>　　据说穿山甲有一定的药用，是活血通经，滋阴补阳的灵丹妙药，所以这种动物才会快被人类吃光。
<br/>　　它不像蛇，繁衍能力强，每年一胎，一胎一仔，而且只能野生，不能被人工养殖。
<br/>　　因为养不活。
<br/>　　涂小安看了几眼水盆中的长尾穿山甲，开始点别的野味菜，点了六七个，迎宾退下了。
<br/>　　“恩公，那只穿山甲好可怜”
<br/>　　海棠纵然是猫精，但是至善至美，都是动物，被人摆着桌上吃，有点接受不了。
<br/>　　涂小安冷淡无波，他心智已经淡漠了很多，没什么异样，肉食强势，穿山甲为什么快要灭绝了。
<br/>　　因为它们太弱，弱便是原罪，谁让它们自己那么容易就被人捕抓。
<br/>　　当今，就连觉醒生物，都要成为人类修武者的灵药，能别说普通的动物了。
<br/>　　厨师小伙将水盆中的穿山甲提了起来，出于恐惧或是自卫的本能，穿山甲将躯体紧紧蜷缩着，卷成一圈。
<br/>　　很多人顾不上自己吃饭，都看厨师小伙现场杀穿山甲烧烤。
<br/>　　“现杀现吃，不错吧，只是可惜包厢今天没订到，要在这吵吵嚷嚷的大厅”
<br/>　　一位领头的年轻人对着同伴开口。
<br/>　　“这样也不错，他们只能眼馋，看着我们吃美味，反而多了一种乐趣”
<br/>　　一人回答。
<br/>　　只见厨师小伙动作娴熟地准备把穿山甲拉直，费了半天力，却怎么也无法把那蜷缩的穿山甲拉开。
<br/>　　“喂，你行不行呀，我们什么时候能吃上这肉”
<br/>　　领头年轻人看着他费了半天劲，连穿山甲的身子都拉不直。
<br/>　　“马上就好，马上就好”厨师小伙十分尴尬，本来处理穿山甲就挺麻烦的。
<br/>　　尤其是它蜷缩着的形态，必须先拉直，因为它的铠甲非常坚硬，就连虎豹锋利的牙齿都拿它没办法。
<br/>　　拉不直，拉不动，厨师小伙气的将穿山甲往地面上狠狠摔去，还边摔边解释说，穿山甲遇痛就会将躯体伸张开。
<br/>　　不曾想连摔几下，眼见穿山甲原本惊恐的小眼睛早已闭合，尖尖的嘴角挂出一缕鲜红的血丝，身体却始终未见张开，反而越蜷越紧。
<br/>　　这一幕，让不少人看到，都觉得有点残忍，现场宰杀这种事情，还真的不能看，看了什么口味都没了。
<br/>　　当然，也要看是什么客人，有些心生冷酷的人，还就喜欢好这口，这样都没胃口，你还来吃什么野味。
<br/>　　厨师小伙脸上有怒气，这小家伙是准备砸自己的饭碗吗，客人要是退单不吃了，他就麻烦了。
<br/>　　气的他直接拿铁钳夹了穿山甲放到火盆上灼烧。
<br/>　　“嘤嘤...”
<br/>　　穿山甲发出一阵痛苦的尖叫，烧到它鳞甲脱尽，焦味弥漫，但它仍然保持原状，身子不直。
<br/>　　好像宁死也不想成为人类口中的美食。
<br/>　　“太残忍了”
<br/>　　海棠杏目圆瞪的站起来，想去阻止，可这时，这只被一直灼烧的穿山甲猛然一阵抽搐，僵硬挺直，既然一下子没了气息。
<br/>　　死了，被活活烧死了。
<br/>　　“哼，烧死你，看你身子还蜷着不”厨师小伙反而笑了。
<br/>　　涂小安冷淡的看着，沉默不语，这只穿山甲是挺可怜的，但也仅仅如此而已，他可不会悲天悯地，这种事情多的是，既然成为野味，就逃不过被杀被吃的命运。
<br/>　　他要是还管这种事情，未免有点太天真。
<br/>　　穿山甲一死，随后它的躯体慢慢的开始伸展了开来，可旋即，很多人突然震惊了起来，只见在它摊平的肚皮上，竟蠕动着一只粉嫩透明的小穿山甲，只有老鼠大小，身上的跻带仍与母体相连，小嘴慢慢张合，仿佛在无声地呼唤着母亲。
<br/>　　“呜呜...原来这只穿山甲是在至死保卫着自己的孩子，好伟大...”
<br/>　　海棠看到顿时眼泪涓涓而流，惹人怜爱。
<br/>　　陈高阳由于眼睛看不到，也并不知道发生了什么，茫然若失。
<br/>　　不少客人看到这一幕，都被感动了起来，纷纷说，这只穿山甲好有灵性。
<br/>　　就这样死了太可惜，而且之后还要沦为美食。
<br/>　　“无量寿佛...”
<br/>　　张天师脸色阴郁了起来，不知是不是想起自己的孩子被村民投井，村花也跟着跳井的一幕。
<br/>　　两者何等相似，都是为了孩子。
<br/>　　天地万物皆有灵。
<br/>　　涂小安看到粉嫩透明的小穿山甲时，刹那间他只觉得一股杀意难以仰止，一股阴冷的气息不自觉的散发了出来，大厅中的温度骤然下降。
<br/>　　“怎么回事，好冷好冷”
<br/>　　不少客人顿时觉得浑身发冷，纷纷受不了起来。
<br/>　　涂小安没想到那只母穿山甲自身体重不超过十斤，却用血肉之躯历经摔打与灼烧，都是为了护卫着自己的孩子，被烤至半熟，竟还能保得孩子的周全。
<br/>　　直至自己被活活烧死，也没让小穿山甲受到一点伤害。
<br/>　　原来穿山甲坚硬的铠甲下，藏着一颗柔软的内心。
<br/>　　涂小安再无保持冷血之心，心中某根弦被波动。
<br/>　　脑海中不禁想起了自己老妈的模样，母爱伟大如斯。
<br/>　　“哈哈...这桌客人您们运气太好了，这是中奖了呀，买一送一”
<br/>　　厨师小伙脸上大笑了起来，：“这刚出生的小崽子，肉质更是鲜美好吃”
<br/>　　“那看来我们确实运气不错，快动手吧，我到要看看老得穿山甲好吃，还是小的穿山甲好吃”
<br/>　　领头的年轻人笑了起来，区区穿山甲，还想着保护孩子，畜生而已，这个世界人类为尊，想吃你，你就要沦为食物。
<br/>　　很显然，之前那一幕，并没有让厨师小伙跟这桌年轻的客人有什么心灵上的触动，他们只当这是食物而已，升华不到别的事情上去。
<br/>　　此时的海棠顾不得许多了，直接冲过去，将厨师小伙手中的没了生命气息的穿山甲抢了过来。
<br/>　　“这位小姐，你在做什么”
<br/>　　“坏蛋”
<br/>　　海棠狠狠的瞪了一眼厨师小伙，直接推了他一把。
<br/>　　这一推，厨师小伙瞬间倒飞了出去，飞到了四五张桌子后方的墙壁上，一头砸上去，直接晕倒。
<br/>　　没几个月住院，恐怕是下不了床了。
<br/>　　海棠的举动，让不少客人惊的目瞪口呆，这漂亮的女人好大的力气。
