<br/>　　走廊中，一个赤裸上衣的身影疾飞了出来，装在门后的石墙上，瞬间，整面墙轰然倒塌。
<br/>　　刘哔眉头皱了一皱，不是博士，这金色的瞳孔，似乎是白小飞吧。
<br/>　　让人意外。

<br/>　　标志又魔性的笑声从屋内传了出来，有些癫狂，紧接着，一个肥胖的身影又飞了出来，狠狠砸在了白小飞身上，棱角分明的石块，将两人身上划开几道伤口。
<br/>　　“叔叔！小鹿来救你了！”
<br/>　　看到前方血淋淋的场景，小鹿脸上露出一抹惊慌，随后，坚定的向白小飞跑去。
<br/>　　但刚走几步，就撞在了一个只穿短裤的怪人身上。
<br/>　　比钢铁还要坚硬的肌肉，让小鹿痛呼一声，捂着头跌坐在地上。
<br/>　　“哪来的小屁孩，找死呢吧！”
<br/>　　话语落音，博士嘴角露出残忍的笑容，一把向小鹿的天灵盖抓去。
<br/>　　白小飞口中立马发出一道惊呼，目呲欲裂的看着这边，他清楚博士的实力，这一抓下去，恐怕小鹿也就得交代到这里了。
<br/>　　“是你找死。”
<br/>　　千钧一发之际，刘哔挡在了小鹿面前，伸手抓起了博士的头发，整个人向走廊尽头疾冲过去。
<br/>　　速度之快，甚至连声音都有些跟不上，直到刘哔将博士拖到走廊尽头，尖锐的破空声方才响起。
<br/>　　“砰！”
<br/>　　一声巨响，刘哔的身体冲破墙壁，纵身从五楼跳了下去，随后，震耳的轰鸣声从楼下传来。
<br/>　　片刻，步行楼梯口处，重新出现了刘哔的身影，众人压下震撼的心情，凑了上去。
<br/>　　“多谢，要不是兄弟相助，恐怕……小鹿就会发生危险了。”
<br/>　　双手抱拳，白小飞脸上带着感激，语气听起来也是十分真诚。
<br/>　　“没关系，你就是白小飞吧，刚才小鹿在下面和我说过，我们就是上来找你的。”
<br/>　　刘哔摆摆了摆手，开口回应了一句，脸上笑吟吟的表情，丝毫没有刚杀过人的样子。
<br/>　　众人再度道谢，唯有小胖有些失魂落魄，他还在自责，自己同伴的‘死去’。
<br/>　　根据博士的说法，他明明有机会可以跑的，但都是，为了救他。
<br/>　　“好了，小胖，人都已经死了，仇也报了，我们……走吧。”拍了拍小胖的肩膀，白小飞叹了口气：“都是难免的事情，至少，我们都还活着，对吧。”
<br/>　　挥了挥手，小胖依旧是闷闷不乐的表情，却也没有反驳什么。
<br/>　　“兄弟。”像是想到了什么，白小飞转过头来，看着站在一旁的刘哔：“你这是要去哪里啊，是一个人吗？”
<br/>　　刘哔眉头微微一皱，脸上的笑容微微有些凝固：“我啊，没什么要去的地方，白兄这么问……是什么意思？”

<br/>　　他知道自己说错了话，在末世这种环境下，问出这种问题，很容易让人感觉是不怀好意。
<br/>　　点了点头，刘哔示意自己没有在意，随后摊了摊手，道：“我无所谓，反正也没地方可去，这世界都变成这种鬼样子了，去哪里也都一样。”
<br/>　　“是啊，很狼藉呢。”抱起小鹿，白小飞脸上带着阳光的笑容：“但……那也不能放弃希望嘛。”
<br/>　　刘哔的目光闪动了一下，却也没说什么。
<br/>　　见大家都没有什么异议，白小飞将众人都聚集了起来，除了正在昏迷的小惠，以及一脸悲痛的小胖，其他的人员，倒也还算正常。
<br/>　　“先去我那里吧，我家离这边很近，毕竟时间这么晚了，外出的话，难免会有些危险。”
<br/>　　小惠身上的束缚早就已经被解开，此时怀中抱着贝奇，脸色苍白，语气也有些虚弱。
<br/>　　她知道，这个队伍中，那个十分恐怖的人就混在其中。
<br/>　　先前刘哔和博士谈话的时候，她听的可是清清楚楚。
<br/>　　小惠昏迷过去，所以才会不知道发生了什么事情，但她可没有。
<br/>　　不过……她并没有开口戳穿，因为她知道，那个神秘人是不会让自己说出实情的。
<br/>　　像那种级别的存在，杀死她，就如同捏死一直蚂蚁一般，甚至她能活下去，都是因为博士拼死保护的原因。
