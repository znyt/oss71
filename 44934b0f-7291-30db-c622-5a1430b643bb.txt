　　了解情况后洛冥和段江伟很快就离开了南署基地战场，中途段江伟在段氏基地降落，洛冥则直奔狼牙基地而去。
<br/>　　段江伟带来了大尸潮战役大胜的消息，让段氏基地都一瞬间沸腾，鲁初雪更是立即派人前往南署基地获取详细情况，同时这一战役大胜的战报也发往了各大基地。
<br/>　　没多久这场大战便已经人尽皆知，大尸潮的战争在准备阶段就没有张扬，爆发的太迅猛更没人来得及扬传，所以当华夏的幸存者知晓这一战役的时候，战斗就已经结束。
<br/>　　而狼牙也因这一战再次扬名，更彻底，也更神乎其神。
<br/>　　九万人的狼牙战团，对抗一千五百万数量的尸潮，竟然赢了！
<br/>　　并且将丧尸屠杀的一个不留！
<br/>　　紧接着当全华夏的幸存者都对这场战事津津乐道的时候，鲁初雪派出查探细节的人也终于回来禀报，但结果却让段氏基地的一干高层目瞪口呆。
<br/>　　睡觉？
<br/>　　大战结束后，狼牙战团从上到下竟然席地而睡？
<br/>　　这次战役没有战后详情？！
<br/>　　想到段江伟就是从南署基地回来，鲁初雪不禁投去询问的目光。
<br/>　　段江伟却没有将这一细节放在心上，只是吩咐道：“随后洛老会以他的名义通知各大基地，特殊晶体的事已经落实，你这里准备好接应这批稀有物资。”
<br/>　　没头没脑的话，与当下他们讨论的事毫无关系。
<br/>　　但鲁初雪还是第一时间反应过来，有些兴奋：“这么说来，解毒剂的事为真？这大概是当下最大的发明了吧！楚涵上将是什么反应？”
<br/>　　段江伟没有回答，再次转移话题：“后勤物资的准备继续，这一代的边境根据苏行的建筑工程队跟上进度，所有人准备应对下一场尸潮。”
<br/>　　鲁初雪神色一禀，其余人也立即收起大胜过后的过分喜悦情绪。
<br/>　　他们只不过是赢了第一场尸潮战而已，后面还有更多的尸潮跌重而来，最重要的是这次大战出力的是狼牙战团，而在这整个边境都是需要防守的地方，狼牙战团不可能负责全部的战区。
<br/>　　“北上战争已经结束，南部联盟军很快就会从北而下。”段江伟继续说道，眼底的神色隐隐透露着压力：“到时候全基地的战团都会驻扎在这一带，而这些战团的后勤，也统统都由我们段氏基地负责。”
<br/>　　听着这番话，鲁初雪一惊：“这……我们要负责所有战团的供需？工作量很大啊！”
<br/>　　“人手不够，你们在这次大战中也做的很好，所以接下来的也统统交由你们来安排。”段江伟平静道：“只不过眼下的南部联盟军并非之前分界线大战时的数量，所有战团都在大肆扩军，理论上的安排为狼牙战团扩军到十万，其余战团加起来总计二十万，一共三十万人。”
<br/>　　“三十万？！”
<br/>　　听着这一数字，所有人都惊呆，三十万人的军需物资，统统都由他们来调配？
<br/>　　这可不仅是工作量大，而是压根在考验他们的极限吧？
<br/>　　对比之下当初的银市倾巢大战，简直不是一个概念！
<br/>　　“等一下，三十万人的数量，新加入的十几万士兵都是普通人吧？”
<br/>　　“这里的尸潮也不是普通尸潮，几乎没有低于二阶的丧尸，普通人应付的过来吗？”
<br/>　　“难道要狼牙研究部提供十几万新人的觉醒剂？”
<br/>　　“还有等阶提升剂……”
<br/>　　“供应的过来吗？”
<br/>　　显然这些军官能想到的事，段江伟也想到了，他不禁露出一个笑容：“所以在研究部所需的材料问题上，也麻烦大家多想办法了。”
<br/>　　“又是我们？”鲁初雪目瞪口呆，忽然觉得自己的长官是在跟他们开星际玩笑。
<br/>　　“我说了，人手不足。”段江伟依旧是那副平淡表情，一点没要分担的意思。
<br/>　　段氏基地的这些军官面色全体如他吞了苦瓜一样难看，南署基地大战的胜利喜讯顿时一扫而空，取而代之的是无尽的无力感。
<br/>　　更重要的是他们已经从段江伟的话中听懂了一个重大消息，这一带边境在不久的将来都会变成战区，应对源源不断而来的尸潮，而他们的段氏基地恰好在这里……
<br/>　　没有逃战也没有离开，选择留下的他们，肩上就这么落下了巨大责任。
<br/>　　段氏基地，将成为这片战区的一处重要要塞，接连这边境地带的所有战区！
<br/>　　苏行的工程队在改造完南署基地之后，立即便又在段氏基地开始进行改造，其根本原因也是如此。
<br/>　　在场的都是聪明人，在巨大任务落下后的苦恼情绪没多久，很多人也都想通了以这一点。
<br/>　　一名年轻的军官松了一口气，笑道：“这么说来，下一场尸潮的进攻点原来不是段氏基地啊，我看到苏行的工程队在火急缭绕的改造，压力那叫一个大啊！上千万的丧尸要是就这么涌进来，段氏基地不就毁了？”
<br/>　　哪知听到他的话，段江伟眉头一挑：“下一场尸潮会出现在哪里，神仙才能预测，要塞不仅是各大战区的连接点，同样应对尸潮的战斗力也必不可少。”
<br/>　　好不容易轻松的气氛，再次因为段江伟的一番话而打回原形，很多军官都没有耐心再继续坐下去，接二连三的将手头事情汇报完毕后，就小跑着离开赶紧展开准备工作。
<br/>　　下场尸潮要是真冲着段氏基地来，那还得了？
<br/>　　最重要的是谁知道尸潮什么时候会来？！
<br/>　　狼牙战团在上一个战场全体睡觉，一看就知道一时半会儿都属于零战力状态。
<br/>　　南部联盟军更刚刚从北方南下，啥时候到根本没个定数。
<br/>　　尸潮要是明天来，他们就功亏一篑了啊！
<br/>　　于是就在这种紧张感之下，段氏基地的这些军官都好似开了挂，一个个都进入到了工作狂状态。
<br/>　　看着这些下属有条不紊进行的准备工作，段江伟望着远处天边神情复杂，腰间的紫色太刀闪过一则光辉，显得贵气异常。
<br/>　　“我的这些下属都是精英，虽比不上你狼牙那么变·态，但我训练他们就是为了对付你和狼牙，却没想到不仅没这个机会，还要和你联手，老天真喜欢跟我开玩笑。”
<br/>
