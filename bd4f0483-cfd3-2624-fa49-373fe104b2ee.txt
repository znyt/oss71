　　回到家中，略作休息后，萧逸离开了萧家，到外面买炼药的材料。
<br/>　　手上有五长老‘送’的五千两，不用白不用，多炼些丹药，也好尽快提升自己的修为。
<br/>　　对于别人来说，武道修炼，很枯燥，很乏味，也很辛苦；但对萧逸来说，武道修炼，是很有趣的事情。
<br/>　　他有一颗别人所难以拥有的武道之心，也是因此，当年在地球他才得以将形意拳练出真意，成为一代宗师。
<br/>　　所谓的武道之心，说起来玄幻，实则，那是一种境界，或者说一种意志。孜孜不倦，持之以恒，只为摸索、探寻那更高的武道。
<br/>　　那是一种追求，更是一份赤子之心。
<br/>　　真正的天才，永远不是单纯指天赋，而是这颗难能可贵的坚韧的心。
<br/>　　买回来材料后，萧逸开始了炼丹。
<br/>　　炼药过程，耗时耗力。
<br/>　　将体内真气转化为火焰，而后又要耐心操控火焰的温度，这造成了又耗费真气，又耗费精神，会对人产生很大的疲劳感。
<br/>　　但，当真气消耗完，又继续吸收天地灵气，补充体内真气，这实则达到了修炼的效果。
<br/>　　而像萧逸那样，一刻不停，高强度的炼丹，对于修炼的效果又极佳。
<br/>　　三天后，当所有材料被炼制完，萧逸才停止了炼丹。
<br/>　　三天时间，以萧逸的完美炼药手段和速度，三天算是比较长的了。
<br/>　　他足足买了三千两的药材，炼制了接近四百粒九转炼体丹。
<br/>　　“四百粒九转炼体丹，应该够了。”萧逸略作思索，自语了一声。
<br/>　　他准备凝结气丹了。
<br/>　　这三天的炼药过程，已经将他凡境九重的境界大致稳固了下来。
<br/>　　现在可以考虑突破了。
<br/>　　武道修炼分九大境界，凡境，后天，先天，洞玄，破玄，地元，天元，地极，天极。
<br/>　　每一层又分九重，达到九重，方可突破下一个大境界。
<br/>　　而每一个境界，又有不同的突破要求。
<br/>　　凡境九重后，想突破到后天，必须将丹田淬炼为气丹。
<br/>　　也就是所谓的凝结气丹。
<br/>　　四百粒九转炼体丹，看似非常多，但实则，以萧逸的估计，应该只是刚够自己凝结气丹。
<br/>　　凝结气丹非常难，且需要极庞大的力量。
<br/>　　偌大个萧家，族人数以千计，但后天执事，却只有寥寥数十人。大部分武者，都被卡在凡境九重之下，可想而知凝结气丹的困难。
<br/>　　甚至有许多武者，修炼了大半辈子，始终无法突破到后天境，可想而知需要多庞大的力量。
<br/>　　这也是为什么之前萧若寒想借着在紫云洞内修炼的帮助，凝结气丹。
<br/>　　以紫云洞的浓郁灵气，确实能提升很大的几率。
<br/>　　而现在萧逸不在紫云洞，便只能依靠丹药了。
<br/>　　大概做了下准备，萧逸开始了吞服丹药。
<br/>　　一粒粒丹药入腹，在冰鸾剑的帮助下，快速化为精纯的灵气，游走于四肢百骸当中。
<br/>　　一部分继续淬炼肉身，剩下的一部分全部被丹田吸收。
<br/>　　连绵不断的灵气冲击，不断地洗刷着丹田，使得丹田越发地凝结、坚固、完美。
<br/>　　量变，渐渐化为质变。
<br/>　　丹田上，散发出一股氤氲的光芒，极为美丽。
<br/>　　这是一种蜕变。
<br/>　　当两百粒九转炼体丹吞服完毕，丹田已经蜕变了一半。
<br/>　　萧逸暗暗点点头，自己的猜测果然没有错。
<br/>　　当初突破凡境五重时，用了二十粒九转炼体丹，以凡境武者每突破一重所需的灵气翻倍来算，凝结气丹，四百粒丹药足够了。
<br/>　　丹田蜕变的过程，很是美妙，那淡淡的氤氲光芒，暖暖的，让人很舒服。
<br/>　　渐渐的，萧逸沉浸在这种舒服中。
<br/>　　然而，不知过了多久，他却猛地一窒，体内一阵锥心的疼痛。
<br/>　　确切来说，这股疼痛，来源于丹田。
<br/>　　“怎么回事？”萧逸一惊，当即内视。
<br/>　　一看之下，顿时大惊失色。
<br/>　　那股氤氲光芒，不知从何时起，竟变成了一股红蓝交错的力量。
<br/>　　两股力量在不断交替、穿梭、翻腾，如同冰火在互相冲击。
<br/>　　萧逸先是疑惑，而后明了。
<br/>　　武者修炼，是靠武魂吸收天地灵气，而后再由武魂传入丹田。
<br/>　　什么属性的武魂，就会将灵气转化为什么属性的力量，最后‘洗刷’丹田。
<br/>　　如果是火属性的武魂，最终凝结的丹田，会成为一颗火红色的气丹。
<br/>　　凝结气丹，之所以称之为蜕变，就是因为气丹调动灵气的速度更快，更猛，让武者的战力更强。
<br/>　　而与自身武魂相匹配的气丹，方能让武者发挥出最强的力量。
<br/>　　可是，自己是双生武魂。
<br/>　　丹田在中，控火兽武魂与冰鸾剑武魂一左一右，两者转化的力量也是一火一冰，自然有了两股力量去冲击洗刷丹田。
<br/>　　就是因为这两股不同且相反的力量，在不断交错，这才造成了自己如今丹田上的疼痛。
<br/>　　所谓水火不容，两种互相克制的力量，竟不断交错冲击，这是极度危险的事。
<br/>　　一个不慎，很可能会造成丹田尽毁。
<br/>　　“该死，竟然出现了这种状况。”萧逸咬了咬牙。
<br/>　　“遭了，再继续下去，一旦两种力量失去平衡，丹田必然受损，甚至修为全失；可是，若现在放弃，就必须完全散去之前的力量。”
<br/>　　“不行，必须坚持下去，富贵险中求，什么大风大浪我见过，我就不信我过不了这一关。”
<br/>　　若是其他人遇到这种情况，必然会惊慌失措。
<br/>　　可萧逸，却立刻稳住了心神，并作出了决定，没有任何犹豫不决。
<br/>　　武者，本身就需要一颗一往直前的勇敢之心。
<br/>　　“拼了。”萧逸心中大喝一声，随后继续吞服丹药。
<br/>　　又是一股股精纯力量入腹，化为冰火力量洗刷丹田。
<br/>　　萧逸紧张地控制着吸收的速度，不敢有丝毫懈怠。
<br/>　　时间一分一秒过去，萧逸也不知过去了多久，只知道，当他将四百粒九转炼体丹完全吞服完时，已然满头大汗，衣衫尽湿。
<br/>　　“终于成了。”萧逸松了口气，淡淡一笑。
<br/>　　在那股高度紧张的情况下，动辄有莫大危险，却还是要稳住心神，不敢有丝毫懈怠，最终完成凝结气丹。
<br/>　　也亏得萧逸意志过人，否则后果还真的难说。
<br/>　　萧逸内视了一眼，如今，丹田的位置，已经变成了一颗红蓝交替的气丹。
<br/>　　左边一半红色，右边一半蓝色。
<br/>　　这颗不一样的气丹，姑且命名为冰火气丹。
<br/>　　冰火气丹的两旁，左边控火兽武魂，右边冰鸾剑武魂，静静地伫立着。
<br/>　　“喝。”萧逸站了起来后，一拳轰出，试了试威力。
<br/>　　利用气丹，调动体内灵气，一拳之下，空气震动，威力强悍。
<br/>　　“好强，调动灵气的速度和猛度，竟是比之前的丹田强了十倍都不止。”萧逸暗暗一惊。
<br/>　　在凝结气丹之前，萧逸就曾经向三长老请教过有关的注意事项。
<br/>　　凝结气丹毕竟是凡境武者的一道坎，且是重中之重，萧逸当然要事先搞清楚情况。
<br/>　　据三长老所说，凝结气丹后，气丹的威力，比之丹田，强了五倍左右。
<br/>　　当然，那是对于普通气丹来说。
<br/>　　可是，冰火气丹，竟然比之丹田，强了十倍都不止。
<br/>　　这代表着，萧逸如今打的基础，比其他同等级武者，强了许多；所能发挥出的实力，也更为强悍，而且强了不止零星半点。
