深入。<br/>　　<br/>　　    一晃，几个小时过去了，三井美奈的队伍越来越深入，他们根据导航仪上的卫星定位判断，自己至少已经下降到有两千米左右的深度了，几乎可以判断马上就要抵达天坑底部了。<br/>　　<br/>　　    三井美奈等人的运气要比王惊蛰他们好了不少，当这一伙人从裂缝深处中出来的时候，迎接他们的没有白森森的人骨，只有一片鸟语花香和令人窒息的美景。<br/>　　<br/>　　    看着一片古老的森林，耳朵里还有着轻微的流水的声音，这伙人都情不自禁的被这幅景象给惊住了。<br/>　　<br/>　　    三井美奈微微的张着嘴唇，呢喃的说道：“不敢相信，在已经被人类踏遍了绝大部分土地的地球上，还会存在着一个未知的世界，这种景象……在我们帝国，根本就是不可能的存在的，因为我们那里太小了”<br/>　　<br/>　　    三井小叶不甘的叹了口气，说道：“当年的决策是正确的，只是可惜了，我们没能占领这片土地，不然华夏这处领土上所有的东西，都将归属我们大和民族了，自然也包括这里”<br/>　　<br/>　　    当一帮小鬼子沉浸在震撼中时，忽然间自林中突然传出一声震天的吼声。<br/>　　<br/>　　    “吼……”<br/>　　<br/>　　    “唰”三井美奈等人当即愣了愣，小林昙一说道：“是在山外碰见的那头藏獒，原来它也在这？”
