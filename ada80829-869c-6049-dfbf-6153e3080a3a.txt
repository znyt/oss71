　　第493章
<br/>　　北人骑马，南人乘船，江南一地水道纵横，李乐天率领的帝国军不习水战，如果真派大军打过去，说不得会陷在扬州出不来。
<br/>　　所以，尽管李乐天心底恨得牙痒痒，也没办法去讨伐李星。只能向天下人下了讨贼令，声明李星是逆贼，天下人人都可攻打他。
<br/>　　扬州一地是天下最富饶的一块地方，不知多少人盯着这里流口水。现在李星也成了帝国人人可杀之贼，不过是大家忌惮他的实力，暂时未动手而已。
<br/>　　一南一北，成了两帝对峙的局面。除了南北两帝外，李乐天另封了十八位诸侯王，在诸侯王的属地中，他们都有立法权，行政权，税收权。
<br/>　　在这次瓜分利益蛋糕的过程中，崔相没能如愿将李行哉推上帝位。但不能说他就失败了，如这样传承许久的大家族，永远不会在赌桌上投入所有赌注，他们总有办法让自己在任何情况下都获得利益。
<br/>　　十八位诸侯王，有两位是相府推出的代表，分别是凉州王宋伯康以及兖州王雷天音。
<br/>　　至于李行哉……他已失去了利用价值，自然而然就被淘汰出局。不过终究是帝国皇子，也在这次讨贼行动中露了脸，虽然表现有些废物，但帝国的大人物终究不能当做什么事情都没有发生过。最后李行哉也混了一个诸侯王的位置，封地在徐州落凤城。
<br/>　　落凤城是一座小城，人口不过十万人，地理位置处于徐州、扬州、兖州交界处。扬州是李星的地盘，兖州是相府的地盘，徐州又是将军府的地盘。所以这里地理形势极其复杂，一旦发生战争，落凤城首当其冲。
<br/>　　再加上这里土地贫瘠，城中居民只能靠出海打鱼为生，又常常有海盗作乱，居民的日子十分难熬。
<br/>　　这一日，一匹黑马出了京城，牵马的是一年轻人，走路一跛一跛。这个跛子，自然就是要去落凤城走马上任的李行哉。
<br/>　　在并州负伤后，他的左腿还是留下些残疾。想今日离了长安，就要去自己的封地，临走前连个送行的人也无，随身不过就是一匹黑马而已。
<br/>　　他苦笑一声，罗凤城的情况，他事先当然已经有所了解，不过从他面容上，并看不出有多少颓废之意，相比从前，眼中倒是多了几分生气。
<br/>　　城门口张贴着榜文，许多人围着看。李行哉倒是没有去凑这个热闹，因为他知道那上面贴的是什么。
<br/>　　这是朝廷下的招贤令，不问出身，只要有一技之长则，都可以到刚刚成立的招贤馆谋个一官半职。
<br/>　　李行哉知道，不仅朝廷，其他诸侯王也有同样的动作，招贤纳才，广养门客，未来或许是武夫、杀手、说客的天下。
<br/>　　李行哉没有多停，牵着马上路，落凤城的情况很复杂，到时该如何应对，自己还没想好。而且从京城到落凤城，两地相距千里之遥，路上会碰上什么事还不知道。不说那占据山林的山贼恶霸，说不得还会有其他势力派出来刺杀自己的杀手，半路上给自己一刀，自己也就一命呜呼了。
<br/>　　当他牵着马离开时，与一个女子擦肩而过，李行哉往那女子身上看了一眼，见她生得平平无奇，也没什么出奇之处，但不知怎地，二人错肩的那一刻，却令李行哉心中一寒。
<br/>　　他摇头笑笑，现在自己面前还是麻烦一大堆呢，就不要多事关心与自己无关的事情了。
<br/>　　……
<br/>　　凉州，蛤蟆城。
<br/>　　程大雷窝在小黑屋中，分析研究着喜字队送来的情报。口中啧啧出声：“这李乐天是养虎为患呐。”
<br/>　　阿喜坐在程大雷对面，道：“大当家有什么看法？”
<br/>　　程大雷抿了口茶，慢条斯理道：“两帝十八王，李乐天的朝廷对帝国的控制也就限制在京州、并州一带。他这不仅是养虎为患，而是一口气养了十八头恶虎。国不可有二君，山不可有二虎，养来养去，最后也只是狗咬狗，看吧，这帝国会越来越乱的。”
<br/>　　“那大当家依你看，最后这场狗咬狗，谁会是活到最后那条狗？”阿喜问。
<br/>　　“这我哪里看得出来。”程大雷道：“现在各方势力，不都在下注么，希望自己押中的人能走到最后。不过天下大势，瞬息万变，现在最不被看好的，未必不能成为最后一个。对了，李行哉现在干嘛呢？”
<br/>　　“他已经在去落凤城的路上，更多情报我们便不知道了。”
<br/>　　“这一路过去，怕是不太平呐。”程大雷想了想，道：“你让白元飞过来，我有件事要他去做。”
<br/>　　过不多时，阿喜带着白元飞一块儿过来，程大雷道：“现在，有件事要你去做，你出趟城，脚力快的话当能赶上李行哉。找到他后，莫要与他相见，只在暗中保护便是。等他什么时候平安到了落凤城，你再回来。对了，你应该能找到他吧？”
<br/>　　“只要他这路上稍微搞出些动静，想要找到他并不难。”白元飞道。
<br/>　　“好了，速速去吧，回来的时候，沿途有什么情报，也记得带回来。”
<br/>　　白元飞走后，立刻骑了一匹快马出城，直奔京州而去。
<br/>　　而程大雷，仍旧窝在屋里，处理其他事情。他还蛮适应现在的生活，这些日子过去，身上又胖了两斤。
<br/>　　阿喜立在一旁，道：“大当家，您悟剑悟得怎么样了？”
<br/>　　“蛮好啊，我早也悟，晚也悟，都快捂出痱子来了。”程大雷随口道。
<br/>　　“那大当家准备什么时候出关，现在大家都在猜测……”
<br/>　　程大雷撇撇嘴，他可不敢拿自己的性命开玩笑，解决不掉聂隐娘，他是不会离开这屋子一步的。
<br/>　　心中忽然一动，程大雷问阿喜：“这段日子，有没有哪个女子入城？”
<br/>　　“大当家要找哪个女子？”
<br/>　　“这……”程大雷叹口气道：“我也不知道她是怎样的女子啊，不过很危险，致命的危险。”
<br/>　　阿喜一头雾水，根本无法理解程大雷在说什么。
<br/>　　“无论是怎样的女子，凡是入城的女子都要仔细盘查，得到的情报都要送到我这里来。”
<br/>(本章完)
