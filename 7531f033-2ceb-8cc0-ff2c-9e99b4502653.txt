<br/>　　“不知道这一局蓝队这边会拿出什么样的阵容来，现在他们士气正红，感觉会拿出一个偏前期的阵容！”看着双方在ban人，夜雪狂歌开口说道。
<br/>　　“应该可能，第一局比赛大家的手已经打热了，这个时候拿出一个强势的阵容倒是可以。”叶帝跟着应道。

<br/>　　双方飞快的选人，很快雷霆战队这边的阵容便是确定了下来，他们这边上单依旧选择了武器、打野则是选择了蜘蛛、中路沙皇、下路女警和娜美这对组合。
<br/>　　而蓝队这边则是确定了四个英雄，只留下最后一个上单counter位没有选择，他们这边确定的四个英雄分别是打野盲僧、中路蛇女、下路是圣枪游侠和婕拉这两个英雄。
<br/>　　“不知道蓝队这边最后一手会选择什么英雄来和武器对线，其实鳄鱼这个英雄是一个很好的选择。”米粒在那里开口分析着。
<br/>　　“没错，在前期鳄鱼还是很好压制武器的，不过真要是被拖到了后期，鳄鱼这个英雄就真的是打不过武器了，蓝队这边的阵容是属于偏前期那种的，选个鳄鱼倒是可以。”林雅楠跟着开口说道。
<br/>　　就在林雅楠这边话音刚落之际，秦霄这边已经是锁定了他们这边的最后一个位置，看到秦霄确定的那个英雄，现场顿时传出一阵倒吸冷气的声音。
<br/>　　“潘森？！”米粒诧异的大声喊道。
<br/>　　“这个阵容我好像见过，潘森、盲僧还有蛇女这个组合在之前大魔王那次直播的时候出现过，只不过那次是香蕉和夜雪两人选择的这两个英雄，而且当时将中路的大魔王给直接打崩了，这个组合只要被潘森的w技能控到，基本上就是无解的。”林雅楠大声的喊道。
<br/>　　直播间里看到蓝队的这个阵容，无数的观众也都是纷纷刷起了666的弹幕。
<br/>　　“66666666666666，这个阵容我见过，厉害了我的蓝队。”
<br/>　　“求白队心里阴影到底有多大，明明是他们的套路，结果被蓝队这边给抢用了。”
<br/>　　“没想到潘森这个英雄真的出现在了比赛场上，而且还是在和雷霆战队这样强力的战队中使用的，真的期待啊。”
<br/>　　“潘森的硬控加上他的被动抗塔，蛇女的w技能剧毒迷雾可以让敌人无法释放出位移技能，加上盲圣前期那超高的爆发，这几个英雄在前期其实就可以越塔秒人的！”
<br/>　　“6666666666666，这个真的是太强了，上一次看大魔王直播的时候怎么就没想到这个套路会上场呢。”
<br/>　　韩国ST战队，
<br/>　　“阿西！！！居然被他们先使用出来了，这个套路可是我们明天对阵KA战队的时候使用的啊。”金在贤一脸的可惜。
<br/>　　“景禹教练还是厉害，看过小叶一场比赛就将这个套路给用出来了。”matin开口称赞道。
<br/>　　“现在最伤心的估计就是小叶子他们了，这个套路明明是他们开发出来的，可是现在居然让蓝队给率先使用出来了。”金相赫倒是一脸的轻松，在那里笑着调侃道。
<br/>　　“教练放心吧，没有这个套路我们也一样可以取胜的，而且就算是KA战队这边知道了，他们也不一定会ban掉这几个英雄吧。”下路的射手coco开口说道。
<br/>　　“就是啊，选择蛇女和盲僧这两个英雄实在是太正常了，除非他们一开始就将潘森这个英雄ban掉，要不然这个套路完全可以执行的。”辅助newlife同样一脸的不以为然。
<br/>　　现场看到这个阵容，叶帝几人的嘴巴也是张的老大，事实上就在那场比赛结束之后，他们这边便是将这个套路当成了半个大招来用了，打算在遇到皇族的时候释放出来，到时候打对方一个绰手不及，可是没想到居然在半道上被蓝队给截胡了，这让他们如何不气。

<br/>　　“无耻啊，居然把我们的套路给偷走了，心疼自己一波！”茄子也是跟着开口谴责道。
<br/>　　“事实证明这个套路还是很强的，这个套路真正的核心就在于潘森和蛇女这两人，打野换成其他输出高的英雄其实也可以，只不过盲僧这个英雄比较灵活，在很远的己方就可以接上自己的技能，他是最好的选择。”叶帝也是有些无奈。
<br/>　　“是啊，只要潘森控到对方，蛇女接上自己的w技能让对方无法使用出位移技能就可以了，这个套路可是很完美的。”夜雪跟着开口。

<br/>　　对方的这个套路实在是不给他留活路啊，不带净化只要给机会，潘森控到自己，那几乎是必死的！

<br/>　　“小刀你一定要做好视野啊，这一局我崩了就没法玩了。”陆伟对着小刀说道，语气中还满是忐忑！
