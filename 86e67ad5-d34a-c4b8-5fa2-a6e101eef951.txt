　　赵城和郭剑青两人，均是击出了巅峰一击，来迎击苏莫的三色剑气。
<br/>　　不过，两人受到吞噬战魂的影响太大，不仅战魂发挥不出多大的威能，连两人的修为玄力，威力都是大大减弱。
<br/>　　虽然两人这一击，都很是强大，但比先前没动用战魂之时，也没有多大的变化。
<br/>　　瞬息之间，两人的攻击便和苏莫的剑气相击在了一起。
<br/>　　轰！轰！
<br/>　　两声爆响声震动万里，拳印和金色的剑光，根本挡不住三色的剑气，顿时被斩击的爆裂开来，化为滔天气浪。
<br/>　　咻！咻！
<br/>　　三色剑气势如破竹，所向披靡，斩开了拳印和剑光之后，闪电般的向赵城两人身上斩去，瞬息而至。
<br/>　　“不好！”两人心中大骇，他们的实力受到了巨大的影响，几乎被削弱了小半，而苏莫的实力，却是直线暴涨。
<br/>　　此消彼长之下，他们已经完全不是苏莫的对手了。
<br/>　　眼见着剑气临身，仓促之间，郭剑青只能急忙巨剑封挡。
<br/>　　而赵城，左手中光芒一闪，出现了一块硕大的黑色盾牌挡在了身前。
<br/>　　此盾黑光流转，气势颇为不凡。
<br/>　　下一息，两道浩大的三色剑气，便纷纷击在了盾牌和郭剑青的长剑上
<br/>　　轰！轰！
<br/>　　又是两声惊天爆响，接下来，战况完全反转了过来，赵城和郭剑青两人，直接被击飞了出去。
<br/>　　赵城还稍好一些，用盾牌挡住了剑气的锋芒，郭剑青身上则是被炸裂的剑气切出数十道伤口，挥洒出一溜长长的血箭。
<br/>　　轰隆隆！！
<br/>　　随即，两人的身躯从高空坠下，砸在了大地之上，让得大地崩裂，出现了两个巨大的深坑。
<br/>　　“咳咳！怎么可能？怎么会有这么多的武魂？”深坑之中，郭剑青不停的咳嗽，口中有血液流出。
<br/>　　他的眼眸之中，留露出深深的震撼，还有浓烈的恐惧。
<br/>　　他倒不是被苏莫的实力吓到了，而是被苏莫的天赋吓到了。
<br/>　　若不是亲眼所见，他根本不会相信，这个世界上，居然有人拥有十几种武魂。
<br/>　　“剑青，我们快走!”
<br/>　　另一边，赵城狼狈不堪的从深坑中飞了出来，招呼郭剑青一声，便想立刻逃走。
<br/>　　赵城倒是没受多大的伤，只是被震出了一点内伤，无伤大雅，这多亏了他手中的盾牌。
<br/>　　此盾虽然是帝级上品宝器，但品质极佳，非常接近尊器，所以才能有如此强大的防御力。
<br/>　　至于杀苏莫，现在已经是不可能了！
<br/>　　他们那里知道，苏莫居然是如此恐怖的妖孽，不然的话根本不会来。
<br/>　　此时，赵城心中甚至非常的愤怒，对太史轩产生了愤恨之心。
<br/>　　太史轩不知道苏莫的底细，居然就派他们出手，这是将他们往火坑里推啊！
<br/>　　嗖！嗖！
<br/>　　赵城和郭剑青丝毫不敢停留，飞出深坑之后，便立刻展开最快的速度逃窜。
<br/>　　“你们逃得了吗？”
<br/>　　苏莫冷漠的声音响起，身形如大鹏展翅，从天而降。
<br/>　　赵城和郭剑青，一直在吞噬之力的笼罩范围之内，速度远远不及苏莫，根本逃无可逃。
<br/>　　嗖！
<br/>　　苏莫速度如电，呼吸之间便接近了两人百里之内。
<br/>　　他距离两人越近，赵城两人感受到的吞噬之力便越强，随着苏莫的不断接近，两人的速度不断的大减，仿若陷入了泥藻之中。
<br/>　　“怎么回事？这是什么力量？”郭剑青吓得亡魂皆冒，大声的吼叫了起来。
<br/>　　他已经受了不轻的伤，若是不赶紧逃走，那就必死无疑了。
<br/>　　赵城同样是脸色苍白，不过他的实力比郭剑青稍强一些，也没受多大的伤，倒不是太过恐惧。
<br/>　　两人对视一眼，随即立刻停了下来，转身望向苏莫。
<br/>　　在吞噬之力下，他们根本逃不走，所以索性不再逃了。
<br/>　　“怎么不逃了？”苏莫飞到了两人身前五十里处，长身而立，巨大的吞噬战魂，以及十几种器武魂悬浮在身后，强大的威势震人心魄。
<br/>　　赵城和郭剑青，极力的控制着自身的身形，压制着体内的玄力和气血，免得被吞噬武魂吸过去，这让他们倍感吃力。
<br/>　　强大的吞噬之力，甚至让他们体内的神丹，都不断的震颤。
<br/>　　赵城瞥了一眼苏莫身后的十几种武魂，暗暗的吞咽了一口吐沫，随即沉声道：“苏莫，我们向你道歉，此事到此为止如何。”
<br/>　　赵城的打算，是和苏莫化干戈为玉帛，然后逃得性命，若是行不通，那就只能拼死一搏了。
<br/>　　“你们来杀我不成，道个谦就想完事？”苏莫闻言冷笑了起来，他也不急着出手，因为时间拖得越久，两人的实力越弱。
<br/>　　在他的吞噬战魂之下，对方两人体内的玄力和气血，根本不压制不住，不断的向外逸散。
<br/>　　虽然逸散的速度不快，但随着时间一长，对于两人实力的影响，可不是一星半点。
<br/>　　所以，他并不着急，到时候不费吹灰之力，就能斩杀两人。
<br/>　　此事，苏莫心中也是有些感慨，他也没想到自己的吞噬武魂，进化为战魂之后，全力催动之下，威力居然如此之强。
<br/>　　不愧是完美战魂！
<br/>　　战魂分为四个级别，这四个档次，分别为低级、中级、高级和完美。
<br/>　　这种级别之分，与战魂的种类、能力无关，只对于自身战魂的威力有影响。
<br/>　　想要进化完美战魂，首先就要有天极九阶的武魂，其次要有非常契合，而且非常强大的奥义，最后需要超强的神魂。
<br/>　　四者缺一不可。
<br/>　　苏莫当初武魂在天极七阶之时，若是用不朽奥义进化战魂，虽然他的精神神魂强大，进化成的吞噬战魂，顶天了也就是高级战魂，最有可能的是中级战魂
<br/>　　事实上，八成以上的武帝境武者，都是低级战魂，还有一成是中级战魂。
<br/>　　而剩下的最后一成之中，基本上都是高级战魂，完美战魂者寥寥无几。
<br/>　　所以，进化为完美战魂之后，吞噬战魂的威力，暴增了十几倍都不止。
<br/>　　“苏莫，我们可是太史世家一脉的人，你知道杀了我们会有什么后果吗？”赵城阴沉着脸，冷冷的说道。
<br/>　　“哦？什么后果？”苏莫冷笑一声，眸中露出戏虐之色。
