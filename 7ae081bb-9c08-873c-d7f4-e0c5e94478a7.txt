
    本书写到这里，基本上第一部分就基本上结束了，多谢各位读者的一直以来的支持，你们的你们的收藏、推荐、评论、打赏，我都非常清楚，因为我一天到晚不知要看多次后台，还有一直以来订阅的朋友，那就更不用说了，简直是“衣食父母”啊，非常感谢！<br/>　　<br/>　　    本书是我第一本，但我是非常用心的，书中很多情节都是来自于现实，因为我觉得“现实要比精彩的多”，我不是个络从业者，但为了不出现大的漏洞，<br/>　　<br/>　　    我可以说是把很多络大佬的人物传记、络视频采访看了一遍，可能只是为了点小故事、小情节，书中很多事件除了时间上有冲突外，基本都能对应起来，所以，我觉得我这本都市后期不会崩溃了，因为都是来自现实、基于现实的描写，<br/>　　<br/>　　    而且我还有很多自己的思考、创意，比如垄断nn软件，甚至一上来的第一桶金，我本打算众筹开个吧，再卖掉来，<br/>　　<br/>　　    而且，我还对很多企业的失败进行了复盘，假设它当年的创始人能像主角重生回到过去，又会怎么做呢！<br/>　　<br/>　　    要说本书最大的问题，应该就是感情戏，一开始没设定好，不该弄个前世老婆来，快把主角快塑造成一个渣男了，<br/>　　<br/>　　    还不如前世就根本没有老婆好，直接开后宫的，新手多担待吧，但有一点我敢肯定的是我只要还写书，肯定开后宫，反正我无法理解不开后宫的都市文，<br/>　　<br/>　　    写着这里，本书也54万字了，不容易了啊，我本以为30万上架的时候能结束与滕讯的战斗呢，没想到写了这么多，可能是水了点，所以要多谢兄弟们的支持，感谢你们的不离不弃，还能如此包容与轩如此任性的更新，<br/>　　<br/>　　    不过我一直也在努力，从刚开始的一天2000字，到一天4000字，再到现在保底4000字，偶尔6000字，我也一直再进步啊，所以还请各位书友再多给与轩点时间，我一直在努力在进步，<br/>　　<br/>　　    看到这里，估计很多书友也熟悉本书的套路了，商业主线很简单，就是干！就是在某个很牛逼的企业崛起的时候，被主角摁倒地上狠狠的蹂躏，之所以选择这么个时间线，也是为了对怼，太早了，可以直接收购，没意思了太晚了，就不好打了，赢了也显得有点假了。<br/>　　<br/>　　    而且下面也要开始进入硬件了，硬件不同软件，互联这种虚的东西，1个亿就算很多了，但像芯片这种最顶端的制造业，别说1亿美金，哪怕是十亿美金、百亿美金，也不过是打个水漂而已，而我喜欢在尊重现实的基础上改进，多点时间准备。<br/>　　<br/>　　    总之，谢谢大家，一直想建个书友群来，最后想想还是放弃了，等上百万的时候，再建个吧，现在估计也没这么多人，<br/>　　<br/>　　    最后再次谢谢大家，多谢诸位书友一直以来的支持！只要能看这里的，不管是看正版，还是看盗版的，都说明你对本书还是有所期待了，还请继续支持我，多谢大家了！8
