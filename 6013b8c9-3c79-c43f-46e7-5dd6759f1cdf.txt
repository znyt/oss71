　　九幽魔渊，地下世界。
<br/>　　高空之上，一片混乱的景象，无数的玄力、气血、武魂和尸气滚荡不休。
<br/>　　苏莫淡然而立，望着这些汇聚而来的资源，他略一沉吟，手掌一翻，便取出了十亿魔剑。
<br/>　　“桀桀桀！！！”
<br/>　　十亿魔剑乍一出现，魔剑中魔灵森然的大笑声便传到了苏莫的脑海中。
<br/>　　“魔灵，这些气血全是你的！”苏莫冷喝一声，随即手臂一抛，便将魔剑抛进了无尽气血之中。
<br/>　　刹那之间，魔剑之上魔光爆射，开始吸纳无尽的气血。
<br/>　　而苏莫，也在高空中盘坐了下来，开始吸纳这些玄力和武魂。
<br/>　　如此多的玄力和武魂，他自然不会让其浪费了。
<br/>　　玄力入体，苏莫立刻极力炼化了起来，大量的各式武魂，亦是不断的被他吞噬。
<br/>　　时间缓缓流逝，约莫半个多时辰之后，一切都归于了平静。
<br/>　　所有的玄力、武魂、气血，全部都消失殆尽。
<br/>　　苏莫的修为，从武皇境四重的初期，提升到了中期，距武皇境四重的巅峰，还相差甚远。
<br/>　　这些黄泉魔宗的弟子，修为太低了，玄力驳杂、稀薄，根本不堪大用。
<br/>　　不过，数十万黄泉魔宗弟子的武魂，却是也没有让苏莫的武魂晋级。
<br/>　　现在他吞噬武魂晋级，所需要吞噬的武魂是在太多了。
<br/>　　抬眼看向不远处，之间十亿魔剑静静的悬浮在虚空中，漆黑的剑身上爆射出万丈的血光，霎时恐怖。
<br/>　　唰！
<br/>　　苏莫大手一张，直接将十亿魔剑吸到了手掌中。
<br/>　　感受着魔剑中再次壮大的魔灵，感受着十亿魔剑中蕴含的力量，苏莫心中惊讶。
<br/>　　他感觉，此刻的十亿魔剑，威力应该不比他的帝皇剑弱了。
<br/>　　他屠灭殷家之时，魔灵便吸收了无尽的血液，现在又吸收了数十万人的血液，估计也差不多恢复了！
<br/>　　“魔灵，恢复到巅峰了吗？”苏莫意念中传出一道询问之声。
<br/>　　“桀桀！快了，已经恢复了七成！”魔灵怪笑道。
<br/>　　“七成！”苏莫闻言双眉一挑，这十亿魔剑之强，看来远超他的预料。
<br/>　　吸收了这么多的气血，变得如此强大，居然才恢复了七成。
<br/>　　那若是全部恢复，能达到什么地步！
<br/>　　要知道，他的帝皇剑可是帝级上品神兵，难道十亿魔剑还能超越帝级不成。
<br/>　　“你若恢复到巅峰，是什么级别的兵器？”苏莫又问道。
<br/>　　“无限接近尊器！”魔灵道。
<br/>　　苏莫闻言倒吸了一口凉气，不是说十亿魔剑只是稍强一些的帝器吗？居然无限接近尊器！
<br/>　　尊器，那可是武帝之上的强者，武尊之境大能的兵器。
<br/>　　“有这么强？”苏莫惊讶的问道。
<br/>　　“嘿嘿！那当然，当年主人炼制本剑，便是为了对抗试炼者！”
<br/>　　魔灵嘿嘿一笑，随即哀叹一声，继续道：“不过那个年代的试炼者太强了，主人还是失败了！”
<br/>　　苏莫闻言点了点头，随即他不再多问，便将十亿魔剑收了起来。
<br/>　　扫视了一番下方的四座城池，苏莫手掌连连拍动，一道道巨大的掌印，如同一座座大山般镇压而下。
<br/>　　轰隆隆！
<br/>　　在一阵惊天动地的爆响之中，烟尘冲天，滚滚涌动，四座城池灰飞烟灭，只留下了四个巨大的深坑。
<br/>　　唰！
<br/>　　随即，苏莫转身离去，不多时便飞出了九幽魔渊。
<br/>　　飞临九幽魔渊上空，望着黝黑的魔渊入口，苏莫略一沉吟，便将远处一座千丈荒山搬了过来，直接砸在了魔渊的入口上。
<br/>　　轰隆隆！
<br/>　　又是一阵惊天动地的巨响，霎时之间，魔渊的入口被彻底堵死了，变成了一处乱石林立的乱石堆。
<br/>　　而后，苏莫便离开了此地，返回苍穹神宫。
<br/>　　……
<br/>　　几日后，当苏莫接近苍穹神宫的山门之时，远远地，他便发现神宫水幕大阵之外，围拢了数以百万计的武者。
<br/>　　“嗯？怎么回事？”苏莫心中惊疑，但还是径直向神宫的山门处飞去。
<br/>　　“苏宫主！”
<br/>　　“苏莫！”
<br/>　　“苏莫宫主回来了！”
<br/>　　苏莫乍一现身，便引起了全场的轰动，所有的人都向他望了过去。
<br/>　　苏莫眸光扫视众人，发现这些人中，有一部分是上次在天海城被他所救之人。
<br/>　　“苏宫主，我们要加入苍穹神宫，来参加考核！”
<br/>　　“我已经在此等了两天了！”
<br/>　　“苏宫主，我们也想加入苍穹神宫，不知道有没有机会？”
<br/>　　“我想成为苍穹神宫的弟子！”
<br/>　　无数嘈杂的声音此起彼伏，纷乱不堪。
<br/>　　这些人，其中一小部分是当初在天海城被苏莫所救之人，那些人说过要加入苍穹神宫，现在则来了一部分。
<br/>　　还有一大部分人，则是听闻了苏莫打败巫森，从各地特意赶来的。
<br/>　　毕竟，苏莫打败巫森，便代表了苍穹世界无敌的实力，再加上苍穹神宫的底蕴，谁不想加入苍穹神宫。
<br/>　　目前在所有人看来，加入苍穹神宫便代表着身份、地位、前途。
<br/>　　只不过，苍穹神宫自从出世以来，从不招收弟子，众人也是不抱太大的希望。
<br/>　　这些到来之人，多是以年轻人为主，老一辈强者也有一小部分。
<br/>　　唰！
<br/>　　苏莫飞临水幕阵法之外，停下身形，转身扫视此地的无数武者。
<br/>　　略一沉吟，他朗声喝道：“明日，我苍穹神宫便会大开山门，广招弟子，只要通过考核，便能成为苍穹神宫的一份子！”
<br/>　　如今苍穹神宫可以扩张了，所以，苏莫准备从明日开始，正式招收弟子。
<br/>　　“耶！”
<br/>　　“苏宫主威武！”
<br/>　　“哈哈！我一定要成为苍穹神宫弟子，光宗耀祖！”
<br/>　　苏莫话音一落，霎时间周围便响起山呼海啸般的声音，无数的人振奋不已。
<br/>　　“苏宫主，你们招收弟子有什么要求？”
<br/>　　“对啊！具体有多高的标准？”
<br/>　　有冷静之人立刻大声询问，这才是关键之处，若是要求太高，怕是没多少人能通过考核。
<br/>　　现场迅速安静了下来，众人凝神静待苏莫回答。
<br/>　　“具体的标准，你们明日自然会知晓，大家明日再来吧！”苏莫朗声说道，招收弟子的标准，他还没有考虑过，要回去仔细商议一番。
<br/>　　言罢，他便不再多言，打开水幕大阵，飞身进入了苍穹神宫之中。
<br/>(本章完)
