　　望着槐树下的黑色棺椁，慕清霄面色如常，走到槐树下，轻轻将棺盖推开，其中露出一副惨白的面孔。
<br/>　　其中躺着的是一名男子，面色惨白如纸，没有一丝血色，眼睛睁着望向天空，一副死不瞑目的模样。
<br/>　　若是普通人看到这一幕，估计早就吓的腿软。
<br/>　　“好浓郁的阴气，脉络中居然没有残留下一丝的阳气，被某些东西吸食殆尽了吗？”
<br/>　　说罢，慕清霄瞥了眼碧山山顶，看向老村长，道：“老村长，能跟我说说自闲山庄的事情吗？”
<br/>　　话语刚落，老村长身躯猛的一颤，瞳孔中满含惊惧之色：“自......自闲山庄！”
<br/>　　已经是几十年前的事情了，以前碧山也和别处一样，非常繁荣，并非像现在这样与世隔绝。
<br/>　　碧山顶上的自闲山庄，叶家是此处德高望重的家族，可就在几十年前的夜晚，山庄内的男女老少，全都被屠杀殆尽。
<br/>　　说到此处，老村长叹息一声，回忆起以前的过往。
<br/>　　“自从那以后，住在碧山周围的村庄都搬走，唯独翠萍村的居民舍不得离开，只能在山下扎根建村，最后碧山也成了乱葬岗，渐渐被人遗忘。”
<br/>　　闻言，慕清霄微微一愣，如果只是如此，刚才老村长为何露出恐惧的神色。
<br/>　　“老村长可曾去过山上？”
<br/>　　老村长欲言又止，不得已只能将事情吐露出来。
<br/>　　“老夫年轻的时候去过一趟，看到了些不好的东西，下山后就生过一场大病，自那以后就不敢在上去，也严禁村里人上去。”
<br/>　　慕清霄当然知道，老村长所指不好的东西是何物。
<br/>　　对于整件事情有了大致的了解，慕清霄微微一笑，道：“老村长可以放心，很快村庄就会恢复以前的样子。”
<br/>　　“多谢仙人。”
<br/>　　说罢，老村长又道：“仙人是要上山吧？老夫派人替您引路可好，村子里的猎户对这片山林非常的熟悉。”
<br/>　　“也好。”
<br/>　　...
<br/>　　...
<br/>　　“仙人，前面就是自闲山庄的范围了，村长吩咐过，我们村民不能上去，就只能送您到这里了。”
<br/>　　阴暗的山间小路上，两道身影站在其中，说话的青年穿着一身兽皮制作的衣裳，背上背着一柄长剑，正是当地猎户的儿子。
<br/>　　“行，你回去吧。”
<br/>　　山间小路早就被草木荆棘遮掩，根本无迹可寻，有猎户的儿子带路，倒是替他节省了许多的时间。
<br/>　　待青年马不停蹄的朝山下跑去，慕清霄则转身向山顶走去。
<br/>　　根据他的判断，村里死去的青年，恐怕与山上的自闲山庄有关，其中妖魔鬼怪众多，非比寻常。
<br/>　　走在山间小道上，树木繁茂的枝叶令得其中昏暗无比，太阳的光线根本无法照射进来，而且越是往上，阴气就越重。
<br/>　　穿过山间小道，来到碧山山顶，这里是一块面积极大的地面，整个山头仿佛被人一剑给削平似得。
<br/>　　而且，整个山顶都被笼罩在一股诡异的雾霾之中。
<br/>　　站在道路的尽头，映入眼中的是一座，若隐若现，且破败不堪的宅院。
<br/>　　宅院饱经风雨冲刷，有的围墙与房屋都被冲垮，台阶上更是长满了苔藓。
<br/>　　“这是......”
<br/>　　就在这时，慕清霄看向长满青苔的阶梯，其上有着无数的脚印，光是瞥了一眼脚印，他就大概了解了事情的来龙去脉。
<br/>　　苔藓上的脚印分明，近期内有三名男子来过此处，正好附和翠萍村中死去的三名男子。
<br/>　　至于目的何在，已经没有必要去追究。
<br/>　　四周放眼望去，破败的房屋高墙，光秃秃的树木，由于环境的影响，显得格外诡异。
<br/>　　收回目光，慕清霄就迈着脚步，向自闲山庄内部走去。
<br/>　　“咯吱——”
<br/>　　细微的开门声，在这寂静的环境中显得格外的清晰。
<br/>　　推开宅院大门，由于内部长时间没有人居住，院落中长满了杂草，中央处有一条由石子铺砌的道路，通往住宅客厅。
<br/>　　整个院落中都充斥着荒凉阴森的气息，从宅子的布置上来看，自闲山庄也算是大户人家，在世俗界恐怕也有相当高的地位。
<br/>　　当慕清霄走进院落中，身后的院门却是悄无声息的关了起来，
<br/>　　院门关闭的瞬间，一道阴风拂过，伴随着窸窸窣窣的声音响起，院落杂草中两气一团团绿幽幽的森冷光芒。
<br/>　　“呜呜呜......”
<br/>　　旋即，整个院落中响起一阵莫名的乌啼声，似哭泣似悲凉。
<br/>　　“魑魅魍魉，不知死活。”
<br/>　　慕清霄冷笑一声，脚步顿住，瞥了眼杂草丛林，随手一挥，一道恐怖的剑气席卷而出。
<br/>　　伴随着剑气席卷而出，整片灌木丛瞬间被剑气肆虐的满目疮痍，其中更是传来一道道凄厉的惨叫声。
<br/>　　击杀一些孤魂野鬼后，慕清霄向院落客厅走去，准备寻找女鬼叶沉香。
<br/>　　叶沉香身为江湖儿女，性格直爽，不拘小节，同时也是娇蛮小姐，遇到晋磊后，对他一往情深，甚至为了他收敛脾气，变得含羞带怯。
<br/>　　而在遭逢情变，含怨身死之后，最炽烈的爱也随之化为最彻骨的恨，徘徊在自闲山庄。
<br/>　　因怀着极大的怨念，过世多年仍然无法升天。
<br/>　　慕清霄对女鬼没什么兴趣，但另一半玉衡就在自闲山庄的小姐，叶沉香身上，找到她就能找到玉衡。
<br/>　　顺着院落中的石子小路，推开大门来到客厅，立刻就有一股霉气扑面而来。
<br/>　　站在门口，视线在其中扫视一圈，布置依旧完整，只是无论地上或者桌椅上，都堆积着一层厚厚的灰尘。
<br/>　　“咯吱...”
<br/>　　毫不犹豫的踏进客厅，身后的门扉再次关起，慕清霄目光一凝，不知何时几米外的玄廊上出现一名红裙女子。
<br/>　　红裙女子容貌秀美，无论着装还是打扮一看就是大家闺秀，唯一美中不足的就是她那苍白的脸色。
<br/>　　慕清霄可不认为这里有活人，瞥了她一眼，道：“你是叶沉香？”
<br/>　　“咯咯咯......”
<br/>　　红裙女子捂嘴轻笑，道：“公子，小女子漂亮吗？”
<br/>　　“......我对女鬼不感兴趣，还有，收起你的鬼魅术。”
<br/>　　说话的同时，一股恐怖的威压自他的体内席卷而出，将整个客厅包括玄廊都笼罩在内。
