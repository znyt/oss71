　　苏莫长身而起，发现姜神夜早已修炼结束，正在凝望着远处的九彩神柱。
<br/>　　而其他人，均是在闭目打坐，融炼真圣之气。
<br/>　　“苏兄，以你的天赋，想必可以多融合几股真圣之气！”姜神夜望向苏莫，轻笑着说道。
<br/>　　“哦，还能多次融合？”苏莫闻言惊讶。
<br/>　　“当然，这要看身体的素质，以及玄力的浑厚程度和玄力的质量！”姜神夜点头说道。
<br/>　　“原来如此！”苏莫心中恍然，这真圣之气看来还要多融合几股，越多越好。
<br/>　　只不过这真圣之气，效果虽然有，但并不是多强大。
<br/>　　“苏兄的天赋绝世，融合四股真圣之气，想必不再话下！”姜神夜笑咪咪的说道。
<br/>　　“或许吧！”
<br/>　　苏莫点了点头，略一沉吟，他看向那九彩旗幡露出的旗杆问道：“姜兄，你知晓这是何物吗？”
<br/>　　对方是三神宗之人，三神宗作为古灵星河最顶级的大势力，说不定了解此物，所以苏莫有此一问。
<br/>　　姜神夜闻言，目光同样望向旗杆，沉声道：“此物，无人能说清来历，至于和真圣界的关系，也是让人难以捉摸。”
<br/>　　苏莫心中有些唏嘘，这么奇怪之物，居然无人能弄清来历，按吴晋导师所言，此物连虚神境大能都能灭杀。
<br/>　　那此物到底是从何而来，难道还有超越虚神境的强者。
<br/>　　“不过，可以肯定的是，这真圣之气不属于这个宇宙！”姜神夜继续说道。
<br/>　　“不属于这个宇宙？那来自哪里？”苏莫闻言疑惑，这真圣之气来自哪里？难道还有其它的宇宙？
<br/>　　“不清楚！”
<br/>　　姜神夜摇了摇头，继续道：“或许虚神境的大能知晓，但也不会告诉所有人！”
<br/>　　就在此时，又有一股真圣之气飞射而出，姜神夜毫不迟疑，立刻将其抓取。
<br/>　　“苏兄，自便了！”姜神夜微微一笑，随即再次打坐，融炼真圣之气。
<br/>　　苏莫点了点头，便继续等待了起来，既然来了这里，这真圣之气也不能不要，尽量多融炼一些吧！
<br/>　　良久之后，苏莫再次抓取了一股真圣之气，再次开始了融炼。
<br/>　　这第二股真圣之气入体，同样融入了他全身的血肉筋骨以及神丹玄力之中，这真圣之气还是颇为玄妙的，不仅的加强天赋，还能增强根基。
<br/>　　只不过，第二股真圣之气，进入体内以后，效果稍稍有些减弱了，和身体融合的速度也变的更慢了。
<br/>　　其他一些天才，融炼了一股真圣之气后，也纷纷开始融炼第二股真圣之气。
<br/>　　真圣界外。
<br/>　　苏莫等四十八名天才进入真圣碑，众人眼前的影像便消失不见，其内逗留的人纷纷主动被传送了出来。
<br/>　　有人欢喜有人忧，有门下弟子或是族中子弟在真圣碑留名的势力，非常欣慰，但有些全部被淘汰的势力，则是颇为郁闷。
<br/>　　尤其是巫族的人，因为巫族的天才之中，无一人留名，全部被淘汰出局。
<br/>　　大祭司面无表情，眸中一片阴霾，其他巫族之人看到大祭司的脸色，也没人敢触其眉头。
<br/>　　他们都知道，苏莫出来之后，必死无疑。
<br/>　　很多人也都在等，等苏莫出来，看看大祭司巫仑如何处理！
<br/>　　巫族和苏莫的仇恨貌似不小，而在真圣界中苏莫又连连挫败巫族天才，擒拿了巫人王和巫殇，所有人都知道，大祭司肯定不会放过苏莫。
<br/>　　最主要的是，苏莫的天赋太恐怖了，若是不除掉，对于巫族来说后患无穷。
<br/>　　所以，只要大祭司不傻，定然会除掉苏莫。
<br/>　　对于此事，一众神武学府的弟子和导师也能猜到，但他们没有任何办法。
<br/>　　因为无人能抗衡大巫仑祭司，此地距离古周星，遥远的不可想象，想搬救兵也无能为力。
<br/>　　“希望你能安然渡过这一劫！”木璃导师心中暗道，他知道苏莫和巫族的仇怨，这个希望怕是很小。
<br/>　　任云都、谢彬等人，也是暗暗为苏莫担心，希望苏莫能渡过此劫。
<br/>　　“苏莫，你一定会死！”尤天韩心中嘶吼，经过一段时间的疗伤，他的伤势已经稍稍恢复，但想要彻底恢复伤势，还差的很远。
<br/>　　真圣碑的空间中。
<br/>　　所有的人都在融炼真圣之气，苏莫和姜神夜两人，或许是天赋最好，融炼的速度稍快不少，已经在进行第三次融炼了，而其他人还是第二次。
<br/>　　第三次融炼，速度变得更慢了，真圣之气融入身体的速度大大减缓。
<br/>　　嗡~~
<br/>　　就在此刻，苏莫感受到不远的空间一阵波动，而后一名天才的身影突然消失不见了。
<br/>　　“传送出去了？”苏莫见此双眉一挑，随即他面露沉吟之色。
<br/>　　看来当无法融合真圣之气的时候，就会被传送出去，绝对是这样。
<br/>　　他现在第三次融炼真圣之气，已经非常的缓慢了，按照他的估计，他还能融炼一股或两股，也将无法融炼了。
<br/>　　到时候，一旦出去，他将面临生死大劫。
<br/>　　少倾之后，苏莫叹了口气，抛却心中的杂念继续融炼，在他融炼的过程中，一名名天才不断的被传送了出去。
<br/>　　很多人因为天赋的原因，只能融炼两股真圣之气，一旦无法融炼就会被传送出去。
<br/>　　在苏莫将第三股真圣之气融炼完成之后，除了他之外，只剩下了三个人。
<br/>　　这三人便是姜神夜、以及飞灵族天才飞长云、黑鳞族天才黑锋。
<br/>　　黑锋和飞长云两人，已经在融炼第三股真圣之气，而姜神夜则在融炼第四股真圣之气了。
<br/>　　苏莫等了片刻，再次抓取了一股真圣之气，继续融炼。
<br/>　　“好恐怖的天赋！”
<br/>　　飞长云和黑锋两人，看了看苏莫，又看了看姜神夜，心中感慨不已。
<br/>　　他们融炼第三股真圣之气，已经到了极限，身体即将饱和，真圣之气马上就会和身体产生排斥了。
<br/>　　而苏莫和姜神夜，已经开始融炼第四股真圣之气，这就是差距，天赋上的差距。
<br/>　　时间缓缓流逝，又过去了几个时辰，飞长云和黑锋也被传送了出去，这片空间中只剩下了姜神夜和苏莫两人。
<br/>　　两人闭目盘膝，寂静无声，均是在静心融炼第四股真圣之气。
