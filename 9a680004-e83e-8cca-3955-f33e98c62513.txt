　　茫茫沙漠中，风沙肆虐，一名白衣青年闲庭信步的走在其中，走在黄沙中，却只留下两行极浅的脚印，仅仅片刻，脚印也被黄沙掩盖了去。
<br/>　　青年手中拿着一副地图，似乎在辨别方向。
<br/>　　塔戈尔大沙漠中的环境，严酷的出乎慕清霄的预料，脚下的黄沙，在烈日的暴晒下，滚烫无比，没有将小医仙带进来，绝对是明智的选择。
<br/>　　看了眼地图，原著中，美杜莎一直守护者青莲地心火，借由异火进化，他自然不必在去其他两处，而是直奔标注着美杜莎所在的地点。
<br/>　　十几分钟后，慕清霄望着面前一片平坦的沙漠，这里的地形极为平凡，没有任何能够引起人注意的地点，但冰皇地图上标注地点就在此处。
<br/>　　而且，身为凤凰的慕清霄，能够感觉到，脚下有诡异，根据温度来判断，估计是岩浆，青莲地心火最有可能就是在地下。
<br/>　　瞥了眼有些弧度的小山坡，慕清霄一掌轰出，地面瞬间炸裂开来，风沙漫天，再次放眼望去，映入眼中的是一个半米左右的地洞口。
<br/>　　嘴角牵起一丝笑容，慕清霄低喃道：“确实是这里，估计没错了。”
<br/>　　不再犹豫，纵身跃入其中，身体在下滑的同时，慕清霄扫过周围的洞壁，其上竟然极其光滑，基本没有半点石头自洞壁中凸出，看上去，仿佛是被极为恐怖的力量一击贯穿。
<br/>　　保持正常速度，下滑近半分钟后，慕清霄已经能看到地洞最底部，轻轻落在地上，目光在四周扫视一圈，其中有着十几条漆黑的通道，不知通往何处。
<br/>　　凭着感觉，慕清霄毫不犹豫的选择了最中间的一条通道，径直迈入其中。
<br/>　　通道呈一片深邃的漆黑，仿佛要将人吞噬，慕清霄是一个理智的人，自然不可能凭这一点就做下决定，而是这条通道中传来的热量最强。
<br/>　　走进漆黑的通道中，除了踩在细沙行的细微声音外，周围一片死寂，通道很长，始终看不到尽头的光亮，而周围的温度却越来越高。
<br/>　　足足走了十多分钟，通过一个转角后，不远处通道的尽头，出现了一抹赤红色的光芒，此景慕清霄在风云位面就见到过，不用猜都知道，接下来会有什么。
<br/>　　走到通道尽头，望着映入眼中的一幕，慕清霄露出一个果然的神色。
<br/>　　地底深处，火红的岩浆，在其中缓缓流淌，偶尔有这巨大的气泡自岩浆中浮现，随着一道轻微的声响，气泡炸裂开来，炽热的岩浆溅射而出，犹如烟花一样绚丽。
<br/>　　站在通道的尽头，眼前的一幕是何等的相似，当初在凌云窟中，雷峰塔下，都目睹过一样的情景。
<br/>　　回过神来，正当慕清霄准备一滩岩浆池的时候，岩浆一阵翻滚，紧接着一头狰狞的巨兽出现在了眼前。
<br/>　　从岩浆中钻出的生物，是一种类似于蛇形的魔兽，体型极长，一眼望去，估计有四五丈左右，通体火红，犹如一条浑圆的火玉，在它的身体表面，巴掌大小的红色鳞片布满其上。
<br/>　　最让慕清霄意外的是，眼前的这头魔兽，有着两颗脑袋，修长的脖颈分叉处，两个狰狞的脑袋，瞪着巨大的菱形瞳孔，其中充斥着狂暴与嗜血的杀意。
<br/>　　双头火灵蛇，一般生长于极热之地，靠吞噬岩浆为生，进化空间极大，出生时仅是一阶魔兽，若是机缘足够，进化成六阶魔兽也未必不可能。
<br/>　　感受到它瞳孔中的杀意，慕清霄双眼一眯，区区一条畜生也敢放肆，心念一动，剑气射出，还未回过神来，双头火灵蛇的两颗头颅就自脖颈滑落而下，身首异处。
<br/>　　双头火灵蛇尸体坠入岩浆中，慕清霄闭上双眼，仔细的在周围感应起来，岩浆估计有几千度的高温，但下还有一股恐怖的高温，来自于岩浆下面。
<br/>　　“这股极高的温度，肯定是青莲地心火了。”
<br/>　　伴随着一声嘹亮的凤鸣声，九霄琉璃焱自慕清霄身体中冒出，随着岩浆拍岸，一头高大两丈的黑风出现在洞穴中。
<br/>　　扇了扇华贵的翅膀，慕清霄扭了扭凤首，暗道：“难道是错觉，怎么感觉这身体比在地球的时候，好像稍微长大了一些。”
<br/>　　将复杂的情绪抛之脑后，现在最重要的是青莲地心火。
<br/>　　也不再多想，慕清霄双翅一展，直接窜入其中，顿时岩浆四射，畅游在岩浆里，就仿佛置身于温泉中一样，舒服程度的难以言明。
<br/>　　根据温度的变化，慕清霄以极快的速度，径直向岩浆底部潜去。
<br/>　　这种事情并不是第一次了，若他现在是人身，估计会极为警惕，但现在身为凤凰，岩浆就像自己的家一样，根本不需要有任何危机感。
<br/>　　下潜了将近一分钟左右，按照他的速度，最起码也下潜了数千米，正当他准备加速的时候，目光就被不远处，青光大盛的东西所吸引。
<br/>　　青色光芒，笼罩在这片岩浆之中，慕清霄定身望去，在那青色的光芒中，一朵青色的莲花，正温婉而立。
<br/>　　青莲地心火，异火榜排名第十九。
<br/>　　顾名思义，即存在于地心熔岩之中的火焰，生于大地深处，历经大地之火的无数次锤炼，融合，压缩，雕制......
<br/>　　十年成灵，百年成形，千年成莲，大成之时，其色偏青，莲心生一簇青火，其名为青莲火，也称青莲地心火，此火威力莫测，在临近火山地带之处，甚至能够引发火山喷发，形成大自然的毁灭力量！
<br/>　　脑海中浮现一系列信息，双翅一振就来到青色光芒笼罩之地，周围的温度陡然下降。
<br/>　　也不在意周围的变化，慕清霄的视线停留在中央位置的那朵青色莲花上，莲花分八叶，八片青色的叶子浑然天成，晶莹剔透，极其的完美。
<br/>　　在莲花中央，还有一个两三尺左右的小小莲台，莲台上的一些孔洞中，散发着点点萤光，这些都是由最精纯火属性能量，凝聚而成的莲子。
<br/>　　慕清霄嘿嘿一笑，空间中却是响起一声凤鸣，青莲地心火加莲子，估计能让自己的实力再上一个台阶。
