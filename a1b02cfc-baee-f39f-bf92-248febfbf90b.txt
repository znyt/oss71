　　蛮荒圣山，秦不死的宫殿之中，苏莫盘膝静坐在地。
<br/>　　光头大汉和黑衣中年人，都在各自的宫殿之中，等待秦不死的消息。
<br/>　　而苏莫，则是准备闭关冲击境界。
<br/>　　取出三十余枚储物戒，苏莫稍一检查，顿时面露喜色。
<br/>　　“不错，不愧是巫族之人！”苏莫嘴角勾起一丝笑容，这三十余枚储物戒中，每一枚储物戒内，最少都要一百亿极品灵石，总共加起来不下于四千亿极品灵石，简直是惊天巨富。
<br/>　　除此之外，灵草灵药也有很多，足够他冲击武皇境九重境界了。
<br/>　　唰！
<br/>　　手掌一翻，苏莫的掌中出现了一枚晶石，这枚晶石有鹅蛋大小，比一般的灵石要稍大一些。
<br/>　　这晶石呈纯白之色，其内蕴含强烈的灵气波动，仿若其内是一个缩小的灵气之海。
<br/>　　“这是什么晶石？”苏莫心中惊讶无比，同时也是怦然心动。
<br/>　　这晶石绝对不凡，定是超越极品灵石的存在，是外界那种大能强者修炼所用。
<br/>　　少倾，苏莫将这枚晶石郑重的收了起来，这种晶石并不多，三十余枚储物戒中，也只有十几块这种晶石而已。
<br/>　　随即，苏莫的身形突兀的消失不见了，原地只留下了一副画卷。
<br/>　　神图空间。
<br/>　　辽阔的平原上，聚集着上百万的苍穹神宫之人。
<br/>　　因为司空炎等人都早已回到了神图空间中，所以，所有人都知道了，他们已经打败了试炼者。
<br/>　　是以，无数的苍穹神宫弟子，均是非常的振奋，震耳的喧嚣声此起彼伏。
<br/>　　“宫主来了！”
<br/>　　“见过宫主！”
<br/>　　“见过宫主！”
<br/>　　苏莫乍一现身，现场立刻安静了下来，旋即无数的人纷纷向他行礼。
<br/>　　苏莫挥了挥手，和众人闲聊了几句之后，便飞到了远处的山峰。
<br/>　　这座山峰正是玄天灵柩所在的山峰，因为怕芊寻月被打扰，苏莫用神图空间中的空间之力隔绝了此地，任何人都无法靠近。
<br/>　　而此刻，一名女子双眸紧闭，在山脚下盘膝而坐。
<br/>　　此女，正是巫娴。
<br/>　　巫娴修为被封印，在此地遭到神图中空间力量的束缚，连移动都做不到。
<br/>　　唰！
<br/>　　苏莫飞身落在了巫娴的面前，冷眼望着对方。
<br/>　　“你为何不杀我？”巫娴睁开了眼眸，冷冷的凝视着苏莫。
<br/>　　“我需要问你几个问题！”苏莫淡淡的说道。
<br/>　　“说吧！”巫娴冷声道。
<br/>　　此女倒是颇为爽快，让苏莫有些意外。
<br/>　　“离开苍穹世界的通道之外，有多少巫族的强者镇守？”苏莫沉声问道。
<br/>　　“看来你是想要离开苍穹世界！”巫娴闻言冷笑了起来，同时她心念电转，苏莫有需求就好，她就能谈条件了。
<br/>　　“不错！希望你老实回答，不然的话别怪我无情！”苏莫点头道。
<br/>　　“告诉你也无妨，通道之外，有我族两名尊者镇守！你是不可能离开的！”巫娴眸中精光一闪，冷冷的说道。
<br/>　　“两名尊者？”苏莫闻言眉头紧皱了起来，一名尊者已经是非常恐怖的存在，居然有两名尊者。
<br/>　　“你的实力虽强，但根本抵挡不住尊者一招！”巫娴继续道。
<br/>　　“但是，我有你在手上！“苏莫面无表情的说道。
<br/>　　“呵呵！”
<br/>　　巫娴闻言冷笑了起来，道：“看来你是想靠我来威胁我族尊者，但你可知，尊者是何等恐怖的存在，岂会受你威胁！”
<br/>　　苏莫闻言沉默了，他的确不知尊者是何等存在，但是，据他从秦不死口中得知，尊者的实力不可对抗。
<br/>　　“我可以给你一个机会，放了我，我保你安然离开！”巫娴眼眸微眯，郑重的说道。
<br/>　　苏莫依旧沉默，他的目光如刀，直视对方的双眸，他可不信对方会这么好心，一旦放了对方，出去以后他必死无疑。
<br/>　　“当我是三岁孩童吗？”
<br/>　　苏莫冷笑一声，突然手指连点，道道指芒没进巫娴的小腹丹田之中，他将封印再次加固了一番。
<br/>　　随即，不由分说，苏莫将对方玉手上一双薄如蝉翼的银色手套和储物戒摘下，闪身扬长而去。
<br/>　　“你这个土著，简直是自寻死路！”巫娴见此，娇喝一声，银牙咬的吱吱作响。
<br/>　　想她作为巫族天冥星支脉中，大祭司的曾孙女，父母都是支脉中武尊境的长老，何时受过如此的屈辱？
<br/>　　她心中的怒火，简直如同火山沸腾。
<br/>　　若不是她大意了，怎么可能会被苏莫生擒？
<br/>　　她还有很多强大的底牌，只不过，她没有想到苏莫能看穿她的青烟身法，被瞬间击败。
<br/>　　让她连很多强大的手段，都无法施展，不然的话，她也不至于成为阶下囚。
<br/>　　巫娴暗暗发誓，待出去以后，定然让苏莫死无葬身之地。
<br/>　　……
<br/>　　苏莫远离了山峰，找到了一片五人的僻静之地，盘膝而坐。
<br/>　　打量了一番手中的银色手套和储物戒，苏莫便将手套收了起来。
<br/>　　这手套应该也是尊器，非常不凡，不过却不适合他用。
<br/>　　随即，苏莫意念一探，储物戒内的一切，他便看的清清楚楚。
<br/>　　巫娴的储物戒内，空间奇大，简直犹如一座小城一般，但东西倒是不多，极品灵石只有数十亿。
<br/>　　不过，极品灵石不多，但那种鹅蛋般大小的晶石，却是有四十余块。
<br/>　　除此之外，还有三枚玉简和少量的兵器，以及衣物等杂物。
<br/>　　这些兵器有刀有剑，全部都是帝级上品的级别。
<br/>　　将三枚玉简取出，苏莫静静的查看了起来。
<br/>　　待查看完三枚玉简的内容，苏莫心中有些唏嘘，不愧是巫族，族中年轻子弟，都能修炼尊级武学。
<br/>　　这三枚玉简，尽皆是尊级下品武学，分别为《银月掌》、《天仓诀》、《如烟步》。
<br/>　　难怪巫娴的实力这么强，原来修炼了一身的尊级武学。
<br/>　　少倾，苏莫将几门武学都收了起来，这几门武学，无论适不适合他修炼，现在都不是修炼的时候。
<br/>　　随即，苏莫手掌一番，三十余枚神丹出现在了他的身前，目前他的第一要务，还是要尽快突破修为。
<br/>　　这些神丹，其主人尽皆是武皇境九重武者，或是准帝武者，神丹内蕴含的力量磅礴浩瀚，强大到了极致。
