　　太苍的气势极盛，爆发出了最强的战力，整个人如同一刀闪电，裹挟着全身的力量，向并冰冻住的火球斩去。
<br/>　　这是他和凤晴暗中商量好的对策，由凤晴冰封司空炎，他出手击败司空炎。
<br/>　　众人见此，心中大震，仿佛是看到了胜利的曙光。
<br/>　　尤其是巫铨，眸光大亮，心中有些激动了起来，凤晴和太苍若是胜了，他也能出一口心中的闷气了。
<br/>　　唯有苏莫面色淡淡，他对司空炎有信心，这个级别的攻击，还不足以击败司空炎。
<br/>　　果然，太苍的攻击，还没斩中冰封的火球，只见火球上的坚冰，冒出冲天的白烟，瞬间消融一空。
<br/>　　轰！
<br/>　　火球的火势瞬间暴涨，变成了直径足有十丈的火球，充满了整个通道，直接将太苍和冰凤裹挟在了火焰之内。
<br/>　　轰！轰隆隆隆！！
<br/>　　随即，众人只见到火球之中，爆响连连，火焰不断的震动。
<br/>　　啊！啊！
<br/>　　紧接着，便是太苍和凤晴的惨叫之色，充满了无尽的痛苦。
<br/>　　众人见此，心中再震，完了！
<br/>　　嗡~~~
<br/>　　就在此刻，无尽的火焰瞬间收敛，消失无踪，司空炎和凤晴、太苍的身影显露而出。
<br/>　　只不过，三人之中，只有司空炎身形挺拔的站立在通道之中，凤晴和太苍两人，几乎被烧成了两块黑炭，躺在地上。
<br/>　　两人不仅肉身遭受重创，神魂也被烧伤，可谓是极为凄惨。
<br/>　　少倾，两人的身影消失无踪，直接被传送了出去。
<br/>　　现场一片死寂，所有人都不可思议的看着这一幕，众人心中，掀起惊涛骇浪。
<br/>　　正面一战之下，太苍和凤晴两人，居然不是一合之敌，败的如此凄惨。
<br/>　　“都出去吧，没你们的事了！”司空炎看向众人，阔气的挥了挥手道。
<br/>　　击败太苍和凤晴，对于他来说，并不困难。
<br/>　　他的天赋和战力，已经超过了万古天骄，可不是太苍和凤晴能比，就算两人联手也无济于事。
<br/>　　众人面面相觑，随即纷纷叹了口气，如同斗败的公鸡一般，向出口处走去。
<br/>　　若是此地只有司空炎一人，这些人或许还会尽力一战，但还有苏莫在此，他们完全提不起战意了。
<br/>　　巫铨彻底绝望了，他已经尽力了，但是，事不可为，他也只能就此作罢。
<br/>　　很快，一众人纷纷离开了迷宫城。
<br/>　　迷宫城之外，无数的人唏嘘，这一战同样没有悬念，屠霄和炎空，简直所向无敌啊。
<br/>　　屠霄的天梦元神，神秘而诡异，炎空的战力同阶无敌，无人能出其右。
<br/>　　龙双双暗叹一声，虽有一点失望，但也没有多少意外，这个结果她猜到了。
<br/>　　司空炎和苏莫两人，并未离开迷宫城，两人又等了一个多时辰，又陆陆续续的抢夺了数十人，最后确定迷宫城之中的人已经所剩无几，才最终才走出迷宫城。
<br/>　　走出迷宫城之后，苏莫和司空炎两人，大步而行，向龙腾走去。
<br/>　　两人，如同两颗璀璨的太阳，极为惹人瞩目，几乎所有人的目光，都落在了两人的身上。
<br/>　　这些目光之中，有惊讶、有震撼、有贪婪、有愤怒，各不相同。
<br/>　　大多数人，是震惊苏莫和司空炎的实力，也有很多人想要将苏莫和司空炎据为己有，成为自己的奴仆。
<br/>　　而那些被苏莫和司空炎击伤的奴仆，或者是奴仆被击杀的龙族之人，则是对苏莫和司空炎怒目而视。
<br/>　　“司空炎，这一次，我们应该没有暴露多少吧？”
<br/>　　苏莫面色不变，大步走向龙腾，意念向司空炎传音问道。
<br/>　　“你没有暴露多少，我暴露的较多一些！”
<br/>　　司空炎意念传音回道，而后，他继续传音，道：“不过，我上次进入终极神王殿，并没有暴露什么，终极神王殿对于我的底细毫不知晓，所以没有什么风险！”
<br/>　　对于司空炎而言，他只要不是闹的太大，基本上不用担心。
<br/>　　“嗯！”苏莫心中稍安，只要没有暴露太多，那就不用担心。
<br/>　　“哈哈，屠霄、炎空，你们表现的还算不错！”眼见苏莫和司空炎走来，龙腾大笑着说道。
<br/>　　不过，龙腾此言一出，无数的人脸皮抽.搐，这也太能嘚瑟了吧？
<br/>　　这种表现，已然逆天了，居然说还算不错？
<br/>　　根据众人的观察，这屠霄和炎空两人，抢夺的宝物，已经超过了全部宝物的三分之一。
<br/>　　“多谢主人夸奖！”
<br/>　　司空炎和苏莫来到龙腾的身前，纷纷向龙腾行礼，表现的很是恭敬。
<br/>　　“嗯！”
<br/>　　龙腾点了点头，随即环视身边的众人，道：“诸位，告辞了！”
<br/>　　言罢，龙腾向苏莫和司空炎示意，便带着两人，大步离去。
<br/>　　并没有人阻拦，高空中的两位执事，也并没有多言。
<br/>　　不过，这个事情，他们肯定会向龙族高层禀报。
<br/>　　毕竟，奴仆之中，出现了两名堪比万古天骄的人物，这不是小事。
<br/>　　苏莫三人，很快便离开了奴务府，回到了龙腾的宫殿之中。
<br/>　　“苏莫，司空炎，你们两人最近就不要出去了，得到了这么多宝物，就安心在这里修炼吧！”
<br/>　　宫殿之中，龙腾向苏莫和司空炎说道。
<br/>　　“龙腾，今日之事，会不会闹出什么大的动静？”苏莫面色郑重，向对方问道。
<br/>　　“动静肯定不小，但是影响不会很大！”龙腾沉声说道，他现在唯一担心的是，有厉害的人物，想要将司空炎和苏莫抢去。
<br/>　　其他的他倒是不担心，龙族的高层，不会因为苏莫和司空炎的天赋，而加以调查。
<br/>　　毕竟，苏莫和司空炎两人，修为还低，而且只是奴仆的身份。
<br/>　　“那就好！”苏莫闻言，便放下了心。

<br/>　　当然，即便是上品的神器，也与终极神剑完全不能相比。
