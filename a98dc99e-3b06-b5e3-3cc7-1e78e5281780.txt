        季雅珊微笑着说道：“关系好不好，我就不知道了。我只知道，五年前，四爷救过八爷的命。四爷算是八爷的救命恩人吧！”
<br/>
<br/>        “原来如此。”童筱颖这才明白过来，只是，如果八爷知道现在的四爷其实是大哥傅夜沉，那大哥岂不是很危险。
<br/>
<br/>        “好了啦！别愣着了！赶紧去洗漱！”季雅珊又催促道。
<br/>
<br/>        童筱颖这才动身从床上下来。
<br/>
<br/>        洗漱完换了衣服后，她见窗外仍旧灰蒙蒙的一片，忍不住去看了一眼墙上的壁钟。
<br/>
<br/>        才凌晨五点！
<br/>
<br/>        不等她反应过来，季雅珊便递给她一个对讲机，然后挽着她的胳膊，带动着她往门外走去。
<br/>
<br/>        “每天四点半起床，然后我们得去厨房做早餐，爷起来后，会通过对讲机呼叫。接着，我们就要替爷把早餐端进屋内，给爷准备衣服，然后是打扫房间，洗衣服等等。反正听从爷的安排就行了。”季雅珊条条框框地说道。
<br/>
<br/>        童筱颖只是听着，什么话也不多说。
<br/>
<br/>        到了厨房后，童筱颖发现季雅珊总共准备了七份早餐，不免有些好奇地问：“我和你加上十爷、八爷、四爷，也就五个人，你为什么准备七份早餐？”
<br/>
<br/>        “古堡里，还住着一对制药师夫妻。”季雅珊手里拿着汤勺，将锅里的汤给舀了出来。
<br/>
<br/>        童筱颖脑子里的第一反应是什么，嘴里便脱口而出：“这座古堡里是在制毒吗？”
<br/>
<br/>        “噗嗤”一声，季雅珊忍俊不禁道：“他们不是在制毒啦！而是在研究解药。”
<br/>
<br/>        “解药？什么解药？”童筱颖不禁纳闷了起来。
<br/>
<br/>        季雅珊笑容渐敛，一脸惆怅地回答：“我也不大清楚到底是什么解药，我听那位何药师说，他和他的妻子只是在负责研究一种‘解药’的副作用的缓解药。总而言之，集团里真正制药房不在古堡里。他们夫妻两，也是因为犯了事，才来城堡里，替‘冒先生’卖命的。”
<br/>
<br/>        “噢！是这样啊！”童筱颖听得有些心不在焉，怎么又与“药”有关？
<br/>
<br/>        他们到底在研究什么药？
<br/>
<br/>        而大哥他们又在追查什么药？
<br/>
<br/>        童筱颖心里觉得，这趟浑水，她是不是趟得有点深？现在后悔，肯定是来不及了。只能走一步，看一步。
<br/>
<br/>        这古堡里，厨房和餐厅是连在一起的。
<br/>
<br/>        童筱颖帮着季雅珊把早餐端上桌后，一对男女携手走了进来。
<br/>
<br/>        他们看到童筱颖的时候，神情惊愕。
<br/>
<br/>        童筱颖无意间抬眸，看向他们的时候，更是目瞪口呆。
<br/>
<br/>        这里不仅仅只是浑水，而且还是冤家路窄。
<br/>
<br/>        原来季雅珊所说的制药师夫妻，指的就是何明旭和林若琴。
<br/>
<br/>        也对，他们俩学的都是生化专业，而且还是同班同学，毕业后成为制药师，一点都不足为奇。
<br/>
<br/>        林若琴看到童筱颖身上穿着女仆装，不禁讥笑道：“哟！傅家的少奶奶不好当，跑来这儿当女佣了啊！”
<br/>
<br/>        “你们认识？”季雅珊诧异道，而且她还感觉到一股火药味。
<br/>
<br/>        童筱颖皱了下眉头，否认：“不认识。”
<br/>
<br/>        何明旭倒是落落大方地向季雅珊解释：“筱颖是我的前女友。”
<br/>
<br/>        季雅珊又下意识地看向童筱颖。
<br/>
<br/>        童筱颖淡定地笑了笑：“噢！是！我想起来了！你确实是我的前男友了。而且，还是劈腿的前男友。”
<br/>
<br/>        “你说谁劈腿呢？”林若琴踱步上前，指着童筱颖的鼻子谩骂，“你这个穷不拉几、山窝窝里出来的乡下人，有什么资格谈恋爱，嗯？”
<br/>
<br/>        “林若琴，你给我闭嘴！筱颖是十爷的小姨子，现在又是四爷的贴身女佣。论关系，还轮不到你在这儿指手画脚。”季雅珊呵斥道。
<br/>
<br/>        何明旭连忙上前拉住林若琴，挤眉弄眼地示意她先安静一会儿。
<br/>
<br/>        “两位吃完早餐后，就赶紧回你们的实验室里去搞研究吧！听说，你们最近在赶时间。别到时候，药没制出来，命也跟着丢了。”季雅珊接着说道。
<br/>
<br/>        何明旭端起桌上的两份早餐，对季雅珊微微一笑：“我和若琴去实验室里吃，盘子到中午吃午饭的时候，我们再带出来。”
<br/>
<br/>        “好。”季雅珊礼貌性地笑了笑。
<br/>
<br/>        何明旭带着林若琴离开厨房后，季雅珊便忍不住八卦起来：“你们是怎么一回事？”
<br/>
<br/>        “我和何明旭是高中同学，大一的时候在一起的，但是不在同一所大学，所以，他在跟我谈恋爱的时候，同时又交了一个女朋友。被我发现后，我们就分手了。”童筱颖慢条斯理地说，明亮的眼睛里没有一丝异样的情绪，好像在叙述着一件无关紧要的事情。
