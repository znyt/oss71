　　碧螺老祖盯着楚风眠，疯狂的吼道。
<br/>　　他的眼神之中蕴含无比炙热的贪婪，在楚风眠的身躯之中，他居然是发现了一株神树。
<br/>　　还是一株幼小的神树，足以是被人降伏的神树。
<br/>　　这个发现让碧螺老祖简直是要发狂了。
<br/>　　身为木系修士，碧螺老祖自然是知晓神树的强大之处，他得到那建木树枝，仅仅是吞噬了其中的一小部分，就足以是让他的实力发生翻天覆地一般的变化。
<br/>　　而楚风眠的身躯之中，可是拥有一整株神树的存在。
<br/>　　“得到了这一株神树，我肯定可以踏入半步天人境，甚至冲击天人境，神树啊，成长到极致的神树，足以媲美天人，这神树我一定要的到！”
<br/>　　碧螺老祖现在也是拼了一切，漫天血雾在他的生命力下，瞬间凝聚成了身形。
<br/>　　以他九劫古帝的境界，肉身被毁，几乎没有太大的损失，数百道翠竹，随着他的灵力控制，向着楚风眠猛然刺了过去。
<br/>　　“就凭你们三个人联手，也想要杀了我？可笑至极，正好我今日就让你们见识一下，我真正的力量！”
<br/>　　楚风眠将建木树枝跟那一段乌木全部得到，如今心情大好。
<br/>　　看到三人联手，他都丝毫不惧，他现在也正好想要看看，他的实力到底是达到了哪一步。
<br/>　　既然已经得到了建木树枝，那现在楚风眠却是毫无顾忌，面对那无数翠竹的轰击，楚风眠甚至都没有理会。
<br/>　　任由这无数翠竹，刺在楚风眠的身躯之上。
<br/>　　咔嚓。
<br/>　　一根根翠竹，刺入到楚风眠的身躯之上。
<br/>　　楚风眠看似薄弱的皮肤，如今却是显得坚如钢铁，每一根翠竹刺过去的一刻，都无法划破楚风眠的皮肤。
<br/>　　在楚风眠的皮肤之上，乃是十二万九千六百枚血肉晶体化为了完美防御。
<br/>　　翠竹之上的锋芒，都被全部折断。
<br/>　　“碧螺老祖，五行刀祖，葵海王，今日你们三个人，都死在这里吧，一个人也别想逃！”
<br/>　　楚风眠大笑一声，双剑齐出，两道剑光对准碧螺老祖，五行刀祖直接斩了过去。
<br/>　　同时在楚风眠的背后，天涯剑匣之中，一道道剑光也不断闪烁，从天涯剑匣之中飞了出来，向着楚风眠身后的葵海王斩去
<br/>　　以一敌三！
<br/>　　楚风眠面对这三名九劫古帝，以一敌三，如今更是占据了上风。
<br/>　　“这建木树枝的力量，正是雄厚！”
<br/>　　楚风眠敢以一敌三，也是感觉得到，在他的身躯之中，那建木树枝被神树炼化之后，几乎是源源不断的力量，涌入到楚风眠的身躯之中。
<br/>　　这建木树枝，虽然纵然是九劫古帝都难以炼化，可对楚风眠来说，想要炼化却是瞬息之间的事情。
<br/>　　神树本就是曾经建木的分支，跟建木树枝，属于同源，这建木树枝落到神树的手中，被瞬间吞噬，其中的力量被神树所吞噬，涌入到楚风眠的身躯之中。
<br/>　　楚风眠本就是吞噬了武祖精血之后，实力暴涨了一倍，在加上如今吞噬了这建木树枝的力量。
<br/>　　哪里还有要逃的道理？
<br/>　　九劫古帝，纵然是对三大势力来说，都是绝对的高层，楚风眠没杀一个，都对三大势力沉重的打击。
<br/>　　现在这样的机会之下，楚风眠当然不会放过他们。
<br/>　　“九玄星河！”
<br/>　　轰隆！
<br/>　　笼罩在楚风眠身边，九玄剑术打出的星河，也随着楚风眠手指一点，向着那五行刀祖，碧螺老祖，葵海王三人轰击而去。
<br/>　　瞬间之间他们三人的身形，都被拉入到了这星河之中，在这星河之中的无数剑气，密密麻麻的斩向他们三人。
<br/>　　“该死，这小子的力量更强了，居然这么快就炼化了那建木树枝？”
<br/>　　碧螺老祖感受着楚风眠的力量，眼神之中无比的疯狂。
<br/>　　他已经是感觉到了，楚风眠身躯之中神树的力量攀升，显然是建木树枝，已经是被楚风眠所炼化了。
<br/>　　几番交手之下，楚风眠身上的力量，非但是没有减弱，反而是越来越强。
<br/>　　随着神树炼化建木树枝，力量节节攀升，也代表着楚风眠的力量也越来越强。
<br/>　　如今这三人联手都难以出手反击。
<br/>　　“不能跟此人纠缠了，我们走！该死，此人果然成了心腹大患了，等我们出去，一定要召集其他的古帝，来一起围杀了此人！”
<br/>　　五行刀祖长啸一声，也做出了决定。
<br/>　　“走！”
<br/>　　葵海王也同样长啸一声。
<br/>　　三人的力量身形，突然间凝聚在一起，化为了一道开天辟地一般的神芒，轰击在了星河之上。
<br/>　　将这九玄剑术所化的星河，硬生生打出了一个缺口来。
<br/>　　这一道神芒，打破星河，直接向着光幕的方向飞去。
<br/>　　这让楚风眠都有些没有想到。
<br/>　　这九劫古帝拼了命，居然还有如此可怕的力量，楚风眠可以压制住他们三人，但是想要将他们三人一网打尽，却是难上加难。
<br/>　　“不能让他们逃了！”
<br/>　　楚风眠眼神之中露出几分杀意。
<br/>　　为了对付这三人，楚风眠将神树的秘密都给暴露了出去，底牌尽出。
<br/>　　不杀了这三人，未来绝对会麻烦无比，会引来大祸。
<br/>　　“无上剑阵！”
<br/>　　一千八百道无上剑气，在楚风眠身躯之中飞了出来，来到了那光幕的入口前，互相交织这，化为了一道无上剑阵。
<br/>　　天下剑阵之中，最强的便是以无上剑气所化的无上剑阵，这万千剑阵之祖，也是天下第一的杀阵。
<br/>　　“无上剑气！”
<br/>　　“这种剑道至宝，怎么也在此人的手里？”
<br/>　　“此人到底是得到了过多少惊天的奇遇！”
<br/>　　看到这无上剑气所化的无上剑阵，封锁了光幕，五行刀祖三人几乎是要嫉妒的发狂了。
<br/>　　剑道至宝，先天神兽精血，甚至还有神树，楚风眠得到的每一个机遇，都让他们嫉妒的发狂。
<br/>　　他们现在都恨不得要杀了楚风眠，瓜分了楚风眠身上的宝物，他们三人的实力都会突飞猛进。
<br/>　　可现在他们三人要做的，是要逃。
<br/>　　“这无上剑阵，并不完整，只是其中的一部分所化！我们直接冲出去！”
<br/>　　五行刀祖看了一眼这无上剑阵，也看得出这无上剑阵，跟着传闻之中的并不相同，有所残缺。
